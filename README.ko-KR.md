# mAI 🚀

mAI는 24시간 내내 운영될 수 있도록 에이전트를 조직합니다. AI 팀 전체를 채용(영입), 예약 및 보고서 생성을 수행합니다. 온라인 상태를 유지할 필요 없이 제어권을 유지할 수 있습니다.

**May**는 모든 작업을 도울 준비가 된 핵심 AI 비서입니다.

## 주요 기능

mAI는 귀하와 함께 진화하는 에이전트를 찾고, 생성하고, 협업할 수 있는 워크스페이스입니다. mAI에서는 에이전트를 작업의 기본 단위로 취급하여 인간과 에이전트가 함께 진화하는 인프라를 제공합니다.

### 👥 작업 단위로서의 에이전트
AI 팀 전체의 활동을 예약하고 모니터링하세요.
- **생산성 향상**: 모든 에이전트를 한곳에 모으세요.
- **통합 메시징**: 선호하는 애플리케이션에서 직접 에이전트와 채팅하세요.

### 🛠️ 에이전트 빌더 (Agent Builder)
맞춤형 AI 팀을 구축하는 것은 에이전트 빌더에서 시작됩니다. 필요 사항을 한 번만 설명하면 에이전트가 즉시 사용할 수 있도록 자동 구성됩니다.
- **통합 인텔리전스**: 사용자의 제어하에 모든 모델과 모달리티에 원활하게 액세스하세요.
- **확장된 기술**: 호환되는 플러그인과 MCP 서버를 사용하여 에이전트를 일상적인 도구에 연결하세요.

### 🤝 협업
mAI는 에이전트 그룹을 도입하여 실제 동료와 일하는 것처럼 에이전트와 함께 작업할 수 있도록 합니다. 시스템은 작업에 적합한 에이전트를 구성하여 병렬 협업과 지속적인 개선을 가능하게 합니다.
- **페이지 (Pages)**: 공유된 컨텍스트를 사용하여 여러 에이전트와 동시에 문서를 작성하고 다듬으세요.
- **스케줄링**: 실행을 예약하여 오프라인 상태일 때도 에이전트가 적절한 시간에 작업할 수 있도록 하세요.
- **프로젝트**: 명확한 구조를 유지하기 위해 프로젝트별로 작업을 구성하세요.

### 🧠 인간-에이전트 공동 진화
가장 훌륭한 AI는 사용자를 깊이 이해하는 AI입니다. mAI는 상호 작용을 통해 스스로 풍부해지는 개인 메모리 기능을 갖추고 있습니다.
- **지속적인 학습**: 에이전트가 사용자의 작업 방식으로부터 학습합니다.
- **투명한 메모리**: 에이전트는 사용자가 전적으로 제어하는 구조화되고 편집 가능한 메모리를 사용합니다.
