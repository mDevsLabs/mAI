# Web-Integration 🌐

**mAI** ist als plattformübergreifende Anwendung konzipiert. Neben der Desktop-App ist mAI auch über eine optimierte Web-Version vollständig zugänglich, was einen reibungslosen Zugriff über jeden Webbrowser ermöglicht.

## Funktionen der Web-Version

Die Web-Version von mAI bietet nahezu vollständige Funktionsparität mit der Desktop-Anwendung:
- Flüssige, responsive und reaktionsschnelle Benutzeroberfläche.
- Zugriff auf alle Sprachmodelle mit Ihren eigenen API-Schlüsseln (lokal im Browser gespeichert oder synchronisiert).
- Nutzung der meisten Plugins (Plugins, die lokale Systemberechtigungen erfordern, wie der direkte Zugriff auf das lokale Dateisystem, können eingeschränkt sein).

## Cloud-Synchronisation

Wenn Sie mAI auf mehreren Geräten verwenden, können Sie eine sichere Cloud-Synchronisation aktivieren. Ihre Chat-Daten, benutzerdefinierten Agenten und Präferenzen können mit Ende-zu-Ende-Verschlüsselung auf Ihrem eigenen Hosting-Server oder über kompatible Cloud-Speicherlösungen (wie WebDAV) gespeichert werden.
