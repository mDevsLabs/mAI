# Agenten-Verwaltung 👥

In **mAI** können Sie spezialisierte Agenten erstellen und konfigurieren, um spezifische Anforderungen und Anwendungsfälle abzudecken. Die Verwaltung der Agenten erfolgt direkt über die grafische Benutzeroberfläche oder über Konfigurationsdateien.

## Neuen Agenten erstellen

So erstellen Sie einen Agenten in mAI:
1. Gehen Sie zur linken Seitenleiste und klicken Sie auf die Schaltfläche **Agenten**.
2. Klicken Sie oben in der Liste auf **Agent erstellen**.
3. Füllen Sie die Basisinformationen aus:
   - **Name**: Der Bezeichner Ihres Agenten.
   - **Avatar**: Wählen Sie ein Symbol oder Emoji aus oder laden Sie ein Bild hoch.
   - **Beschreibung**: Eine kurze Zusammenfassung der Rolle des Agenten.

## Bestehenden Agenten bearbeiten

Sie können jeden benutzerdefinierten Agenten bearbeiten, indem Sie auf das zugehörige Einstellungssymbol in der Liste klicken. Sie haben Zugriff auf erweiterte Konfigurationen wie die Definition des System-Prompts, die Auswahl des Standard-Sprachmodells und die Verwaltung seiner Plugins.

## Agenten löschen

Wenn ein benutzerdefinierter Agent nicht mehr benötigt wird, können Sie ihn dauerhaft über die Einstellungen löschen. Bestehende Konversationen, die mit diesem Agenten verknüpft sind, werden nicht gelöscht, aber der Agent selbst steht für neue Chats nicht mehr zur Verfügung.
