# Bildschirmfoto 📸

Die Anwendung **mAI** enthält ein intuitives Tool für Bildschirmfotos, mit dem Benutzer schnell Bereiche ihres Bildschirms erfassen und direkt in den Chat senden können, damit der Agent sie visuell analysieren kann.

## Verwendung des Erfassungswerkzeugs

So machen Sie einen Screenshot:
1. Klicken Sie auf das Symbol **Kamera / Erfassen** in der Chat-Eingabeleiste oder verwenden Sie den entsprechenden Tastaturkurzbefehl.
2. Die Benutzeroberfläche wird vorübergehend ausgeblendet, damit Sie den Bereich oder das Fenster auswählen können, das Sie erfassen möchten.
3. Nach der Auswahl wird das Bild automatisch als Anhang an Ihr Chat-Eingabefeld angehängt.
