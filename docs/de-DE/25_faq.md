# FAQ ❓

Hier finden Sie Antworten auf die am häufigsten gestellten Fragen zur Konfiguration und Nutzung von **mAI**.

## Allgemeine Nutzung

### Kann ich mAI ohne Internetverbindung nutzen?
Ja, vorausgesetzt, Sie nutzen lokale Modelle über Dienste wie **Ollama** oder LocalAI. Bei der Nutzung von Cloud-Modellen (wie OpenAI oder Anthropic) ist eine aktive Internetverbindung erforderlich.

### Wie aktualisiere ich meine API-Schlüssel?
Gehen Sie zu den Einstellungen (`Strg + ,`) und öffnen Sie die Registerkarte **Sprachmodelle**. Dort können Sie API-Schlüssel für jeden unterstützten Anbieter hinzufügen, ändern oder entfernen.

## Agenten-Anpassung

### Wer ist May?
**May** ist die standardmäßige intelligente Hauptassistentin in mAI. Sie ist äußerst vielseitig und so konfiguriert, dass sie Ihnen ab dem ersten Start hilft.

### Werden meine Agenten mit anderen Nutzern geteilt?
Nein. Standardmäßig werden alle von Ihnen erstellten benutzerdefinierten Agenten lokal auf Ihrem Gerät gespeichert. Sie können sie jedoch manuell exportieren, um sie mit anderen zu teilen.
