# Benutzergedächtnis 🧠

Das Benutzergedächtnis in **mAI** ermöglicht es Ihren Agenten und insbesondere **May**, sich über verschiedene Gespräche hinweg an wichtige Fakten über Sie zu erinnern. Dies macht die Benutzererfahrung viel persönlicher und nahtloser.

## Wie das Gedächtnis funktioniert

Während Sie mit mAI interagieren, extrahiert die Anwendung automatisch relevante Kontextinformationen und speichert sie in einer lokalen Gedächtnisdatenbank. Dazu können gehören:
- Ihre Sprachpräferenzen oder Ihr Schreibstil.
- Projekte, an denen Sie gerade arbeiten.
- Ihre Entwicklungspräferenzen (bevorzugte Frameworks, Codestile).
- Persönliche Details, die Sie teilen möchten (wie Ihr Name oder Ihre berufliche Rolle).

## Verwaltung Ihres Gedächtnisses

Sie haben die volle Kontrolle über die von mAI gespeicherten Informationen:
1. Öffnen Sie die allgemeinen Anwendungseinstellungen.
2. Gehen Sie auf die Registerkarte **Gedächtnis**.
3. Sie sehen eine Liste der gespeicherten Fakten in Form von Karten.
4. Sie können jeden Eintrag manuell bearbeiten oder einen Fakt dauerhaft aus der Datenbank löschen, damit der Agent ihn vergisst.
