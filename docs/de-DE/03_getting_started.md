# Schnellstart-Anleitung 🚀

Folgen Sie diesen Schritten, um **mAI** in wenigen Minuten auf Ihrem lokalen Computer einzurichten und auszuführen.

## Systemanforderungen

- **Node.js**: Version 18.x oder höher (LTS empfohlen).
- **Paketmanager**: **pnpm** wird für die Installation empfohlen.
- **Git**: Zum Klonen des Repositories.

## Installation

1. **Repository klonen**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```
2. **Abhängigkeiten installieren**:
   ```bash
   pnpm install
   ```
3. **Entwicklungsserver starten**:
   ```bash
   pnpm run dev
   ```
4. **Öffnen Sie Ihren Browser** unter `http://localhost:3000`, um die mAI-Benutzeroberfläche anzuzeigen und mit Ihrer Assistentin **May** zu chatten.
