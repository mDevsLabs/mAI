# Unterstützte Sprachmodelle 🧠

**mAI** ist eine modellagnostische Schnittstelle. Sie unterstützt eine Vielzahl kommerzieller Cloud-basierter und lokaler Open-Source-KI-Anbieter.

## Cloud-Anbieter

Sie können die folgenden Cloud-Anbieter konfigurieren, indem Sie Ihren persönlichen API-Schlüssel eingeben:
- **OpenAI**: GPT-4o, GPT-4 Turbo, GPT-3.5 usw.
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku.
- **Google Gemini**: Gemini 1.5 Pro, Gemini 1.5 Flash.
- **Mistral AI**: Mistral Large, Mistral Medium, Codestral.
- **Groq**: Ultraschneller Zugriff auf Open-Source-Modelle wie Llama 3 und Mixtral.

## Integration lokaler Modelle

Für Anwender, die vollständigen Datenschutz wünschen, unterstützt mAI die lokale Ausführung:
- **Ollama**: Vereinfachte native Integration. Führen Sie Ollama im Hintergrund aus, und mAI erkennt Ihre lokal installierten Modelle (wie Llama 3 oder Mistral) automatisch.
- **OpenAI-kompatible APIs**: Verbinden Sie einen beliebigen lokalen oder entfernten Inferenzserver (wie LM Studio) über eine benutzerdefinierte API-Basis-URL.
