## Einen Fehler melden

Haben Sie einen Fehler gefunden oder eine Funktionsanfrage? Öffnen Sie ein Issue auf GitHub — wir verfolgen dort alles.

- [**GitHub Issues**](https://github.com/mDevsLabs/mAI/issues/new/choose) — Fehlerberichte, Funktionsanfragen und Diskussionen

Suchen Sie vor dem Einreichen in den vorhandenen Issues, um Duplikate zu vermeiden. Fügen Sie Schritte zur Reproduktion für Fehler hinzu; beschreiben Sie Ihren Anwendungsfall für Funktionsanfragen.
