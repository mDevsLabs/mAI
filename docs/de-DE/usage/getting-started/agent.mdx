# Agent

Jeder Agent in mAI ist ein dauerhafter Teamkollege — keine einmalige Konversation. Erstellen Sie einen Übersetzungs-Agenten, der sich an Ihre Terminologie erinnert, einen Programmier-Agenten, der auf Ihren Tech-Stack abgestimmt ist, oder einen Schreib-Agenten, der zu Ihrer Tonalität passt. Konfigurieren Sie die Rolle, das Modell, die Skills und die Wissensdatenbank einmal; lassen Sie ihn von da an intelligenter werden.

Diese Anleitung führt Sie durch das Erstellen und Konfigurieren Ihres ersten Agenten.

## Was einen Agenten ausmacht

Jeder Agent basiert auf sechs Komponenten:

| Komponente | Zweck |
| :--- | :--- |
| **Systemrolle** | Die Persönlichkeit, das Fachwissen und die Verhaltensrichtlinien des Agenten |
| **KI-Modell** | Das LLM, das den Agenten antreibt — GPT-4o, Claude, Gemini und mehr |
| **Skills** | Fähigkeiten wie Websuche, Code-Ausführung und Bildgenerierung |
| **Integrationen** | MCP-Verbindungen und Integrationen mit externen Diensten |
| **Wissensdatenbank** | Dokumente und Daten, die der Agent durchsuchen und referenzieren kann |
| **Gedächtnis** | Persönlicher Kontext, den der Agent über Sitzungen hinweg lernt und behält |

Fangen Sie einfach an — eine Systemrolle und ein Modell sind alles, was Sie brauchen. Fügen Sie Skills, Integrationen und Wissensdatenbanken hinzu, wenn Ihre Arbeit es erfordert.

## Arten von Agenten

- **Integrierte Agenten** — Agent Builder (hilft Ihnen beim Erstellen von Agenten), Pages Agent (Schreiben), Memory Agent (verwaltet Ihr persönliches Gedächtnis)
- **Benutzerdefinierte Agenten** — Agenten, die Sie selbst für Ihre spezifischen Arbeitsabläufe erstellen und konfigurieren
- **Community-Agenten** — Gebrauchsfertige Agenten, die von der mAI-Community geteilt werden — mit einem Klick hinzugefügt

## Einen neuen Agenten erstellen

Drei Wege zu Ihrem ersten Agenten: Intelligente Erstellung mit dem Agent Builder, Erstellung von Grund auf neu oder Hinzufügen aus der Community. Bei allen drei können Sie im Nachhinein jedes Detail anpassen.

### Intelligente Erstellung mit dem Agent Builder

<Steps>
  ### Agent Builder öffnen

  Klicken Sie in der linken Seitenleiste auf **Agent erstellen** oder verwenden Sie die Option **Agent erstellen** unter der Chat-Eingabe.

  ### Beschreiben Sie Ihr Ziel

  Chatten Sie auf der Agenten-Profilseite mit dem Agent Builder im rechten Panel. Beschreiben Sie in natürlicher Sprache, was Sie benötigen — der Agent Builder generiert die Rolleneinstellungen, den Systemprompt und die Skill-Konfiguration automatisch.

  ### Überprüfen und anpassen

  Überprüfen Sie die generierte Konfiguration und passen Sie alles an, was nicht passt.
</Steps>

![Intelligente Erstellung mit Agent Builder](/blog/assets2964497066067ca0588a7767eb4c1709.webp)

### Einen benutzerdefinierten Agenten erstellen

Für die volle Kontrolle: Legen Sie Avatar, Name, Systemprompt, Modell und Integrationen selbst auf der Agenten-Profilseite fest. Beginnen Sie mit dem Systemprompt — alles andere folgt.

### Aus der Community hinzufügen

Die Community verfügt über Tausende gebrauchsfertiger Agenten — Design-Experten, Programmierhilfen, Texter, akademische Tutoren und mehr.

- Gehen Sie in der linken Seitenleiste auf **Community → Agenten** oder scrollen Sie auf dem Hauptbildschirm nach unten zu **Community-Agenten**.
- Öffnen Sie die Detailseite eines Agenten und klicken Sie auf **Agent hinzufügen & chatten**, um sofort zu starten.

![Agent aus Community hinzufügen](/blog/assetsbcd98b0913d2dfc30d5a2b5523115d33.webp)

### Skills und Integrationen verbinden

Nachdem Sie einen Agenten erstellt haben, können Sie seine Fähigkeiten durch Hinzufügen von Skills und Verknüpfen externer Dienste erweitern.

Gehen Sie in der linken Seitenleiste auf **Agenten-Profil → + Skill hinzufügen**.

![Skill aus Agenten-Profil hinzufügen](/blog/assetsdc6362a2c6db476fc1b7b9d6bf443af2.webp)

### Agenteninformationen bearbeiten

Ihr Agent sollte sich mit Ihrer Arbeit weiterentwickeln. Öffnen Sie eine Konversation, klicken Sie in der linken Seitenleiste auf **Agenten-Profil** und aktualisieren Sie jederzeit beliebige Angaben — Name, Rolle, Modell oder Skills.

### Eine Wissensdatenbank verknüpfen

Verknüpfen Sie eine Wissensdatenbank, damit Ihr Agent auf der Grundlage Ihrer eigenen Daten (Dokumente, Spezifikationen oder Recherchen) antwortet und nicht nur auf der Grundlage seines allgemeinen Trainings.

Verwenden Sie in der Chat-Oberfläche die Schaltfläche für die Wissensdatenbank, um eine Wissensdatenbank auszuwählen und anzuhängen.

![Wissensdatenbank im Chat verknüpfen](/blog/assetsc89931ace11a936d87d0b87a6bca1069.webp)

## Effektive Systemprompts schreiben

Der Systemprompt ist der wichtigste Teil Ihres Agenten. Er definiert das Fachwissen, das Verhalten und den Kommunikationsstil des Agenten. Ein gut geschriebener Systemprompt führt zu konsistenten, qualitativ hochwertigen Ergebnissen.

**Beispielstruktur:**

```
Du bist ein erfahrener [ROLLE], spezialisiert auf [BEREICH].

Dein Fachwissen umfasst:
- [Spezifischer Bereich 1]
- [Spezifischer Bereich 2]
- [Spezifischer Bereich 3]

Bei der Beantwortung:
- [Kommunikationsrichtlinie 1]
- [Kommunikationsrichtlinie 2]
- [Formatierungspräferenz]
```

**Tipps für gute Prompts:**

- **Werden Sie konkret bezüglich des Fachwissens** — „Du bist ein Senior-TypeScript-Entwickler, der auf React und Leistungsoptimierung spezialisiert ist“ ist weitaus nützlicher als „Du bist ein Programmier-Assistent“.
- **Definieren Sie den Kommunikationsstil** — Legen Sie Tonalität (professionell, locker), Antwortlänge (prägnant/detailliert) und Format (Markdown, Aufzählungspunkte, Fließtext) fest.
- **Bauen Sie bereichsspezifische Richtlinien ein** — Für einen Agenten für juristische Texte: „Zitieren Sie gegebenenfalls relevante Gesetze. Weisen Sie darauf hin, wenn der Rat eine professionelle Rechtsberatung erfordert.“
- **Fangen Sie einfach an und iterieren Sie** — Beginnen Sie mit einem einfachen Prompt und verfeinern Sie ihn basierend auf den tatsächlichen Antworten des Agenten.

## Das richtige Modell wählen

Verschiedene Modelle haben unterschiedliche Stärken:

| Modell | Am besten geeignet für |
| :--- | :--- |
| GPT-4o, Claude Sonnet | Ausgewogene Leistung, Programmierung, Analyse |
| Claude Opus, GPT-o1 | Komplexes logisches Denken, nuancierte Aufgaben |
| Gemini | Multimodale Aufgaben, langer Kontext |
| GPT-4o-mini, Claude Haiku | Schnelle Antworten, einfache Aufgaben, Kosteneffizienz |
| Ollama (lokal) | Datenschutzsensible Aufgaben, Offline-Nutzung |

Stimmen Sie das Modell auf die Komplexität der Aufgabe und Ihr Budget ab. Für Agenten, die Routineaufgaben erledigen, ist ein kleineres Modell oft ausreichend und schneller.

## Best Practices

- **Testen Sie Ihre Prompts** — Führen Sie nach dem Erstellen eines Agenten einige Testgespräche durch, um zu überprüfen, ob er sich wie erwartet verhält. Verfeinern Sie den Systemprompt basierend auf dem, was nicht funktioniert.
- **Eine Verantwortung pro Agent** — Spezialisierte Agenten übertreffen Generalisten. Erstellen Sie separate Agenten für Programmierung, Schreiben und Recherche, anstatt einen Agenten für alles zu verwenden.
- **Aktualisieren Sie Agenten bei neuen Anforderungen** — Agenten sind keine statischen Systeme. Überarbeiten Sie den Systemprompt, wenn sich Ihre Arbeitsabläufe oder Anforderungen ändern.
- **Nutzen Sie den Agent Builder für komplexe Setups** — Lassen Sie den Agent Builder bei Agenten, die mehrere Skills und einen anspruchsvollen Systemprompt benötigen, die Ersteinrichtung übernehmen und verfeinern Sie diese dann manuell.

## Agenten verwalten

Wenn Sie viele Agenten und Gruppenchats haben, hält die Organisation in Gruppen die Seitenleiste sauber und erleichtert das Wechseln.

### Eine Gruppe erstellen

Öffnen Sie auf der mAI-Homepage das Menü der Agentenliste und wählen Sie **Neue Gruppe hinzufügen**.

![Neue Gruppe in Agentenliste hinzufügen](/blog/assets98cddf4b80b8bac0c250a5236062d198.webp)

### In eine Gruppe verschieben

Wählen Sie einen vorhandenen Agenten oder Gruppenchat aus und klicken Sie dann auf **In Gruppe verschieben**. Sie können neue Agenten oder Gruppenchats auch direkt innerhalb einer Gruppe erstellen.

### Gruppen verwalten

Gehen Sie zur Agentenliste oder zum Gruppenmenü und wählen Sie **Gruppen verwalten**, um sie umzubenennen oder neu anzuordnen.

### Häufig genutzte Agenten anheften

Heften Sie einen Agenten für einen schnelleren Zugriff an den Anfang der Liste an. Wählen Sie den Agenten aus und wählen Sie **Anheften** aus dem Menü. Angeheftete Agenten bleiben ganz oben.

## Eine Kopie erstellen

Duplizieren Sie einen Agenten, um verschiedene Konfigurationen auszuprobieren, ohne das Original zu ändern. Wählen Sie den Agenten aus und wählen Sie **Kopie erstellen** aus dem Menü.

## Einen Agenten löschen

Wählen Sie den Agenten aus, wählen Sie **Löschen** aus dem Menü und bestätigen Sie den Vorgang. Der Agent wird entfernt.
