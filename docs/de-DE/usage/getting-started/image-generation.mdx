# Bildgenerierung

Beschreiben Sie, was Sie möchten — mAI verwandelt Text in Bilder. Produktprototypen, Design-Inspirationen, Illustrationen oder kreative Erkundungen: Wählen Sie ein Modell, legen Sie Ihre Parameter fest und erhalten Sie in Sekundenschnelle hochwertige Bilder. Alle generierten Bilder werden automatisch in Ihrer Ressourcenbibliothek gespeichert.

## Erste Schritte

Klicken Sie auf der mAI-Hauptoberfläche auf **Zeichnen**, um die Seite zur Bildgenerierung zu öffnen.

## Einen Prompt eingeben

Beschreiben Sie das gewünschte Bild im Eingabefeld. Je genauer Ihre Beschreibung ist, desto präziser ist das Ergebnis.

**Effektive Prompt-Struktur:**

```
[Motiv] [Stil/Medium] [Umgebung/Hintergrund] [Beleuchtung] [Stimmung] [Technische Details]
```

Beispiele:

```
"Die Skyline einer futuristischen Stadt bei Sonnenuntergang, digitale Kunst, Cyberpunk-Stil, Neonlichter, die sich auf nassen Straßen spiegeln, kinematografische Beleuchtung, 4K-Details"

"Das Innere eines gemütlichen Cafés, Aquarell-Illustration, warmes goldenes Licht, das durch die Fenster fällt, Topfpflanzen auf den Fensterbrettern, weiche und einladende Atmosphäre"

"Ein Produktfoto einer minimalistischen Lederbörse auf einem sauberen weißen Hintergrund, Studiobeleuchtung, scharfer Fokus, kommerzieller Fotografie-Stil"
```

**Prompt-Tipps:**

- **Werden Sie konkret bezüglich des Stils** — "Ölgemälde", "Aquarell", "digitale Kunst", "fotorealistisch", "Anime", "Vektorillustration"
- **Beschreiben Sie das Licht** — "dramatische Schatten", "weiches diffuses Licht", "goldene Stunde", "Studiobeleuchtung"
- **Geben Sie die Bildkomposition an** — "Porträtansicht", "Weitwinkel", "Nahaufnahme", "Vogelperspektive"
- **Fügen Sie Qualitätsmodifikatoren hinzu** — "hohe Detailgenauigkeit", "4K", "scharfer Fokus", "professionelle Qualität"
- **Vermeiden Sie vage Formulierungen** — Wörter wie "schön", "nett" oder "gut" bringen wenig — beschreiben Sie, was Sie tatsächlich sehen möchten.

## Ein KI-Modell wählen

mAI bietet verschiedene KI-Modelle zur Bildgenerierung an. Die Modelle haben unterschiedliche Stärken:

![Ein Modell wählen](/blog/assetsdd913561927c64d32bd390cee6846f9a.webp)

| Modell | Am besten geeignet für |
| :--- | :--- |
| **DALL-E 3** | Realistische Fotos, Illustrationen, präzise Umsetzung des Prompts |
| **Flux** | Künstlerische Stile, kreative Bilder, schnelle Generierung |
| **Stable Diffusion** | Hochgradig anpassbar, Community-Stile und feinabgestimmte (fine-tuned) Modelle |
| **fal.ai-Modelle** | Verschiedene spezialisierte Stile und schnelle Generierung |

Probieren Sie verschiedene Modelle mit demselben Prompt aus, um zu sehen, welches die besten Ergebnisse für Ihren Anwendungsfall liefert.

## Referenzbilder auswählen (Optional)

Wenn Sie Referenzbilder haben, laden Sie diese hoch, um den Generierungsprozess zu steuern. Klicken Sie auf die Schaltfläche zum Hochladen oder ziehen Sie Ihre Referenzbilder direkt per Drag-and-Drop hinein. Sie können mehrere Referenzbilder hochladen.

![Referenzbilder für Bildgenerierung hochladen](/blog/assets3c160860feef0bd7c653eeb46f683445.webp)

Referenzbilder helfen dem Modell, Ihren gewünschten Stil, die Bildkomposition oder die Farbpalette zu verstehen.

## Seitenverhältnis des Bildes wählen

Wählen Sie ein Seitenverhältnis basierend auf dem Verwendungszweck aus:

- **1:1** — Social-Media-Beiträge, Profilbilder
- **16:9** — Breitbild, Präsentationen, Banner
- **9:16** — Smartphone-Bildschirme, Stories, Reels
- **4:3** — Allgemeine Verwendung, ältere Bildschirmformate
- **3:2** — Standard-Fotografie, Drucke

## Anzahl der Bilder festlegen

Wählen Sie, wie viele Bilder in einem Durchgang generiert werden sollen. Die gleichzeitige Generierung mehrerer Bilder bietet Ihnen Varianten zur Auswahl. Beginnen Sie mit 2–4, um das beste Ergebnis zu finden.

## Bilder anzeigen und herunterladen

Nach der Generierung erscheinen die Bilder auf der Zeichenseite. Sie können:

- Jedes Bild durch Anklicken in voller Größe in der Vorschau anzeigen
- Favoriten auswählen und herunterladen
- Direkt aus dem Bildbetrachter teilen

Alle generierten Bilder werden automatisch in Ihrer Ressourcenbibliothek gespeichert.

![Generierte Bilder in Ressourcenbibliothek](/blog/assets974acc551878f2f395518a3fbb9bd924.webp)

## Tipps für bessere Ergebnisse

**Iterieren Sie Ihre Prompts** — Wenn das erste Ergebnis nicht ganz passt, passen Sie ein Element nach dem anderen an, anstatt den gesamten Prompt neu zu schreiben. Fügen Sie mehr Details hinzu, ändern Sie die Stilbeschreibung oder geben Sie an, was Sie nicht im Bild haben möchten.

**Verwenden Sie ein Referenzbild** — Das Hochladen eines Referenzbildes zusammen mit Ihrem Prompt hilft dem Modell, den gewünschten Stil, die Farbpalette oder die Komposition zu treffen.

**Probieren Sie mehrere Varianten aus** — Generieren Sie mehr als 4 Bilder gleichzeitig und wählen Sie das beste aus. Die KI-Bildgenerierung unterliegt einer gewissen Zufälligkeit — einige Variationen werden deutlich besser sein als andere.

**Modell auf die Aufgabe abstimmen** — Verwenden Sie fotorealistische Modelle (DALL-E 3, Flux) für Produktfotos und realistische Szenen; verwenden Sie stilorientierte Modelle für künstlerische Illustrationen.
