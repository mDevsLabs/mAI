# Chat-Benutzeroberfläche & Verzweigung

mAI bietet eine hochentwickelte Chat-Benutzeroberfläche, die über lineare Konversationen hinausgeht. Erkunden Sie mehrere Pfade und Richtungen in Ihren Diskussionen mit KI-Agenten mithilfe von Konversationsverzweigungen, Nachrichtenbearbeitung und Unterstützung für Rich-Media-Inhalte.

## Kern-Chat-Funktionen

Die Chat-Benutzeroberfläche bietet ein modernes, responsives Erlebnis, das sowohl für Desktop- als auch für Mobilgeräte optimiert ist:

- **Echtzeit-Streaming-Antworten** — Verfolgen Sie, wie KI-Antworten in Echtzeit mit flüssigen Animationen generiert werden.
- **Unterstützung für Rich-Media-Inhalte** — Formatierter Text, Codeblöcke, Bilder und Multimedia-Inhalte werden ansprechend dargestellt.
- **Nachrichtenbearbeitung** — Bearbeiten Sie vorherige Nachrichten, um Ihre Fragen zu präzisieren.
- **Nachrichten-Reaktionen** — Reagieren Sie mit schnellem Feedback auf Nachrichten.
- **Kontexterhalt** — Konversationen behalten den vollen Kontext für kohärente Diskussionen bei.

## Konversationsverzweigung

Eine der leistungsstärksten Funktionen von mAI ist die Konversationsverzweigung, die lineare Chats in dynamische, baumartige Strukturen verwandelt.

### Wie Verzweigung funktioniert

Von jeder beliebigen Nachricht in Ihrer Konversation aus können Sie einen neuen Zweig erstellen, um verschiedene Richtungen zu erkunden, während der ursprüngliche Kontext erhalten bleibt.

**So erstellen Sie einen Zweig:**

1. Bewegen Sie den Mauszeiger über eine beliebige Nachricht in der Konversation.
2. Klicken Sie in den Nachrichtenaktionen auf das Verzweigungssymbol.
3. Wählen Sie Ihren Verzweigungsmodus.
4. Setzen Sie das Gespräch in einer neuen Richtung fort.

### Verzweigungsmodi

<Tabs>
  <Tab title="Fortsetzungsmodus">
    Der **Fortsetzungsmodus** erweitert Ihre aktuelle Diskussion, während der wertvolle Kontext aus vorherigen Nachrichten erhalten bleibt.

    **Wann zu verwenden:**

    - Erkunden alternativer Lösungen für dasselbe Problem.
    - Testen verschiedener Ansätze unter Beibehaltung der Hintergrundinformationen.
    - Vergleich mehrerer Antworten auf dieselbe Frage.

    **Beispiel-Workflow:**

    ```
    Original: "Wie kann ich meine React-App optimieren?"
    ├─ Zweig 1: "Fokus auf Code-Splitting-Strategien"
    ├─ Zweig 2: "Fokus auf Bildoptimierung"
    └─ Zweig 3: "Fokus auf Reduzierung der Bundle-Größe"
    ```
  </Tab>

  <Tab title="Eigenständiger Modus">
    Der **Eigenständige Modus** startet ein neues Thema basierend auf einer vorherigen Nachricht von Grund auf neu.

    **Wann zu verwenden:**

    - Tieferes Eintauchen in ein bestimmtes Unterthema.
    - Starten einer verwandten, aber unabhängigen Diskussion.
    - Isolierung eines bestimmten Aspekts für eine fokussierte Untersuchung.

    **Beispiel-Workflow:**

    ```
    Hauptkonversation: "Datenbankdesign für E-Commerce"
    └─ Eigenständiger Zweig: "Erzähl mir speziell mehr über die Sicherheit bei der Zahlungsabwicklung"
    ```
  </Tab>
</Tabs>

### Navigieren in Zweigen

Wenn eine Nachricht mehrere Zweige hat, sehen Sie Steuerelemente zur Zweignavigation:

- **Pfeiltasten** — Navigieren Sie zwischen verschiedenen Zweigen.
- **Zweiganzeige** — Zeigt den aktuellen Zweig an (z. B. "2/5").
- **Zweigsymbol** — Visueller Hinweis auf Verzweigungspunkte.

Klicken Sie auf die Links-/Rechts-Pfeile, um zwischen verschiedenen Konversationspfaden zu wechseln und Antworten zu vergleichen.

## Anwendungsfälle

<Tabs>
  <Tab title="Softwareentwicklung">
    **Erkunden mehrerer Lösungen**

    Beim Debuggen oder Entwerfen von Architekturlösungen:

    - Erstellen Sie Zweige für verschiedene Implementierungsansätze.
    - Vergleichen Sie Vor- und Nachteile verschiedener Entwurfsmuster.
    - Testen Sie Edge-Cases, ohne Ihren Hauptkonversationsfluss zu verlieren.

    ```
    "Wie soll ich meine API strukturieren?"
    ├─ REST-Architektur-Untersuchung
    ├─ GraphQL-Ansatz
    └─ gRPC-Implementierung
    ```
  </Tab>

  <Tab title="Kreatives Schreiben">
    **Inhalte iterieren**

    Für die Erstellung und Bearbeitung von Inhalten:

    - Generieren Sie mehrere Varianten von Überschriften oder Einleitungen.
    - Erkunden Sie verschiedene Erzählrichtungen.
    - Vergleichen Sie Ton- und Stilvariationen.
  </Tab>

  <Tab title="Recherche & Analyse">
    **Themen vertiefen**

    Bei der Durchführung von Recherchen:

    - Untersuchen Sie verschiedene Aspekte eines komplexen Themas.
    - Vergleichen Sie gegensätzliche Standpunkte.
    - Führen Sie separate Analyse-Threads.
  </Tab>

  <Tab title="Lernen">
    **Adaptive Lernpfade**

    Für Bildungszwecke:

    - Probieren Sie verschiedene Erklärungsstile aus, um herauszufinden, was am besten funktioniert.
    - Erkunden Sie verwandte Konzepte, ohne den Hauptfaden zu verlieren.
    - Bauen Sie Wissen schrittweise über Zweige hinweg auf.
  </Tab>
</Tabs>

## Tipps für effektives Verzweigen

<AccordionGroup>
  <Accordion title="Zweige organisiert halten">
    Verwenden Sie aussagekräftige erste Nachrichten in jedem Zweig, um sich zu merken, was jeder Pfad untersucht. Dies erleichtert die spätere Rücknavigation.
  </Accordion>

  <Accordion title="Ergebnisse nebeneinander vergleichen">
    Nutzen Sie die Zweignavigation, um schnell zwischen den Antworten hin- und herzuwechseln und den besten Ansatz für Ihre Bedürfnisse zu ermitteln.
  </Accordion>

  <Accordion title="Nicht zu stark verzweigen">
    Obwohl Verzweigungen mächtig sind, kann das Erstellen von zu vielen Zweigen unübersichtlich werden. Konzentrieren Sie sich auf 2–4 Schlüsselalternativen.
  </Accordion>

  <Accordion title="Nutzen Sie die Fortsetzung für Iterationen">
    Bei der Verfeinerung von Antworten hilft der Fortsetzungsmodus, den Kontext zu wahren, während Variationen untersucht werden.
  </Accordion>

  <Accordion title="Nutzen Sie den eigenständigen Modus für Deep Dives">
    Wenn Sie ein Seitenthema erkunden möchten, ohne Ihre Hauptkonversation zu überfrachten, sorgt der eigenständige Modus für Ordnung.
  </Accordion>
</AccordionGroup>

## Chat-Verwaltung

### Konversationen organisieren

- **Suche** — Finden Sie bestimmte Nachrichten oder Themen über alle Konversationen hinweg.
- **Wichtige Chats anheften** — Halten Sie häufig genutzte Konversationen ganz oben.
- **Archivieren** — Räumen Sie Ihren Arbeitsbereich auf, ohne wertvolle Konversationen zu löschen.
- **Exportieren** — Speichern Sie Konversationen in mehreren Formaten zum Teilen oder Sichern.

### Nachrichtenaktionen

Jede Nachricht enthält Schnellaktionen:

- **Kopieren** — Kopieren Sie den Nachrichteninhalt in die Zwischenablage.
- **Bearbeiten** — Ändern Sie Ihre gesendeten Nachrichten.
- **Löschen** — Entfernen Sie Nachrichten aus der Konversation.
- **Regenerieren** — Fordern Sie eine neue Antwort von der KI an.
- **Verzweigen** — Erstellen Sie einen neuen Konversationspfad.
- **Teilen** — Exportieren Sie einzelne Nachrichten oder Threads.

Verzweigte Konversationen bleiben in Ihrem Konversationsverlauf erhalten, sodass Sie jederzeit zurückkehren können, um später andere Pfade zu erkunden.
