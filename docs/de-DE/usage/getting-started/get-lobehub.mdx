# mAI erhalten

mAI läuft überall dort, wo Sie arbeiten — Browser, Desktop oder Telefon. Melden Sie sich einmal an, und Ihre Agenten, Seiten und Ihr Speicher bleiben auf jedem Gerät synchronisiert.

## Plattform-Übersicht

| Plattform | Verfügbarkeit | Herunterladen |
| :--- | :--- | :--- |
| **Web** | Jeder moderne Browser | [m-ai-officiel.vercel.app](https://m-ai-officiel.vercel.app) |
| **macOS** | Native App | [Herunterladen](https://mprojects.odoo.com/download) |
| **Windows** | Native App | [Herunterladen](https://mprojects.odoo.com/download) |
| **iOS** | App Store | [App Store](https://apps.apple.com/app/id6471212236) |
| **Android** | Google Play / APK | [Google Play](https://play.google.com/store/apps/details?id=com.lobehub.app) · [APK](https://mprojects.odoo.com/download) |

## Web

Keine Installation erforderlich. Öffnen Sie [m-ai-officiel.vercel.app](https://m-ai-officiel.vercel.app) in einem beliebigen modernen Browser und legen Sie sofort los. Die Daten werden mit der Cloud synchronisiert — wechseln Sie jederzeit das Gerät.

**Am besten für:** Schnellen Zugriff, das Ausprobieren von mAI vor der Installation oder die Nutzung auf freigegebenen / eingeschränkten Computern.

## Desktop

Laden Sie die native App von der [Download-Seite](https://mprojects.odoo.com/download) herunter. Verfügbar für **macOS** und **Windows** (noch nicht im Mac App Store oder Microsoft Store).

**Am besten für:** Offline-Zugriff, systemnahe Integration und ein dediziertes Fenster für fokussiertes Arbeiten.

## Mobilgerät

| | iOS | Android |
| :--- | :--- | :--- |
| **Store** | [App Store](https://apps.apple.com/app/id6471212236) | [Google Play](https://play.google.com/store/apps/details?id=com.lobehub.app) |
| **Alt. Download** | — | [APK von der Download-Seite](https://mprojects.odoo.com/download) |

Melden Sie sich mit Ihrem mAI-Konto an, um auch unterwegs auf Ihre Agenten und Unterhaltungen zuzugreifen.

**Am besten für:** Sprachunterhaltungen, schnelle Check-ins und die Überprüfung von Ergebnissen abseits des Schreibtischs.
