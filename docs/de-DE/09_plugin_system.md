# Plugin-System 🔌

**mAI** verfügt über ein leistungsstarkes Plugin-Ökosystem, das es Konversationsagenten ermöglicht, die Grenzen ihres Trainingswissens zu überwinden, indem sie mit externen Tools und Diensten interagieren.

## Integrierte Plugins

Hier sind die wichtigsten standardmäßig verfügbaren Plugins:

1. **Websuche**: Ermöglicht dem Agenten, das Internet in Echtzeit zu durchsuchen, um aktuelle Informationen abzurufen.
2. **Artefakte**: Ermöglicht das interaktive Generieren, Bearbeiten und Vorschauen von Code, Dokumenten oder UI-Designs in einer dedizierten Seitenleiste.
3. **Wetter**: Fragt Wetter-APIs ab, um lokale Vorhersagen abzurufen.
4. **Code-Rechner**: Führt lokale Skripte aus, um komplexe Berechnungen präzise zu lösen.

## Aktivierung und Konfiguration

Jeder Agent kann über eigene aktivierte Plugins verfügen. Sie können Plugins für einen bestimmten Agenten über die Registerkarte **Plugins** in den Agenteneinstellungen aktivieren oder deaktivieren. Einige Plugins (wie die kommerzielle Websuche) erfordern möglicherweise zusätzliche Einstellungen wie API-Schlüssel in den allgemeinen Einstellungen.
