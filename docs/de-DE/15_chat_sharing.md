# Chat-Teilen 📤

**mAI** macht es einfach, Ihre Gespräche mit Kollegen, Freunden oder in sozialen Medien über ansprechende visuelle und textbasierte Sharing-Tools zu teilen.

## Generieren eines Share-Bildes

So erstellen Sie eine ästhetische und stilisierte Vorschau Ihres Chats:
1. Klicken Sie auf die Schaltfläche **Teilen** (Export-Symbol) oben rechts auf dem Konversationsbildschirm.
2. Wählen Sie die Option **Bild generieren**.
3. Passen Sie die Ausgabe an:
   - Wählen Sie die spezifischen Nachrichten aus, die Sie einschließen möchten.
   - Wählen Sie einen Hintergrundstil (moderner Farbverlauf, Volltonfarbe oder transparent).
   - Blenden Sie den Agenten-Avatar (z. B. **May**) und das Anwendungslogo ein oder aus.
4. Klicken Sie auf **Herunterladen**, um das Bild zu speichern, oder auf **In die Zwischenablage kopieren**, um es direkt einzufügen.

## Teilen über einen öffentlichen Link

Wenn Sie die Web-Version verwenden oder einen gemeinsamen Speicherdienst konfiguriert haben:
- Wählen Sie die Option **Share-Link erstellen**.
- mAI generiert eine eindeutige, anonymisierte URL, die das Transkript Ihres Gesprächs enthält.
- Jeder mit dem Link kann das Gespräch lesen und es in seine eigene mAI-Instanz importieren.
