# Lokales Dateisystem 📂

Die **mAI**-Desktop-Anwendung bietet eine tiefe Integration mit dem Dateisystem Ihres Betriebssystems, sodass Sie Ihre lokalen Dateien sicher nutzen und analysieren können.

## Dateien lesen und analysieren

Sie können Dateien direkt an Ihre Agenten (wie **May**) zur Analyse oder Verarbeitung senden:
- **Drag and Drop**: Ziehen Sie ein Textdokument, eine PDF-Datei, eine Excel-Tabelle oder ein Bild direkt in den mAI-Chat-Bereich.
- **Anhang**: Klicken Sie auf das Büroklammer-Symbol im Chat, um Ihren Datei-Explorer zu durchsuchen.
- Dateien werden lokal geparst, und nur der extrahierte Text (oder das Bild selbst für multimodale Modelle) wird an das LLM gesendet.

## Sicherheitsgrenzen

Um Ihr System sicher zu halten:
- mAI liest niemals Dateien ohne Ihre ausdrückliche Aktion.
- Sie können in den allgemeinen Einstellungen ein dediziertes Verzeichnis angeben, um den Zugriff von mAI auf diesen Ordner zu beschränken.
