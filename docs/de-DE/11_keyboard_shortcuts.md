# Tastaturkurzbefehle ⌨️

Um Ihre Produktivität zu maximieren und schnell zu navigieren, ohne die Tastatur zu verlassen, enthält **mAI** eine Reihe konfigurierbarer Tastenkombinationen.

## Globale Tastaturkurzbefehle

- `Strg + P` (oder `Cmd + P` auf dem Mac): Öffnet das Befehlsmenü.
- `Strg + ,` (oder `Cmd + ,` auf dem Mac): Öffnet die Anwendungseinstellungen.
- `Strg + N` (oder `Cmd + N` auf dem Mac): Startet eine neue Konversation.
- `Strg + \` (oder `Cmd + \` auf dem Mac): Blendet die linke Seitenleiste ein oder aus.

## Tastenkombinationen im Chat

- `Eingabetaste`: Nachricht senden.
- `Umschalt + Eingabetaste`: Zeilenumbruch im Eingabefeld einfügen.
- `Strg + E`: Ihre letzte gesendete Nachricht bearbeiten.
- `Strg + R`: Die letzte Antwort des Agenten regenerieren.
- `Strg + Entf`: Die aktuelle Konversation löschen.
