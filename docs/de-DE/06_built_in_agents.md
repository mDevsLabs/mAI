# Integrierte Agenten 📦

**mAI** wird standardmäßig mit mehreren vorkonfigurierten Systemagenten ausgeliefert, um Sie sofort bei alltäglichen Aufgaben zu unterstützen, ohne dass eine manuelle Einrichtung erforderlich ist.

## Liste der Systemagenten

Hier sind die wichtigsten integrierten Agenten der Anwendung:

1. **May**: Ihre vielseitige Hauptassistentin, bereit für alle Schreib-, Brainstorming- und allgemeinen Hilfsaufgaben.
2. **Software-Entwickler**: Ein Agent, dessen System-Prompt auf Programmierung, das Schreiben von sauberem Code und Fehlerbehebung ausgerichtet ist.
3. **Universal-Übersetzer**: Spezialisiert auf die fließende Übersetzung zwischen Dutzenden von Sprachen unter Berücksichtigung idiomatischer Ausdrücke.
4. **Content Writer**: Optimiert für Marketing, das Entwerfen von Blog-Beiträgen und das Schreiben klarer Texte für soziale Medien.

## Nutzung integrierter Agenten

Diese Agenten sind nach der Installation sofort in Ihrer Agentenliste sichtbar. Sie können mit einem einzigen Klick eine neue Konversation mit einem von ihnen starten. Obwohl sie gegen bestimmte Systemänderungen gesperrt sind, können Sie ihre lokalen Parameter (wie das zugehörige Sprachmodell) für jeden Chat anpassen.
