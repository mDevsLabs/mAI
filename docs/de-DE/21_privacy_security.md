# Datenschutz und Sicherheit 🔒

Datensicherheit und Respekt vor Ihrer Privatsphäre sind grundlegende Prinzipien beim Design von **mAI**.

## Verschlüsselter lokaler Speicher

Alle Ihre sensiblen Daten werden lokal auf Ihrem Gerät gespeichert:
- **API-Schlüssel**: Sie werden mit branchenüblichen Algorithmen verschlüsselt, bevor sie in Ihrem lokalen Speicher gesichert werden. Ihre API-Schlüssel werden niemals an unsere Server gesendet.
- **Chat-Verlauf**: Ihre Gespräche verbleiben auf Ihrer Maschine (es sei denn, Sie aktivieren explizit die sichere Cloud-Synchronisation).

## Sichere Verbindungen

Alle ausgehenden Anfragen an APIs verwenden sichere Kommunikationsprotokolle (HTTPS / SSL), um sicherzustellen, dass die ausgetauschten Daten im Netzwerk nicht abgefangen werden können.
