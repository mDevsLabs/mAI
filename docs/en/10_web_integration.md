# Web Integration 🌐

**mAI** is designed as a cross-platform application. In addition to the desktop app, mAI is fully accessible via an optimized web version, allowing fluid access from any web browser.

## Web Version Features

The web version of mAI offers almost complete feature parity with the desktop application:
- Fluid, responsive, and reactive user interface.
- Access to all language models using your own API keys (saved locally in your browser or synchronized).
- Use of most plugins (plugins requiring local system privileges, like direct local file system access, may be restricted).

## Cloud Synchronization

If you use mAI on multiple devices, you can enable secure cloud synchronization. Your chat data, custom agents, and preferences can be saved with end-to-end encryption on your own hosting server or via compatible cloud storage solutions (such as WebDAV or synchronized database services).
