# Command Menu 🔍

The quick command menu in **mAI** is the central control hub to navigate and control the application without ever touching your mouse. It is activated instantly using the shortcut `Ctrl + P` (or `Cmd + P` on macOS).

## Core Features

Once the menu is open, you can type commands to:
- **Switch Agents**: Type the name of an agent (such as "May" or "Developer") to switch instantly.
- **Search or Create Chats**: Search your existing discussions by keywords or type `/new` to start a new chat session.
- **Access Settings**: Type `settings` to go directly to the system settings pages.
- **Change Models**: Select a different LLM for the active discussion (for example, typing "gpt" or "claude" will filter matching models).

## Keyboard Controls and Tips

- Use the `Up` and `Down` arrow keys on your keyboard to navigate the list.
- Press `Enter` to confirm a selection.
- Press `Esc` to close the command menu at any time.
