# Keyboard Shortcuts ⌨️

To maximize your productivity and navigate quickly without leaving your keyboard, **mAI** includes a range of configurable shortcuts.

## Global Navigation Shortcuts

- `Ctrl + P` (or `Cmd + P` on Mac): Opens the quick command menu.
- `Ctrl + ,` (or `Cmd + ,` on Mac): Opens the application settings.
- `Ctrl + N` (or `Cmd + N` on Mac): Starts a new conversation.
- `Ctrl + \` (or `Cmd + \` on Mac): Toggles the left sidebar.

## Chat Shortcuts

- `Enter`: Send the message.
- `Shift + Enter`: Insert a line break in the input field.
- `Ctrl + E`: Edit your last sent message.
- `Ctrl + R`: Regenerate the agent's last response.
- `Ctrl + Delete`: Delete the current discussion.

## Display Management Shortcuts

- `Alt + 1`: Switch to the Chats tab.
- `Alt + 2`: Switch to the Agents tab.
- `Alt + 3`: Switch to the Discover (Marketplace) tab.
