# Plugin System 🔌

**mAI** features a powerful plugin ecosystem that allows conversational agents to overcome the limitations of their training data by interacting with external tools and services.

## Built-in Plugins

Here are the primary plugins available by default:

1. **Web Browsing**: Allows the agent to search the internet in real time to fetch up-to-date information.
2. **Artifacts**: Enables generating, editing, and previewing code, documents, or UI designs interactively in a dedicated sidebar.
3. **Weather**: Queries weather APIs to retrieve local forecasts.
4. **Code Calculator**: Executes local scripts to solve complex calculations accurately.

## Activation and Configuration

Each agent can have its own set of enabled plugins. You can enable or disable plugins for a specific agent via the **Plugins** tab in the agent settings. Some plugins (like commercial web search) might require additional settings, such as API keys, in the general configuration.
