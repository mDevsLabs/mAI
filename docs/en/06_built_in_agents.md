# Built-in Agents 📦

**mAI** comes pre-packaged with several preconfigured system agents to assist you immediately in common tasks without requiring any manual setup.

## List of System Agents

Here are the main built-in agents in the application:

1. **May**: Your versatile core assistant, ready for all writing, brainstorming, and general help tasks.
2. **Software Developer**: An agent configured with a system prompt tailored for programming, clean code writing, and debugging.
3. **Universal Translator**: Specialized in fluent translation between dozens of languages, respecting idiomatic expressions.
4. **Content Writer**: Optimized for marketing, blog post drafting, and writing clear social media copy.

## Using Built-in Agents

These agents are immediately visible in your agents list after installation. You can start a new conversation with any of them with a single click. While they are locked against certain system modifications, you can customize their local parameters (such as the associated language model) for each discussion.
