# Supported Language Models 🧠

**mAI** is a model-agnostic interface. It supports a wide variety of commercial cloud-based and local open-source artificial intelligence providers.

## Major Cloud Providers

You can configure the following cloud providers by entering your personal API key:
- **OpenAI**: GPT-4o, GPT-4 Turbo, GPT-3.5, etc.
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku.
- **Google Gemini**: Gemini 1.5 Pro, Gemini 1.5 Flash.
- **Mistral AI**: Mistral Large, Mistral Medium, Codestral.
- **Groq**: Ultra-fast access to open-source models like Llama 3 and Mixtral.

## Local Models Integration

For users concerned with total privacy of their data, mAI fully supports local execution:
- **Ollama**: Simplified native integration. Run Ollama in the background, and mAI will automatically detect your locally installed models (such as Llama 3, Mistral, or Phi-3).
- **OpenAI-Compatible APIs**: Connect any local or remote inference server (such as LocalAI or LM Studio) by specifying a custom API base URL.
