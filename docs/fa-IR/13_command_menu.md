# منوی دستورات 🔍

منوی دستورات سریع در **mAI** هاب کنترل مرکزی برای ناوبری و کنترل برنامه بدون نیاز به لمس موس است. این منو بلافاصله با استفاده از میانبر `Ctrl + P` (یا `Cmd + P` در macOS) فعال می‌شود.

## ویژگی‌های اصلی

به محض باز شدن منو، می‌توانید دستوراتی را برای موارد زیر تایپ کنید:
- **تغییر عامل‌ها**: نام یک عامل (مانند "May" یا "Developer") را تایپ کنید تا فوراً تغییر وضعیت دهید.
- **جستجو یا ایجاد چت‌ها**: گفتگوهای موجود خود را با کلمات کلیدی جستجو کنید یا `/new` را برای شروع یک جلسه چت جدید تایپ کنید.
- **دسترسی به تنظیمات**: کلمه `settings` را تایپ کنید تا مستقیماً به صفحات تنظیمات سیستم بروید.
- **تغییر مدل‌ها**: یک مدل زبانی بزرگ (LLM) دیگر را برای گفتگوی فعال انتخاب کنید (به عنوان مثال، با تایپ کردن "gpt" یا "claude" مدل‌های مطابقت‌یافته فیلتر می‌شوند).

## کنترل‌ها و نکاتی درباره صفحه‌کلید

- از کلیدهای جهت‌نما `Up` (بالا) و `Down` (پایین) در صفحه‌کلید خود برای پیمایش در لیست استفاده کنید.
- کلید `Enter` را برای تأیید انتخاب فشار دهید.
- کلید `Esc` را فشار دهید تا در هر زمان منوی دستورات بسته شود.
