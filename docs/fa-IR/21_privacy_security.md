# حریم خصوصی و امنیت 🔒

امنیت داده‌ها و احترام به حریم خصوصی شما، اصول اساسی در طراحی **mAI** هستند.

## ذخیره‌سازی محلی رمزگذاری‌شده

تمام داده‌های حساس شما به صورت محلی روی دستگاهتان ذخیره می‌شوند:
- **کلیدهای API**: آن‌ها قبل از ذخیره شدن در فضای ذخیره‌سازی محلی شما، با استفاده از الگوریتم‌های استاندارد صنعتی رمزگذاری می‌شوند. کلیدهای API شما هرگز به سرورهای ما ارسال نمی‌شوند؛ بلکه مستقیماً از دستگاه شما به سرورهای ارائه‌دهنده مربوطه LLM ارسال می‌گردند.
- **تاریخچه گفتگوها**: گفتگوهای شما روی سیستم خودتان باقی می‌مانند (مگر اینکه به صورت صریح همگام‌سازی ابری امن را فعال کنید).

## اتصالات امن

تمام درخواست‌های خروجی به APIهای مدل زبانی از پروتکل‌های ارتباطی امن (HTTPS / SSL) استفاده می‌کنند که تضمین می‌کند داده‌های تبادل شده در شبکه قابل شنود یا رهگیری نیستند.

## روش‌های توصیه شده برای سازمان‌ها

برای بالاترین سطح امنیت در محیط‌های سازمانی:
- از مدل‌های میزبانی شده به صورت محلی از طریق **Ollama** یا یک سرور استنتاج خصوصی استفاده کنید (به این معنی که هیچ داده‌ای از شبکه سازمانی خارج نمی‌شود).
- تلمتری ناشناس را در تب تنظیمات عمومی غیرفعال کنید.
