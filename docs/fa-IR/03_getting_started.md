# شروع کار ⚡

این راهنما به شما کمک می‌کند تا **mAI** را به صورت محلی بر روی سیستم توسعه خود راه‌اندازی و اجرا کنید.

## پیش‌نیازها

مطمئن شوید که موارد زیر را نصب کرده‌اید:
- **Node.js** (نسخه ۱۸ یا بالاتر توصیه می‌شود)
- **pnpm** یا **bun** برای مدیریت بسته‌ها و فضای کاری (Workspace)

## نصب و راه‌اندازی

۱. **کلون کردن ریپازیتوری**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

۲. **نصب وابستگی‌ها**:
   با استفاده از pnpm:
   ```bash
   pnpm install
   ```
   با استفاده از bun:
   ```bash
   bun install
   ```

۳. **اجرای سرور توسعه**:
   ```bash
   pnpm dev
   # یا
   bun run dev
   ```

برنامه در مرورگر شما در آدرس `http://localhost:3010` در دسترس خواهد بود. در اولین اجرا، **May**، دستیار اصلی شما، به شما خوش‌آمد می‌گوید تا ترجیحات اولیه خود را پیکربندی کنید!
