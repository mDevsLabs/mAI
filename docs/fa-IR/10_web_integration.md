# یکپارچه‌سازی وب 🌐

برنامه **mAI** به عنوان یک اپلیکیشن چندسکویی (Cross-Platform) طراحی شده است. علاوه بر نسخه دسکتاپ، mAI از طریق یک نسخه وب بهینه‌سازی‌شده کاملاً در دسترس است که اجازه دسترسی روان را از هر مرورگر وبی می‌دهد.

## ویژگی‌های نسخه وب

نسخه وب mAI همتایی تقریباً کاملی از ویژگی‌ها را با نسخه دسکتاپ ارائه می‌دهد:
- رابط کاربری روان، واکنش‌گرا (Responsive) و حساس.
- دسترسی به تمامی مدل‌های زبانی با استفاده از کلیدهای API خودتان (که به صورت محلی در مرورگر شما ذخیره شده یا همگام‌سازی می‌شوند).
- استفاده از بیشتر پلاگین‌ها (پلاگین‌هایی که نیاز به مجوزهای سیستم محلی دارند، مانند دسترسی مستقیم به سیستم فایل محلی، ممکن است محدود باشند).

## همگام‌سازی ابری

اگر از mAI در چندین دستگاه استفاده می‌کنید، می‌توانید همگام‌سازی ابری امن را فعال کنید. داده‌های چت، عامل‌های سفارشی و ترجیحات شما را می‌توان با رمزگذاری سرتاسری (End-to-End) روی سرور میزبانی خودتان یا از طریق راهکارهای ذخیره‌سازی ابری سازگار (مانند WebDAV یا سرویس‌های پایگاه داده همگام‌سازی‌شده) ذخیره کرد.
