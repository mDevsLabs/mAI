# راهنمای توسعه 🛠️

این راهنما دستورالعمل‌های اساسی را برای توسعه‌دهندگانی که می‌خواهند در پروژه **mAI** مشارکت کنند یا کد منبع را در حالت توسعه (Development Mode) اجرا نمایند، ارائه می‌دهد.

## ساختار پروژه

پروژه mAI به صورت یک مونو‌ریپو (Monorepo) مدرن (با استفاده از pnpm و Turborepo) سازماندهی شده است:
- `apps/web`: برنامه اصلی Next.js.
- `packages/`: ماژول‌های مشترک و پیکربندی‌های قابل استفاده مجدد (کامپوننت‌های UI، تایپ‌ها، توابع کمکی یا helperها).

## اجرا در حالت توسعه (Development Mode)

۱. **پیش‌نیازها**: مطمئن شوید که Node.js (نسخه LTS توصیه می‌شود) و pnpm یا bun را نصب کرده‌اید.
۲. **نصب وابستگی‌ها**:
   ```bash
   pnpm install
   ```
۳. **شروع سرور توسعه**:
   ```bash
   pnpm run dev
   ```
   برنامه به صورت محلی در آدرس `http://localhost:3000` در دسترس خواهد بود.

## تست و آزمایش

ما از **Vitest** برای مجموعه‌های تست خود استفاده می‌کنیم:
- اجرای یک‌باره تست‌ها: `pnpm run test`
- اجرای تست‌ها در حالت تحت‌نظر (Watch Mode): `pnpm run test:watch`

لطفاً قبل از ارسال هرگونه تغییرات، مطمئن شوید که تمام تست‌ها با موفقیت پاس می‌شوند.
