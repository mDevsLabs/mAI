# مدل‌های زبانی پشتیبانی‌شده 🧠

برنامه **mAI** یک رابط مستقل از مدل (Model-Agnostic) است. این برنامه از طیف گسترده‌ای از ارائه‌دهندگان هوش مصنوعی تجاری مبتنی بر ابر و مدل‌های منبع‌باز محلی پشتیبانی می‌کند.

## ارائه‌دهندگان ابری اصلی

شما می‌توانید ارائه‌دهندگان ابری زیر را با وارد کردن کلید API شخصی خود پیکربندی کنید:
- **OpenAI**: مدل‌های GPT-4o، GPT-4 Turbo، GPT-3.5 و غیره.
- **Anthropic**: مدل‌های Claude 3.5 Sonnet، Claude 3 Opus، Claude 3 Haiku.
- **Google Gemini**: مدل‌های Gemini 1.5 Pro، Gemini 1.5 Flash.
- **Mistral AI**: مدل‌های Mistral Large، Mistral Medium، Codestral.
- **Groq**: دسترسی فوق سریع به مدل‌های منبع‌باز مانند Llama 3 و Mixtral.

## یکپارچه‌سازی مدل‌های محلی

برای کاربرانی که نگران حفظ حریم خصوصی کامل داده‌های خود هستند، mAI به طور کامل از اجرای محلی پشتیبانی می‌کند:
- **Ollama**: یکپارچه‌سازی بومی و ساده‌شده. کافیست Ollama را در پس‌زمینه اجرا کنید تا mAI به طور خودکار مدل‌های نصب‌شده محلی شما (مانند Llama 3، Mistral یا Phi-3) را شناسایی کند.
- **APIهای سازگار با OpenAI**: هر سرور استنتاج محلی یا راه دوری (مانند LocalAI یا LM Studio) را با تعیین یک URL پایه API سفارشی متصل کنید.
