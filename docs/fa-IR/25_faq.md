# سوالات متداول ❓

در اینجا پاسخ متداول‌ترین سوالات درباره پیکربندی و استفاده از **mAI** آورده شده است.

## استفاده عمومی

### آیا می‌توانم بدون اتصال به اینترنت از mAI استفاده کنم؟
بله، به شرطی که از مدل‌های محلی استفاده کنید که از طریق سرویس‌هایی مانند **Ollama** یا LocalAI اجرا می‌شوند. اگر از مدل‌های ابری (مانند OpenAI یا Anthropic) استفاده می‌کنید، یک اتصال اینترنت فعال برای برقراری ارتباط با سرورهای آن‌ها مورد نیاز است.

### چگونه کلیدهای API خود را به‌روزرسانی کنم؟
به تنظیمات (`Ctrl + ,`) بروید و تب **مدل‌های زبانی (Language Models)** را باز کنید. از آنجا می‌توانید کلیدهای API را برای هر یک از ارائه‌دهندگان پشتیبانی‌شده اضافه، تغییر یا حذف کنید.

## شخصی‌سازی عامل‌ها

### May کیست؟
**May** دستیار پیش‌فرض و هوشمند اصلی در mAI است. او بسیار همه‌کاره است و به گونه‌ای پیکربندی شده که از اولین راه‌اندازی به شما کمک کند.

### آیا عامل‌های من با سایر کاربران به اشتراک گذاشته می‌شوند؟
خیر. به صورت پیش‌فرض، تمام عامل‌های سفارشی که ایجاد می‌کنید به صورت محلی ذخیره می‌شوند و روی دستگاه شما خصوصی باقی می‌مانند. با این حال، می‌توانید آن‌ها را به صورت دستی اکسپورت (صادر) کنید تا با دیگران به اشتراک بگذارید.

## هزینه و صورت‌حساب

### آیا استفاده از mAI رایگان است؟
خود برنامه mAI کاملاً رایگان و منبع‌باز است. با این حال، استفاده از مدل‌های ابری تجاری (مانند OpenAI یا Anthropic) مستلزم ارائه کلیدهای API خودتان است. شما فقط هزینه استفاده واقعی از API خود را مستقیماً به ارائه‌دهندگان مدل پرداخت خواهید کرد.
