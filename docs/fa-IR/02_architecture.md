# معماری mAI 🏗️

برنامه **mAI** بر روی یک معماری مدرن ساخته شده است که برای کارایی، توسعه‌پذیری و نگهداری آسان طراحی گردیده است. ساختار این پروژه به صورت مونو‌ریپو (Monorepo) سازماندهی شده است.

## فناوری‌های کلیدی

- **فریم‌ورک اصلی**: Next.js (با App Router) برای رندرینگ و مسیریابی.
- **مدیریت وضعیت**: Zustand برای کنترل وضعیت سبک و واکنش‌گرا در سمت کلاینت.
- **طراحی و رابط کاربری**: کامپوننت‌های Ant Design که با `antd-style` و `@lobehub/ui` استایل‌دهی شده‌اند.
- **پایگاه داده**: SQLite به صورت محلی (یا PostgreSQL در محیط عملیاتی) که توسط Drizzle ORM مدیریت می‌شود.

## ساختار مونو‌ریپو (Monorepo)

کد پایه به بسته‌های قابل استفاده مجدد در دایرکتوری `packages/` تقسیم شده است:
- `packages/const`: ثابت‌ها و پیکربندی‌های سراسری.
- `packages/builtin-agents`: عامل‌های پیش‌فرض سیستم (از جمله May).
- `packages/database`: مدل‌های داده، اسکیماهای Drizzle و میگریشن‌ها.
- `packages/types`: تعاریف تایپ‌های مشترک TypeScript.

این جداسازی، منطق اصلی کسب‌وکار (مانند اجرای درخواست‌های API مدل یا مدیریت پلاگین‌ها) را از رابط کاربری وب اصلی که در دایرکتوری `src/` قرار دارد، متمایز می‌کند.
