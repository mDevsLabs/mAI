# رابط برنامه‌نویسی (API) و وب‌هوک‌ها 🔗

برنامه **mAI** رابط‌های برنامه‌نویسی را ارائه می‌دهد تا امکان یکپارچه‌سازی برنامه‌نویسی با برنامه‌های خودتان، سرویس‌های وب یا اسکریپت‌های اتوماسیون فراهم شود.

## رابط برنامه‌نویسی محلی (API محلی بر بستر HTTP)

نسخه دسکتاپ mAI می‌تواند یک سرور وب محلی امن را روی پورت تعیین‌شده اجرا کند. این سرور چندین اندپوینت (Endpoint) را در دسترس قرار می‌دهد:
- `POST /api/chat`: پیامی را به یک عامل خاص ارسال می‌کند و پاسخی ساختاریافته بازمی‌گرداند (از فرمت‌های استاندارد یا استریمینگ پشتیبانی می‌کند).
- `GET /api/agents`: لیستی از تمام عامل‌های در دسترس و پیکربندی‌های آن‌ها را ارائه می‌دهد.
- `POST /api/agents`: امکان ایجاد یک عامل جدید را به صورت برنامه‌نویسی فراهم می‌سازد.

## یکپارچه‌سازی وب‌هوک‌ها (Webhooks)

شما می‌توانید وب‌هوک‌های خروجی را در تنظیمات پیشرفته mAI پیکربندی کنید. یک وب‌هوک با وقوع رویدادهای خاص، به طور خودکار یک پی‌لود JSON را به یک URL خارجی ارسال (Post) می‌کند:
- **تکمیل پیام گفتگو (Chat message completed)**: به محض اینکه عامل تولید پاسخ را به پایان رساند، راه انداخته می‌شود.
- **ایجاد عامل (Agent created)**: هنگام اضافه شدن یک پروفایل عامل جدید فعال می‌گردد.
- **خطای سیستم (System error)**: در طول بروز مشکلات اعتبارسنجی کلید API یا قطع خدمات ارائه‌دهنده LLM رخ می‌دهد.
