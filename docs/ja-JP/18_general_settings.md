# 一般設定 ⚙️

**mAI** の一般設定では、アプリケーションをあなたの美的、言語的、および機能的な好みに合わせて調整できます。

## ユーザーインターフェースとテーマ

- **テーマ**：ライトモード、ダークモードを選択するか、アプリケーションがコンピューターのシステム設定に従うようにします。
- **アクセントカラー**：グラフィカルUIのアクティブなボタンや主要なインタラクティブ要素の色をカスタマイズします。
- **インターフェース言語**：アプリケーションの言語（フランス語、英語など）を選択します。
- **フォントサイズ**：読みやすさを最適化するためにテキストサイズを調整します。

## データ管理

- **データの出力**：すべての会話、カスタムエージェント、および設定を1つのJSONアーカイブファイルにエクスポートします。
- **データの入力**：以前のバックアップを復元するか、外部のエージェント設定をインポートします。
- **アプリケーションのリセット**：すべてのローカルデータ（履歴、APIキー、設定）を完全に消去して、アプリケーションを初期状態に戻します。
