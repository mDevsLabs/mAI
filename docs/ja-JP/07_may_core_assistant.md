# May：コアアシスタント 🤖

**May** は、**mAI** 体験の中心にあるデフォルトのアシスタントです。単なるチャットエージェントにとどまらず、ガイドとして、またあなたとシステムの強力な機能との間の架け橋として機能します。

## Mayの役割と機能

Mayは日々のヘルパーとなるよう設計されています。主な役割は次のとおりです：
- **歓迎とオンボーディング**：最初のAPIキーの設定や、ユーザーインターフェースの探索をサポートします。
- **技術的および一般的なサポート**：アプリケーション自体に関する質問や、一般文化、科学、芸術、技術のあらゆるトピックに対応します。
- **開発支援**：複数のプログラミング言語のコードを理解し、記述します。
- **プラグインの統合**：デフォルトで、Webブラウジングやアーティファクトシステムなどの有効なプラグインにアクセスして、回答を充実させることができます。

## Mayのカスタマイズ

Mayはベースとなるアシスタントですが、設定で動作を変更できます：
- 基本のシステムプロンプトを調整する。
- デフォルトの言語モデルを変更する。
- チャットセッション用に特定のプラグインを有効または無効にする。
