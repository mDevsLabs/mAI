# チャットとレンダリング機能 💬

**mAI** の対話インターフェースは、視覚的な機能と正確なコントロールオプションを詰め込んだ、流動的でインタラクティブな体験を提供するよう設計されています。

## コアチャット機能

- **インライン編集**: いつでも過去のメッセージを編集できます。mAIは、そのターンから自動的に会話を再実行します。
- **豊かなMarkdownレンダリング**: mAIは、テーブル、LaTeX数式、構文ハイライト付きコードブロック、インタラクティブなプレビューペインなどの完全なMarkdownフォーマットを処理します。
- **クイックアクション**: 各メッセージバブルにはコンテキストメニューがあり、未加工コンテンツのコピー、AIの応答の再生成、編集、または履歴からのメッセージの削除を行えます。
- **履歴追跡**: チャットセッションはローカルのSQLiteデータベースにアーカイブされ、過去の会話を即座に検索して復元できます。
