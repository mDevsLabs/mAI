# エージェントのカスタマイズ ⚙️

**mAI** でエージェントを微調整することで、その動作、応答スタイル、およびアクセスできるツールを深く変更できます。

## 基本設定

各エージェントについて、以下を設定できます：
- **システムプロンプト (System Prompt)**：エージェントの個性、ルール、および役割を定義する基本的な指示。これはエージェントの応答に最も影響を与える設定です。
- **言語モデル**：使用するデフォルトのLLMを選択します（例：GPT-4o、Claude 3.5 Sonnet、Llama 3）。

## 詳細設定

生成時のハイパーパラメータを変更することもできます：
- **温度 (Temperature)**：応答の創造性を制御します。温度が低い（例：0.2）と、より事実に基づいた決定論的な回答が生成されます。温度が高い（例：0.9）と、創造性が促進されます。
- **Top P**：応答の多様性を制御するもう1つの方法。
- **存在 / 頻度ペナルティ (Presence / Frequency Penalty)**：エージェントが同じ言葉を繰り返したり、議論されているトピックから逸脱したりするのを防いだり促したりします。
- **最大トークン数 (Max Tokens)**：生成される応答の最大長を制限します。
