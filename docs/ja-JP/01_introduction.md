# mAIの紹介 🚀

大容量言語モデル（LLM）との対話を簡素化し、生産性を最大化するために設計された次世代の人工知能アプリケーションである **mAI** の公式ドキュメントへようこそ。

## mAIとは？

**mAI** は、モダンで高速、かつ完全にカスタマイズ可能なチャットフレームワークおよび対話型エージェントプラットフォームです。ユーザーは、洗練された流動的なユーザーインターフェース内で、複数のAIプロバイダー（OpenAI GPT、Claude、Gemini、またはOllamaを介したローカルモデルなど）へのアクセスを一元化できます。

## May：あなたのコアアシスタント 🤖

mAIの中心にいるのは、デフォルトのインテリジェントアシスタントである **May**（旧Lobe AI）です。高度な汎用性を備えたMayは、最初の起動時からあなたをガイドし、複雑な質問に答え、プログラミングを支援し、思考を構造化するためのサポートを行います。Mayは持続的なメモリと外部ツール（プラグイン）を使用する機能を使用して、あなたのニーズに適応します。

## なぜmAIを選ぶのか？

- **マルチモデルのサポート**：異なるAIプロバイダー間を即座に切り替えることができます。
- **プライバシー第一**：APIキーとチャット履歴は、ローカルまたは独自のインフラストラクチャに安全に保存されます。
- **豊富な機能**：統合されたスクリーンキャプチャ、ビジュアルなチャット共有、プラグインのサポートなど。
