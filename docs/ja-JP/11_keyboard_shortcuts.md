# キーボードショートカット ⌨️

生産性を最大化し、キーボードから手を離さずに素早く移動できるようにするため、**mAI** にはカスタマイズ可能なさまざまなショートカットが含まれています。

## グローバルナビゲーションのショートカット

- `Ctrl + P`（Macでは `Cmd + P`）：クイックコマンドメニューを開きます。
- `Ctrl + ,`（Macでは `Cmd + ,`）：アプリケーション設定を開きます。
- `Ctrl + N`（Macでは `Cmd + N`）：新しい会話を開始します。
- `Ctrl + \`（Macでは `Cmd + \`）：左サイドバーの表示を切り替えます。

## チャットのショートカット

- `Enter`：メッセージを送信します。
- `Shift + Enter`：入力フィールド内で改行を挿入します。
- `Ctrl + E`：最後に送信したメッセージを編集します。
- `Ctrl + R`：エージェントの最後の応答を再生成します。
- `Ctrl + Delete`：現在の会話を削除します。

## 表示管理ショートカット

- `Alt + 1`：チャットタブに切り替えます。
- `Alt + 2`：エージェントタブに切り替えます。
- `Alt + 3`：発見（マーケットプレイス）タブに切り替えます。
