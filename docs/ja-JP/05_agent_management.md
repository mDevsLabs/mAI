# エージェント管理 👥

**mAI** では、特定のニーズやユースケースに対応するための専用エージェントを作成および構成できます。エージェントの管理は、グラフィカルユーザーインターフェースから直接、または設定ファイルを介して行われます。

## 新しいエージェントの作成

mAIでエージェントを作成するには：
1. 左側のサイドバーに移動し、**エージェント** ボタンをクリックします。
2. リストの上部にある **エージェントの作成** をクリックします。
3. 基本情報を入力します：
   - **名前**: エージェントの識別子。
   - **アバター**: アイコン、絵文字を選択するか、画像をアップロードします。
   - **説明**: エージェントの役割の簡単な概要。

## 既存のエージェントの編集

エージェントリスト内の関連する設定アイコンをクリックして、カスタムエージェントを編集できます。システムプロンプトの定義、デフォルトの言語モデルの選択、プラグインの管理などの詳細設定にアクセスできます。

## エージェントの削除

カスタムエージェントが不要になった場合は、エージェントの設定オプションから永久に削除できます。このエージェントに関連付けられている既存のディスカッションは削除されませんが、エージェント自体は新しい会話に使用できなくなります。
