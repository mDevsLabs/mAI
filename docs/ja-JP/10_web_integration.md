# Webの統合 🌐

**mAI** はクロスプラットフォームアプリケーションとして設計されています。デスクトップアプリに加えて、最適化されたWebバージョンを介して完全にアクセス可能であり、あらゆるWebブラウザからスムーズにアクセスできます。

## Webバージョンの機能

mAIのWebバージョンは、デスクトップアプリケーションとほぼ完全な機能同等性を提供します：
- スムーズでレスポンシブ、かつ反応の良いユーザーインターフェース。
- 独自のAPIキーを使用したすべての言語モデルへのアクセス（ブラウザにローカルに保存されるか同期されます）。
- ほとんどのプラグインの使用（直接のローカルファイルシステムアクセスなど、ローカルシステム特権を必要とするプラグインは制限される場合があります）。

## クラウド同期

複数のデバイスでmAIを使用する場合は、安全なクラウド同期を有効にできます。チャットデータ、カスタムエージェント、および設定は、独自のホスティングサーバーまたは互換性のあるクラウドストレージソリューション（WebDAVや同期型データベースサービスなど）でエンドツーエンドの暗号化を使用して保存できます。
