# オンボーディングガイド 🚀

**mAI** へようこそ！このガイドは、新規ユーザーがアプリケーションでの最初のステップを段階的に進めることができるよう設計されています。

## ステップ 1：Mayからのウェルカムメッセージ

アプリケーションを初めて起動すると、デフォルトのバーチャルアシスタントである **May** が自動的に開きます。彼女は最初の設定手順を案内し、アプリに関する質問に答えます。

## ステップ 2：APIキーの設定

mAIを機能させるには、少なくとも1つの言語モデルプロバイダーを構成する必要があります：
1. Mayが案内する設定ボタンをクリックするか、設定を開きます（`Ctrl + ,`）。
2. **言語モデル** タブを選択します。
3. 選択したプロバイダー（OpenAI、Anthropic Claude、Google Geminiなど）のAPIキーを入力するか、Ollamaなどのローカルプロバイダーを有効にします。
4. 保存して、デバイス上にローカルにキーを安全に保存します。

## ステップ 3：最初の会話

1. 左側のサイドバーにある **新しいチャット** をクリックします。
2. 画面下部のテキストエリアにメッセージを入力します。
3. **Enter** キーを押して、Mayとの最初の会話を開始します！

## ステップ 4：エージェントの探索と作成

**エージェント** タブに移動して、事前構成されたプロファイルを探索するか、**エージェントの作成** をクリックして特定のニーズに合わせたバーチャルコンパニオンを設計します。
