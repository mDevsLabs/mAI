# チャットの共有 📤

**mAI** では、美しいビジュアルおよびテキストベースの共有ツールを使用して、同僚や友人、またはソーシャルメディアで会話を簡単に共有できます。

## 共有画像の生成

チャットの美しくスタイリッシュなプレビューを作成するには：
1. ディスカッション画面の右上にある **共有** ボタン（エクスポートアイコン）をクリックします。
2. **画像の生成** オプションを選択します。
3. 出力をカスタマイズします：
   - 含めたい特定のメッセージを選択します。
   - 背景スタイルを選択します（モダンなグラデーション、単色、または透明）。
   - エージェントのアバター（**May** など）とアプリケーションのロゴの表示/非表示を切り替えます。
4. **ダウンロード** をクリックして画像を保存するか、**クリップボードにコピー** して別のアプリケーションに直接貼り付けます。

## 公開リンク（共有リンク）による共有

Webバージョンを使用している場合、または共有ストレージサービスを設定している場合：
- **共有リンクの作成** オプションを選択します。
- mAIは、会話の文字起こしを含む一意の匿名化されたURLを生成します。
- リンクを知っている人は誰でも、読み取り専用ビューアで会話を読むことができ、それを独自のmAIインスタンスにインポートしてディスカッションを継続できます。
