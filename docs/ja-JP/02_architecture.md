# mAIのアーキテクチャ 🏗️

**mAI** アプリケーションは、パフォーマンス、拡張性、および保守性のために設計されたモダンなアーキテクチャに基づいています。モノレポとして構成されています。

## 主なテクノロジー

- **コアフレームワーク**: レンダリングとルーティングのためのNext.js（App Router採用）。
- **状態管理**: 軽量で反応的なクライアント側の状態制御のためのZustand。
- **デザイン＆UI**: `antd-style` と `@lobehub/ui` でスタイリングされたAnt Designコンポーネント。
- **データベース**: Drizzle ORMによって管理されるローカルのSQLite（または本番環境のPostgreSQL）。

## モノレポの構造

コードベースは、`packages/` ディレクトリ配下の再利用可能なパッケージに分割されています。
- `packages/const`: グローバルな定数と設定。
- `packages/builtin-agents`: デフォルトのシステムエージェント（Mayを含む）。
- `packages/database`: データモデル、Drizzleスキーマ、およびマイグレーション。
- `packages/types`: 共有されるTypeScriptの型定義。

この分離により、`src/` にあるメインのWebユーザーインターフェースから、コアのビジネスロジック（モデルのAPIリクエストの実行やプラグインの管理など）が隔離されます。
