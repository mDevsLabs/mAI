# ローカルファイルシステム 📂

**mAI** デスクトップアプリケーションは、オペレーティングシステムのファイルシステムと深く統合されており、ローカルファイルに安全にアクセスして分析することができます。

## ファイルの読み込みと分析

分析や処理のために、ファイルをエージェント（**May** など）に直接送信できます：
- **ドラッグ＆ドロップ**：テキストドキュメント、PDFファイル、Excelシート、または画像をmAIチャット領域に直接ドラッグします。
- **添付ファイル**：チャット内のクリップアイコンをクリックして、ファイルエクスプローラーを参照します。
- ファイルはローカルで解析および処理され、抽出されたテキスト（またはマルチモーダルモデル用の画像自体）のみがLLMに送信されます。

## セキュリティ境界

システムの安全を維持するために：
- mAIは、あなたの明示的なアクション（ドラッグ＆ドロップまたは手動のファイル選択）なしにファイルを読み取ることはありません。
- 一般設定で専用のディレクトリを指定し、mAIのアクセスをそのフォルダーのみに制限することができます。
