# ユーザーメモリ 🧠

**mAI** のユーザーメモリ機能を使用すると、エージェント、特に **May** が、異なる会話をまたいであなたに関する重要な事実を記憶できます。これにより、ユーザー体験が大幅にパーソナライズされ、シームレスになります。

## メモリの仕組み

mAIと対話するにつれて、アプリケーションは自動的に関連するコンテキスト情報を抽出し、ローカルのメモリデータベースに保存します。これには以下が含まれます：
- あなたの言語設定やライティングスタイル。
- 現在進行中のプロジェクト。
- 開発の好み（好みのフレームワーク、コードスタイル）。
- 共有することを選択した個人情報（名前や役職など）。

## メモリの管理

mAIによって保存された情報は完全に制御できます：
1. アプリケーションの一般設定に移動します。
2. **メモリ** タブを開きます。
3. カードとして保存された事実のリストが表示されます。
4. 各エントリを手動で編集するか、メモリデータベースから事実を完全に削除してエージェントに忘れさせることができます。
