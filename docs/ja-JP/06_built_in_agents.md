# 組み込みエージェント 📦

**mAI** には、手動でのセットアップを必要とせず、一般的なタスクですぐにサポートを提供できるよう、事前に構成された複数のシステムエージェントが同梱されています。

## システムエージェントの一覧

アプリケーション内の主な組み込みエージェントは次のとおりです：

1. **May**: あらゆる執筆、ブレインストーミング、および一般的なヘルプタスクに対応できる、多才なコアアシスタント。
2. **Software Developer**: プログラミング、クリーンコードの記述、およびデバッグに合わせたシステムプロンプトで構成されたエージェント。
3. **Universal Translator**: 慣用表現を尊重し、数十の言語間での流暢な翻訳に特化したエージェント。
4. **Content Writer**: マーケティング、ブログ投稿の下書き、および明確なソーシャルメディア向けコピーの作成に最適化されたエージェント。

## 組み込みエージェントの使用

これらのエージェントは、インストール直後にエージェントリストに表示されます。ワンクリックでいずれかのエージェントと新しい会話を開始できます。特定のシステム変更に対してはロックされていますが、ディスカッションごとにローカルパラメータ（関連する言語モデルなど）をカスタマイズできます。
