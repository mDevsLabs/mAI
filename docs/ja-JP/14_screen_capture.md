# スクリーンキャプチャ 📸

**mAI** アプリケーションには直感的なスクリーンキャプチャツールが含まれており、画面の一部を素早く撮影してディスカッションに直接送信し、エージェントによる視覚的な分析を行うことができます。

## キャプチャツールの使用方法

スクリーンショットを撮るには：
1. チャット入力バーにある **カメラ / キャプチャ** アイコンをクリックするか、専用のショートカットを使用します。
2. キャプチャしたい領域またはウィンドウを選択できるように、一時的にUIが非表示になります。
3. 選択すると、画像が自動的にチャット入力ボックスに添付されます。

## Mayによる視覚的分析

mAIで構成された言語モデルのマルチモーダル機能を活用して、キャプチャされた画像について質問できます：
- 「このエラーメッセージを説明してください。」
- 「この画像内のテキストを翻訳してください。」
- 「このレイアウトを確認し、デザインの改善案を提案してください。」
- 「このUIモックアップに対応するコードを書いてください。」
