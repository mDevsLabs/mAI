# コマンドメニュー 🔍

**mAI** のクイックコマンドメニューは、マウスに触れることなくアプリケーション内を移動して制御するための中心的なコントロールハブです。ショートカットキー `Ctrl + P`（macOSでは `Cmd + P`）を使用して即座にアクティブになります。

## 主な機能

メニューが開いたら、コマンドを入力して以下を実行できます：
- **エージェントの切り替え**：エージェント名（「May」や「Developer」など）を入力すると、即座に切り替わります。
- **チャットの検索または作成**：既存のチャットをキーワードで検索するか、`/new` と入力して新しいチャットセッションを開始します。
- **設定へのアクセス**：`settings` と入力すると、システム設定ページに直接移動します。
- **モデルの変更**：アクティブな対話に使用するLLMを選択します（たとえば、「gpt」や「claude」と入力すると、一致するモデルがフィルタリングされます）。

## キーボード操作とヒント

- キーボードの `Up` および `Down` 矢印キーを使用して、リストを移動します。
- `Enter` キーを押して選択を確定します。
- いつでも `Esc` キーを押してコマンドメニューを閉じます。
