# テレメトリ 📊

**mAI** では、透明性とプライバシーの尊重が最優先事項です。このドキュメントでは、アプリケーションにおけるテレメトリの仕組みと、収集される可能性のあるデータについて詳しく説明します。

## 匿名データの収集

アプリケーションの安定性を向上させ、インターフェースのバグを特定するために、mAIには匿名テレメトリが含まれています。オプトインした場合、以下の情報が完全に匿名化された形式で送信されます：
- クラッシュレポートと未処理のJavaScript例外。
- グローバルな使用統計（たとえば、作成されたカスタムエージェントの数、最も頻繁に選択される言語モデル、またはプラグインの有効化率）。
- 基本的なシステム情報（アプリケーションのバージョン、オペレーティングシステム、画面解像度）。

## 収集しないもの

mAIのテレメトリは、お客様のプライバシーを厳格に尊重します。私たちは以下の情報を**決して**収集、閲覧、または保存しません：
- メッセージや会話の内容。
- カスタムシステムプロンプトやエージェントの構成。
- APIキーや資格情報の詳細。
- 個人を特定できる情報（名前、メールアドレスなど）。

## テレメトリの有効化と無効化

いつでもmAIの改善に参加するか、オプトアウトするかを選択できます：
- アプリケーションの **設定** に移動します。
- **バージョン情報 / テレメトリ** タブをクリックします。
- **匿名のクラッシュレポートと使用統計の送信を許可する** オプションを切り替えます。
