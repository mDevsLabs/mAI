# 開発ガイド 🛠️

このガイドは、**mAI** プロジェクトに貢献したい、または開発モードでソースコードを実行したい開発者向けの基本的な手順を提供します。

## プロジェクトの構造

mAIは、モダンなモノレポ（pnpmおよびTurborepoを使用）として構造化されています：
- `apps/web`: メインのNext.jsアプリケーション。
- `packages/`: 共有モジュールと再利用可能な設定（UIコンポーネント、型定義、ヘルパーなど）。

## 開発モードでの実行

1. **前提条件**: Node.js（LTSバージョン推奨）と pnpm または bun がインストールされていることを確認してください。
2. **依存関係のインストール**:
   ```bash
   pnpm install
   ```
3. **開発サーバーの起動**:
   ```bash
   pnpm run dev
   ```
   アプリケーションはローカルの `http://localhost:3000` でアクセス可能になります。

## テスト

テストスイートには **Vitest** を使用します：
- テストの単発実行: `pnpm run test`
- ウォッチモードでのテスト実行: `pnpm run test:watch`

変更を送信する前に、すべてのテストが正常にパスすることを確認してください。
