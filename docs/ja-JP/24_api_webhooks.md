# APIとウェブフック 🔗

**mAI** は、独自のアプリケーション、Webサービス、または自動化スクリプトとの外部統合を可能にするためのプログラムインターフェースを提供します。

## ローカルAPI (HTTP)

mAIデスクトップアプリは、指定されたポートで安全なローカルWebサーバーを実行できます。このサーバーはいくつかのエンドポイントを公開しています：
- `POST /api/chat`：特定のエージェントにメッセージを送信し、構造化された応答を返します（標準またはストリーミング形式をサポート）。
- `GET /api/agents`：利用可能なすべてのエージェントとその設定を一覧表示します。
- `POST /api/agents`：プログラムで新しいエージェントを作成できるようにします。

## ウェブフックの統合

mAIの詳細設定で送信ウェブフックを構成できます。ウェブフックは、特定のイベントが発生するたびに、外部のURLにJSONペイロードを自動的に送信します：
- **チャットメッセージの完了**：エージェントが応答の生成を完了するとすぐにトリガーされます。
- **エージェントの作成**：新しいエージェントプロファイルが追加されたときにトリガーされます。
- **システムエラー**：APIキーの検証問題やLLMプロバイダーの停止中にトリガーされます。
