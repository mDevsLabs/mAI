# プラグインシステム 🔌

**mAI** は、対話型エージェントが外部のツールやサービスと対話することで、トレーニングデータの限界を克服できるようにする強力なプラグインエコシステムを備えています。

## 組み込みプラグイン

デフォルトで利用可能な主なプラグインは次のとおりです：

1. **Webブラウジング (Web Browsing)**：エージェントがインターネットをリアルタイムで検索し、最新の情報を取得できるようにします。
2. **アーティファクト (Artifacts)**：コード、ドキュメント、またはUIデザインを専用のサイドバーでインタラクティブに生成、編集、プレビューできるようにします。
3. **天気 (Weather)**：気象APIに問い合わせて、ローカルの天気予報を取得します。
4. **コード計算機 (Code Calculator)**：ローカルスクリプトを実行して、複雑な計算を正確に解決します。

## 有効化と構成

各エージェントは、それぞれ有効化された独自のプラグインセットを持つことができます。エージェント設定の **プラグイン** タブから、特定のエージェントのプラグインを有効または無効にできます。一部のプラグイン（商用Web検索など）は、一般設定でAPIキーなどの追加設定が必要になる場合があります。
