# プライバシーとセキュリティ 🔒

データセキュリティとプライバシーの尊重は、**mAI** の設計における基本原則です。

## 暗号化されたローカルストレージ

すべての機密データは、デバイスにローカルに保存されます：
- **APIキー**：ローカルストレージに保存される前に、業界標準のアルゴリズムを使用して暗号化されます。APIキーが当社のサーバーに送信されることはありません。デバイスから各LLMプロバイダーのサーバーに直接送信されます。
- **チャット履歴**：会話はマシン上に残ります（安全なクラウド同期を明示的に有効にしている場合を除く）。

## セキュアな接続

言語モデルAPIへのすべての送信リクエストは、安全な通信プロトコル（HTTPS / SSL）を使用し、交換されるデータがネットワーク上で傍受されないようにします。

## 推奨されるエンタープライズプラクティス

企業環境で最高レベルのセキュリティを確保するためには：
- **Ollama** またはプライベート推論サーバーを介してローカルにホストされたモデルを使用します（企業ネットワークからデータが外部に出ないようにするため）。
- 一般設定タブで匿名のテレメトリ送信を無効にします。
