# Integracja z Siecią Web 🌐

**mAI** został zaprojektowany jako aplikacja wieloplatformowa. Oprócz aplikacji desktopowej, mAI jest w pełni dostępny poprzez zoptymalizowaną wersję internetową, co umożliwia płynny dostęp z dowolnej przeglądarki internetowej.

## Funkcje Wersji Internetowej

Wersja internetowa mAI oferuje niemal całkowitą spójność funkcji z aplikacją desktopową:
- Płynny, responsywny i reaktywny interfejs użytkownika.
- Dostęp do wszystkich modeli językowych przy użyciu własnych kluczy API (zapisanych lokalnie w przeglądarce lub zsynchronizowanych).
- Korzystanie z większości wtyczek (wtyczki wymagające lokalnych uprawnień systemowych, takie jak bezpośredni dostęp do lokalnego systemu plików, mogą być ograniczone).

## Synchronizacja z Chmurą

Jeśli używasz mAI na wielu urządzeniach, możesz włączyć bezpieczną synchronizację z chmurą. Twoje dane czatu, niestandardowi agenci i preferencje mogą być zapisywane z szyfrowaniem end-to-end na Twoim własnym serwerze hostingowym lub za pomocą kompatybilnych rozwiązań do przechowywania w chmurze (takich jak WebDAV lub zsynchronizowane usługi baz danych).
