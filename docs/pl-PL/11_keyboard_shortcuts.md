# Skróty Klawiszowe ⌨️

Aby zmaksymalizować produktywność i szybko nawigować bez oderwania rąk od klawiatury, **mAI** zawiera zestaw konfigurowalnych skrótów.

## Globalne Skróty Nawigacyjne

- `Ctrl + P` (lub `Cmd + P` na Macu): Otwiera szybkie menu poleceń.
- `Ctrl + ,` (lub `Cmd + ,` na Macu): Otwiera ustawienia aplikacji.
- `Ctrl + N` (lub `Cmd + N` na Macu): Rozpoczyna nową rozmowę.
- `Ctrl + \` (lub `Cmd + \` na Macu): Przełącza lewy pasek boczny.

## Skróty Czatu

- `Enter`: Wyślij wiadomość.
- `Shift + Enter`: Wstawia podział linii w polu wprowadzania tekstu.
- `Ctrl + E`: Edytuj swoją ostatnio wysłaną wiadomość.
- `Ctrl + R`: Ponownie wygeneruj ostatnią odpowiedź agenta.
- `Ctrl + Delete`: Usuń bieżącą dyskusję.

## Skróty Zarządzania Wyświetlaniem

- `Alt + 1`: Przełącz na kartę Czaty (Chats).
- `Alt + 2`: Przełącz na kartę Agenci (Agents).
- `Alt + 3`: Przełącz na kartę Odkrywaj (Discover / Marketplace).
