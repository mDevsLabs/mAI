# Pamięć Użytkownika 🧠

Pamięć użytkownika w **mAI** pozwala Twoim agentom, a w szczególności **May**, zapamiętywać kluczowe fakty o Tobie w różnych rozmowach. Dzięki temu doświadczenie użytkownika staje się znacznie bardziej spersonalizowane i płynne.

## Jak Działa Pamięć

Gdy wchodzisz w interakcję z mAI, aplikacja automatycznie wyodrębnia istotne informacje kontekstowe i przechowuje je w lokalnej bazie danych pamięci. Może to obejmować:
- Twoje preferencje językowe lub styl pisania.
- Projekty, nad którymi obecnie pracujesz.
- Twoje preferencje programistyczne (preferowane frameworki, style kodu).
- Dane osobowe, które zdecydujesz się udostępnić (takie jak Twoje imię lub rola zawodowa).

## Zarządzanie Swoją Pamięcią

Masz pełną kontrolę nad informacjami przechowywanymi przez mAI:
1. Przejdź do ogólnych ustawień aplikacji.
2. Otwórz kartę **Pamięć (Memory)**.
3. Zobaczysz listę zapisanych faktów w postaci kart.
4. Możesz ręcznie edytować dowolny wpis lub trwale usunąć fakt z bazy danych pamięci, aby agent o nim zapomniał.
