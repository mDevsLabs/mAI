# Telemetria 📊

W **mAI** przejrzystość i szacunek dla Twojej prywatności są naszymi najważniejszymi priorytetami. Ten dokument szczegółowo opisuje, jak działa telemetria w aplikacji i jakie dane mogą być zbierane.

## Anonimowe Zbieranie Danych

Aby pomóc nam ulepszać stabilność aplikacji i identyfikować błędy interfejsu, mAI zawiera anonimową telemetrię. Jeśli wyrazisz zgodę, następujące informacje będą przesyłane w pełni zanonimizowanym formacie:
- Raporty o awariach i nieobsługiwane wyjątki JavaScript.
- Globalne statystyki użytkowania (na przykład liczba utworzonych niestandardowych agentów, najczęściej wybierane modele językowe lub wskaźniki aktywacji wtyczek).
- Podstawowe informacje o systemie (wersja aplikacji, system operacyjny, rozdzielczość ekranu).

## Czego NIGDY nie Zbieramy

Telemetria w mAI ściśle szanuje Twoją prywatność. **Nigdy** nie zbieramy, nie czytamy ani nie przechowujemy:
- Treści Twoich wiadomości lub rozmów.
- Twoich niestandardowych promptów systemowych lub konfiguracji agentów.
- Twoich kluczy API ani szczegółów poświadczeń.
- Wszelkich danych umożliwiających identyfikację tożsamości (imiona i nazwiska, adresy e-mail itp.).

## Włączanie lub Wyłączanie Telemetrii

Możesz zdecydować o udziale w ulepszaniu mAI lub zrezygnować z niego w dowolnym momencie:
1. Przejdź do **Ustawień** aplikacji.
2. Kliknij kartę **O aplikacji / Telemetria (About / Telemetry)**.
3. Przełącz opcję **Zezwalaj na anonimowe raporty o awariach i statystyki użytkowania**.
