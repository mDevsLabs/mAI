# System Wtyczek 🔌

**mAI** posiada potężny ekosystem wtyczek, który pozwala agentom konwersacyjnym na pokonywanie ograniczeń ich danych treningowych poprzez interakcję z zewnętrznymi narzędziami i usługami.

## Wbudowane Wtyczki

Oto główne wtyczki dostępne domyślnie:

1. **Przeglądanie sieci Web (Web Browsing)**: Pozwala agentowi na przeszukiwanie Internetu w czasie rzeczywistym w celu pobrania aktualnych informacji.
2. **Artefakty (Artifacts)**: Umożliwia interaktywne generowanie, edytowanie i podgląd kodu, dokumentów lub projektów interfejsu użytkownika w dedykowanym panelu bocznym.
3. **Pogoda (Weather)**: Odpytuje API pogodowe w celu pobrania lokalnych prognoz.
4. **Kalkulator Kodu (Code Calculator)**: Uruchamia lokalne skrypty, aby dokładnie rozwiązywać złożone obliczenia.

## Aktywacja i Konfiguracja

Każdy agent może mieć własny zestaw włączonych wtyczek. Możesz włączać lub wyłączać wtyczki dla konkretnego agenta za pomocą karty **Wtyczki** w ustawieniach agenta. Niektóre wtyczki (takie jak komercyjne wyszukiwanie w sieci) mogą wymagać dodatkowych ustawień, takich jak klucze API, w ogólnej konfiguracji.
