# Lokalny System Plików 📂

Aplikacja desktopowa **mAI** charakteryzuje się głęboką integracją z systemem plików Twojego systemu operacyjnego, umożliwiając bezpieczny dostęp do lokalnych plików i ich analizę.

## Odczytywanie i Analizowanie Plików

Możesz wysyłać pliki bezpośrednio do swoich agentów (takich jak **May**) w celu analizy lub przetworzenia:
- **Przeciągnij i upuść (Drag and Drop)**: Przeciągnij dokument tekstowy, plik PDF, arkusz Excel lub obraz bezpośrednio do obszaru czatu mAI.
- **Załącznik**: Kliknij ikonę spinacza na czacie, aby przeglądać eksplorator plików.
- Pliki są analizowane i przetwarzane lokalnie, a do LLM wysyłany jest tylko wyodrębniony tekst (lub sam obraz w przypadku modeli multimodalnych).

## Granice Bezpieczeństwa

Aby zapewnić bezpieczeństwo Twojego systemu:
- mAI nigdy nie czyta plików bez Twojego wyraźnego działania (przeciągnięcie i upuszczenie lub ręczny wybór pliku).
- W ustawieniach ogólnych możesz określić dedykowany katalog, aby ograniczyć dostęp mAI tylko do tego folderu.
