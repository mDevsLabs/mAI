# Zarządzanie Agentami 👥

W **mAI** możesz tworzyć i konfigurować wyspecjalizowanych agentów do określonych potrzeb i zastosowań. Zarządzanie agentami odbywa się bezpośrednio przez graficzny interfejs użytkownika lub za pomocą plików konfiguracyjnych.

## Tworzenie Nowego Agenta

Aby utworzyć agenta w mAI:
1. Przejdź do lewego paska bocznego i kliknij przycisk **Agenci**.
2. Kliknij **Utwórz Agenta** u góry listy.
3. Wypełnij podstawowe informacje:
   - **Nazwa**: Identyfikator Twojego agenta.
   - **Awatar**: Wybierz ikonę, emoji lub prześlij obraz.
   - **Opis**: Szybkie podsumowanie roli agenta.

## Edycja Istniejącego Agenta

Możesz edytować dowolnego niestandardowego agenta, klikając powiązaną ikonę ustawień na liście agentów. Uzyskasz dostęp do zaawansowanych konfiguracji, takich jak definiowanie promptu systemowego, wybór domyślnego modelu językowego i zarządzanie jego wtyczkami.

## Usuwanie Agenta

Jeśli niestandardowy agent nie jest już potrzebny, możesz go trwale usunąć w opcjach ustawień agenta. Istniejące dyskusje powiązane z tym agentem nie zostaną usunięte, ale sam agent nie będzie już dostępny do prowadzenia nowych rozmów.
