# 에이전트 관리 👥

**mAI**에서는 특정 요구 사항과 사용 사례에 맞게 전문화된 에이전트를 생성하고 구성할 수 있습니다. 에이전트 관리는 그래픽 사용자 인터페이스(GUI)를 통해 직접 수행하거나 설정 파일을 통해 진행할 수 있습니다.

## 새 에이전트 생성

mAI에서 에이전트를 생성하는 방법:
1. 왼쪽 사이드바에서 **에이전트(Agents)** 버튼을 클릭합니다.
2. 목록 상단에서 **에이전트 생성(Create Agent)**을 클릭합니다.
3. 기본 정보를 입력합니다.
   - **이름**: 에이전트 식별자.
   - **아바타**: 아이콘이나 이모지를 선택하거나 이미지를 업로드합니다.
   - **설명**: 에이전트의 역할에 대한 간략한 요약.

## 기존 에이전트 편집

에이전트 목록에서 해당 설정 아이콘을 클릭하여 맞춤 에이전트를 수정할 수 있습니다. 시스템 프롬프트 정의, 기본 언어 모델 선택, 플러그인 관리와 같은 고급 구성에 액세스할 수 있습니다.

## 에이전트 삭제

더 이상 맞춤 에이전트가 필요하지 않은 경우, 에이전트 설정 옵션을 통해 영구적으로 삭제할 수 있습니다. 이 에이전트와 연결된 기존 대화는 삭제되지 않지만, 해당 에이전트 자체는 새로운 대화에 더 이상 사용할 수 없게 됩니다.
