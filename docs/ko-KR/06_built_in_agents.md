# 기본 내장 에이전트 📦

**mAI**는 수동 설정 없이도 공통 작업에서 즉시 도움을 받을 수 있도록 미리 구성된 여러 시스템 에이전트를 제공합니다.

## 시스템 에이전트 목록

애플리케이션에 내장된 주요 기본 에이전트는 다음과 같습니다.

1. **May**: 작문, 브레인스토밍 및 일반적인 지원 작업을 돕는 다재다능한 핵심 비서.
2. **소프트웨어 개발자(Software Developer)**: 프로그래밍, 클린 코드 작성 및 디버깅에 특화된 시스템 프롬프트로 구성된 에이전트.
3. **만능 번역기(Universal Translator)**: 관용구를 준수하며 수십 개 언어 간의 유창한 번역에 특화된 에이전트.
4. **콘텐츠 작가(Content Writer)**: 마케팅, 블로그 게시물 초안 작성 및 명확한 소셜 미디어 카피 라이팅에 최적화된 에이전트.

## 내장 에이전트 사용하기

이 에이전트들은 설치 후 에이전트 목록에 즉시 나타납니다. 단 한 번의 클릭으로 이 중 누구와도 새로운 대화를 시작할 수 있습니다. 시스템 설정을 변경하는 것은 제한되어 있지만, 각 대화별로 로컬 매개변수(예: 연결된 언어 모델)를 사용자 정의할 수는 있습니다.
