# 일반 설정 ⚙️

**mAI**의 일반 설정을 사용하면 애플리케이션을 미적, 언어적 및 기능적 취향에 맞게 조정할 수 있습니다.

## 사용자 인터페이스 및 테마

- **테마**: 라이트 모드, 다크 모드 중에서 선택하거나 애플리케이션이 컴퓨터의 시스템 설정을 따르도록 지정합니다.
- **강조 색상 (Accent Color)**: 활성화된 버튼 및 그래픽 UI의 주요 대화형 요소의 색상을 사용자 정의합니다.
- **인터페이스 언어**: 애플리케이션 언어(프랑스어, 영어 등)를 선택합니다.
- **글꼴 크기**: 최적의 가독성을 위해 텍스트 크기를 조정합니다.

## 데이터 관리

- **데이터 내보내기**: 모든 대화 기록, 맞춤형 에이전트 및 구성을 단일 JSON 아카이브 파일로 내보냅니다.
- **데이터 가져오기**: 이전 백업을 복원하거나 외부 에이전트 구성을 가져옵니다.
- **애플리케이션 초기화**: 애플리케이션을 초기 상태로 복원하기 위해 모든 로컬 데이터(기록, API 키, 설정)를 영구적으로 삭제합니다.
