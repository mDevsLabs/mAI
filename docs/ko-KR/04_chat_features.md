# 채팅 및 렌더링 기능 💬

**mAI** 대화형 인터페이스는 시각적 기능과 정밀한 제어 옵션으로 가득 찬 유연하고 대화형 환경을 제공하도록 설계되었습니다.

## 핵심 채팅 기능

- **인라인 편집 (Inline Editing)**: 이전 메시지를 언제든지 직접 수정할 수 있습니다. mAI는 해당 시점부터 대화를 자동으로 다시 실행합니다.
- **풍부한 마크다운 렌더링**: mAI는 표, LaTeX 수학 공식, 구문 강조가 적용된 코드 블록 및 대화형 미리 보기 창을 포함한 전체 마크다운 서식을 처리합니다.
- **빠른 작업**: 각 메시지 버블에는 원본 내용 복사, AI 응답 재생성, 편집 또는 기록에서 메시지 삭제를 수행할 수 있는 콘텍스트 메뉴가 제공됩니다.
- **기록 추적**: 채팅 세션은 로컬 SQLite 데이터베이스에 아카이브되어 지난 대화를 즉시 검색하고 복구할 수 있습니다.
