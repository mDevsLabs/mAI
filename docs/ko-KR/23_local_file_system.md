# 로컬 파일 시스템 📂

**mAI** 데스크톱 애플리케이션은 운영 체제의 파일 시스템과 긴밀하게 통합되어 로컬 파일에 안전하게 액세스하고 이를 분석할 수 있도록 지원합니다.

## 파일 읽기 및 분석하기

분석 또는 처리를 위해 에이전트(예: **May**)에게 직접 파일을 보낼 수 있습니다.
- **드래그 앤 드롭**: 텍스트 문서, PDF 파일, Excel 시트 또는 이미지를 mAI 채팅 영역으로 직접 끌어다 놓습니다.
- **첨부 파일**: 채팅창의 클립 아이콘을 클릭하여 파일 탐색기를 탐색합니다.
- 파일은 로컬에서 구문 분석 및 처리되며, 추출된 텍스트(또는 멀티모달 모델의 경우 이미지 자체)만 LLM으로 전송됩니다.

## 보안 경계

시스템의 보안을 유지하기 위해:
- mAI는 사용자의 명시적인 동작(드래그 앤 드롭 또는 수동 파일 선택) 없이는 절대 파일을 읽지 않습니다.
- 일반 설정에서 전용 디렉터리를 지정하여 mAI의 액세스 권한을 해당 폴더로만 제한할 수 있습니다.
