# mAI 소개 🚀

대규모 언어 모델(LLM)과의 상호 작용을 단순화하고 생산성을 극대화하도록 설계된 차세대 인공지능 애플리케이션 **mAI**의 공식 문서에 오신 것을 환영합니다.

## mAI란 무엇인가요?

**mAI**는 현대적이고 빠르며 완전히 사용자 정의가 가능한 채팅 프레임워크이자 대화형 에이전트 플랫폼입니다. 사용자는 매끄럽고 유연한 사용자 인터페이스 내에서 여러 AI 제공업체(OpenAI GPT, Claude, Gemini 또는 Ollama를 통한 로컬 모델 등)에 대한 액세스를 중앙 집중화할 수 있습니다.

## May: 귀하의 핵심 비서 🤖

mAI의 중심에는 기본 지능형 비서인 **May**(이전 명칭 Lobe AI)가 있습니다. 매우 다재다능하게 설계된 May는 첫 실행부터 귀하를 안내하고, 복잡한 질문에 답변하며, 코딩을 돕거나 생각을 정리할 수 있도록 지원합니다. May는 영구 메모리와 외부 도구(플러그인) 사용 능력을 활용하여 귀하의 필요에 맞춰 적응합니다.

## 왜 mAI를 선택해야 할까요?

- **다중 모델 지원**: 다양한 AI 제공업체 간에 즉시 전환할 수 있습니다.
- **개인정보 보호 우선**: API 키와 채팅 기록은 로컬 또는 귀하의 자체 인프라에 안전하게 저장됩니다.
- **풍부한 기능**: 통합 스크린샷 캡처, 시각적 채팅 공유, 플러그인 지원 등 다양한 기능을 제공합니다.
