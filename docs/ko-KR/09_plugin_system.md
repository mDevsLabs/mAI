# 플러그인 시스템 🔌

**mAI**는 대화형 에이전트가 외부 도구 및 서비스와 상호 작용함으로써 학습 데이터의 한계를 극복할 수 있도록 지원하는 강력한 플러그인 에코시스템을 갖추고 있습니다.

## 내장 플러그인

기본적으로 사용할 수 있는 주요 플러그인은 다음과 같습니다.

1. **웹 브라우징 (Web Browsing)**: 에이전트가 인터넷을 실시간으로 검색하여 최신 정보를 가져올 수 있도록 합니다.
2. **아티팩트 (Artifacts)**: 전용 사이드바에서 코드, 문서 또는 UI 디자인을 대화식으로 생성, 편집 및 미리 볼 수 있도록 지원합니다.
3. **날씨 (Weather)**: 날씨 API를 쿼리하여 로컬 예보를 검색합니다.
4. **코드 계산기 (Code Calculator)**: 복잡한 계산을 정확하게 해결하기 위해 로컬 스크립트를 실행합니다.

## 활성화 및 구성

각 에이전트는 자체적으로 활성화된 플러그인 세트를 가질 수 있습니다. 에이전트 설정의 **플러그인 (Plugins)** 탭을 통해 특정 에이전트에 대한 플러그인을 활성화하거나 비활성화할 수 있습니다. 일부 플러그인(예: 상용 웹 검색)은 전역 설정에서 API 키와 같은 추가 구성이 필요할 수 있습니다.
