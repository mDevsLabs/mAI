# API 및 웹훅 🔗

**mAI**는 사용자의 자체 애플리케이션, 웹 서비스 또는 자동화 스크립트와 외부 통합을 허용하기 위해 프로그래밍 방식의 인터페이스를 제공합니다.

## 로컬 API (HTTP)

mAI 데스크톱 앱은 지정된 포트에서 안전한 로컬 웹 서버를 실행할 수 있습니다. 이 서버는 다음과 같은 여러 엔드포인트를 제공합니다.
- `POST /api/chat`: 특정 에이전트에게 메시지를 전송하고 구조화된 응답을 반환합니다 (표준 또는 스트리밍 형식 지원).
- `GET /api/agents`: 사용 가능한 모든 에이전트와 해당 구성 설정을 나열합니다.
- `POST /api/agents`: 프로그래밍 방식으로 새 에이전트를 생성할 수 있도록 지원합니다.

## 웹훅 통합

mAI의 고급 설정에서 발신 웹훅을 구성할 수 있습니다. 웹훅은 특정 이벤트가 발생할 때마다 JSON 페이로드를 외부 URL로 자동으로 게시(POST)합니다.
- **채팅 메시지 완료**: 에이전트가 응답 생성을 마치는 즉시 트리거됩니다.
- **에이전트 생성**: 새로운 에이전트 프로필이 추가될 때 트리거됩니다.
- **시스템 오류**: API 키 확인 문제 또는 LLM 제공업체 장애 시 트리거됩니다.
