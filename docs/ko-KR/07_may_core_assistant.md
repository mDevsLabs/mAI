# May: 핵심 비서 🤖

**May**는 **mAI** 환경의 중심에 있는 기본 비서입니다. 단순한 채팅 에이전트 이상으로, 사용자와 시스템의 강력한 기능 사이의 다리이자 가이드 역할을 수행합니다.

## May의 역할 및 기능

May는 일상적인 도우미가 되도록 설계되었습니다. 주요 역할은 다음과 같습니다.
- **환영 및 온보딩**: 첫 번째 API 키를 구성하고 사용자 인터페이스를 탐색하도록 돕습니다.
- **기술 및 일반 지원**: 애플리케이션 자체에 대한 질문이나 일반 문화, 과학, 예술, 기술 분야의 모든 주제에 대해 답변합니다.
- **개발 지원**: 여러 프로그래밍 언어로 작성된 코드를 이해하고 작성합니다.
- **플러그인 통합**: 기본적으로 May는 웹 브라우징이나 아티팩트 시스템과 같이 활성화된 플러그인에 액세스하여 답변을 더욱 풍부하게 만듭니다.

## May 사용자 정의하기

May는 기본 비서이지만 설정을 통해 다음과 같이 동작을 수정할 수 있습니다.
- 기본 시스템 프롬프트 조정.
- 기본 언어 모델 변경.
- 채팅 세션에 대한 특정 플러그인 활성화 또는 비활성화.
