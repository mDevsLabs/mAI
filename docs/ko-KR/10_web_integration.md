# 웹 통합 🌐

**mAI**는 크로스 플랫폼 애플리케이션으로 설계되었습니다. 데스크톱 앱뿐만 아니라 최적화된 웹 버전을 통해 mAI에 완전히 접속할 수 있으므로, 어떤 웹 브라우저에서든 유연하게 접근할 수 있습니다.

## 웹 버전 기능

mAI 웹 버전은 데스크톱 애플리케이션과 거의 동일한 기능을 제공합니다.
- 부드럽고 반응이 빠르며 유연한 사용자 인터페이스.
- 자체 API 키를 사용하여 모든 언어 모델에 액세스 (브라우저에 로컬로 저장되거나 동기화됨).
- 대부분의 플러그인 사용 가능 (로컬 파일 시스템에 직접 액세스하는 등 로컬 시스템 권한이 필요한 플러그인은 제한될 수 있음).

## 클라우드 동기화

여러 기기에서 mAI를 사용하는 경우 안전한 클라우드 동기화를 활성화할 수 있습니다. 채팅 데이터, 맞춤형 에이전트 및 환경설정은 자체 호스팅 서버 또는 호환되는 클라우드 스토리지 솔루션(예: WebDAV 또는 동기화된 데이터베이스 서비스)을 통해 종단간 암호화(E2EE) 상태로 저장될 수 있습니다.
