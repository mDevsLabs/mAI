# 자주 묻는 질문 (FAQ) ❓

**mAI** 구성 및 사용에 대해 가장 자주 묻는 질문과 답변입니다.

## 일반 사용법

### 인터넷 연결 없이 mAI를 사용할 수 있나요?
네, **Ollama** 또는 LocalAI와 같은 서비스를 통해 실행되는 로컬 모델을 사용하는 경우 가능합니다. 클라우드 모델(OpenAI 또는 Anthropic 등)을 사용하는 경우 해당 서버와 통신하려면 활성화된 인터넷 연결이 필요합니다.

### API 키는 어떻게 업데이트하나요?
설정(`Ctrl + ,`)으로 이동하여 **언어 모델 (Language Models)** 탭을 엽니다. 여기에서 지원되는 각 제공업체의 API 키를 추가, 변경 또는 제거할 수 있습니다.

## 에이전트 사용자 정의

### May가 누구인가요?
**May**는 mAI의 기본 핵심 지능형 비서입니다. 매우 다재다능하며 첫 실행부터 귀하를 도울 수 있도록 구성되어 있습니다.

### 제 에이전트들이 다른 사용자와 공유되나요?
아니요. 기본적으로 귀하가 생성한 모든 맞춤형 에이전트는 로컬에 저장되며 귀하의 기기에 비공개로 유지됩니다. 하지만 다른 사람과 공유하기 위해 수동으로 내보낼 수는 있습니다.

## 비용 및 결제

### mAI는 무료로 사용할 수 있나요?
mAI 애플리케이션 자체는 완전히 무료이며 오픈소스입니다. 다만, 상용 클라우드 모델(OpenAI 또는 Anthropic 등)을 활용하려면 본인의 API 키를 제공해야 합니다. 사용자는 실제 API 사용량에 대한 비용만 모델 제공업체에 직접 지불하게 됩니다.
