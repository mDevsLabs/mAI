# 시작하기 ⚡

이 가이드는 개발 머신에 **mAI**를 로컬로 설정하고 실행하는 데 도움을 줍니다.

## 필수 요구 사항

다음이 설치되어 있는지 확인하세요.
- **Node.js** (v18 이상 권장)
- 패키지 및 워크스페이스 관리를 위한 **pnpm** 또는 **bun**

## 설치

1. **리포지토리 클론**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **의존성 설치**:
   pnpm 사용 시:
   ```bash
   pnpm install
   ```
   bun 사용 시:
   ```bash
   bun install
   ```

3. **개발 서버 실행**:
   ```bash
   pnpm dev
   # 또는
   bun run dev
   ```

애플리케이션은 브라우저의 `http://localhost:3010`에서 접속할 수 있습니다. 첫 실행 시, 핵심 비서인 **May**가 귀하를 환영하며 초기 기본 설정을 구성할 수 있도록 안내해 드립니다!
