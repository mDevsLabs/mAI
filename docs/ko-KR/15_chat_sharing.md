# 채팅 공유 📤

**mAI**는 아름다운 시각적 및 텍스트 기반 공유 도구를 사용하여 동료, 친구 또는 소셜 미디어에 대화를 쉽게 공유할 수 있도록 지원합니다.

## 공유 이미지 생성하기

채팅의 미적이고 스타일리시한 미리보기를 만들려면:
1. 대화 화면 우측 상단의 **공유(Share)** 버튼(내보내기 아이콘)을 클릭합니다.
2. **이미지 생성(Generate Image)** 옵션을 선택합니다.
3. 출력을 사용자 정의합니다.
   - 포함하고 싶은 특정 메시지들을 선택합니다.
   - 배경 스타일(현대적인 그라데이션, 단색 또는 투명)을 선택합니다.
   - 에이전트 아바타(예: **May**) 및 애플리케이션 로고 표시 여부를 선택합니다.
4. 이미지를 저장하려면 **다운로드(Download)**를 클릭하고, 다른 애플리케이션에 직접 붙여넣으려면 **클립보드에 복사(Copy to Clipboard)**를 클릭합니다.

## 공개 링크를 통한 공유 (공유 링크)

웹 버전을 사용 중이거나 공유 스토리지 서비스를 구성한 경우:
- **공유 링크 만들기(Create Share Link)** 옵션을 선택합니다.
- mAI는 대화 기록이 포함된 고유하고 익명화된 URL을 생성합니다.
- 링크가 있는 사람은 누구나 읽기 전용 뷰어에서 대화를 읽고, 자신의 mAI 인스턴스로 대화를 가져와서 대화를 이어갈 수 있습니다.
