# 개발 가이드 🛠️

이 가이드는 **mAI** 프로젝트에 기여하거나 개발 모드에서 소스 코드를 실행하려는 개발자를 위한 기본 지침을 제공합니다.

## 프로젝트 구조

mAI는 현대적인 모노레포 구조(pnpm 및 Turborepo 사용)로 구성되어 있습니다.
- `apps/web`: 메인 Next.js 애플리케이션.
- `packages/`: 공유 모듈 및 재사용 가능한 구성 (UI 컴포넌트, 타입, 헬퍼 함수 등).

## 개발 모드로 실행하기

1. **필수 요구 사항**: Node.js(LTS 버전 권장)와 pnpm 또는 bun이 설치되어 있는지 확인하세요.
2. **의존성 설치**:
   ```bash
   pnpm install
   ```
3. **개발 서버 시작**:
   ```bash
   pnpm run dev
   ```
   애플리케이션은 로컬 `http://localhost:3000`에서 접속할 수 있습니다.

## 테스트

테스트 스위트에는 **Vitest**를 사용합니다.
- 테스트 1회 실행: `pnpm run test`
- 워치 모드로 테스트 실행: `pnpm run test:watch`

수정 사항을 제출하기 전에 모든 테스트가 성공적으로 통과하는지 확인하시기 바랍니다.
