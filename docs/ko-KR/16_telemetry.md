# 텔레메트리 📊

**mAI**에서는 투명성과 개인정보 보호가 가장 최우선 과제입니다. 이 문서는 애플리케이션에서 텔레메트리가 작동하는 방식과 수집될 수 있는 데이터에 대해 자세히 설명합니다.

## 익명 데이터 수집

애플리케이션의 안정성을 높이고 인터페이스 오류를 식별하는 데 도움을 주기 위해 mAI는 익명 텔레메트리를 포함하고 있습니다. 사용자가 동의하는 경우, 다음 정보가 완전히 익명화된 형식으로 전송됩니다.
- 충돌(Crash) 보고서 및 처리되지 않은 JavaScript 예외 발생 사례.
- 전역 사용 통계 (예: 생성된 사용자 정의 에이전트 수, 가장 자주 선택된 언어 모델, 플러그인 활성화율 등).
- 기본 시스템 정보 (애플리케이션 버전, 운영 체제, 화면 해상도).

## 절대 수집하지 않는 정보

mAI의 텔레메트리는 사용자의 개인정보를 엄격히 존중합니다. 당사는 다음 사항을 **절대** 수집, 판독 또는 저장하지 않습니다.
- 메시지 또는 대화 내용.
- 맞춤형 시스템 프롬프트 또는 에이전트 구성 설정.
- API 키 또는 자격 증명 세부 정보.
- 개인 식별 정보 (이름, 이메일 주소 등).

## 텔레메트리 활성화 또는 비활성화

원할 때 언제든지 mAI 개선에 참여하거나 참여를 중단할 수 있습니다.
1. 애플리케이션 **설정(Settings)**으로 이동합니다.
2. **정보 / 텔레메트리(About / Telemetry)** 탭을 클릭합니다.
3. **익명 충돌 보고서 및 사용 통계 허용** 옵션을 토글하여 켜거나 끕니다.
