# 개인정보 보호 및 보안 🔒

데이터 보안과 사용자의 개인정보 존중은 **mAI** 설계의 근본적인 원칙입니다.

## 암호화된 로컬 스토리지

모든 민감한 데이터는 기기에 로컬로 저장됩니다.
- **API 키**: 로컬 스토리지에 저장되기 전에 업계 표준 알고리즘을 사용하여 암호화됩니다. API 키는 당사 서버로 절대 전송되지 않으며, 사용자 기기에서 해당 LLM 제공업체 서버로 직접 전송됩니다.
- **채팅 기록**: 대화 내용은 사용자의 컴퓨터에만 유지됩니다 (사용자가 명시적으로 안전한 클라우드 동기화를 활성화하지 않는 한).

## 보안 연결

언어 모델 API로 향하는 모든 나가는 요청은 보안 통신 프로토콜(HTTPS / SSL)을 사용하여 교환되는 데이터가 네트워크를 통해 가로채어지지 않도록 보장합니다.

## 권장되는 기업 보안 관행

기업 환경에서 최고 수준의 보안을 유지하려면:
- **Ollama** 또는 사설 추론 서버를 통해 로컬에서 호스팅되는 모델을 사용하세요 (기업 네트워크 외부로 데이터가 유출되지 않음을 의미함).
- 일반 설정 탭에서 익명 텔레메트리 참여를 비활성화하세요.
