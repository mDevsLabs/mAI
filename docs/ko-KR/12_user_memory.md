# 사용자 메모리 🧠

**mAI**의 사용자 메모리를 사용하면 에이전트(특히 **May**)가 서로 다른 대화 전반에 걸쳐 사용자에 대한 핵심 사실을 기억할 수 있습니다. 이를 통해 훨씬 더 개인화되고 매끄러운 사용자 경험을 제공합니다.

## 메모리 작동 방식

mAI와 상호 작용할 때, 애플리케이션은 관련 컨텍스트 정보를 자동으로 추출하여 로컬 메모리 데이터베이스에 저장합니다. 여기에는 다음이 포함될 수 있습니다.
- 언어 설정 또는 작문 스타일.
- 현재 진행 중인 프로젝트.
- 개발 환경설정 (선호하는 프레임워크, 코드 스타일 등).
- 공유하기로 선택한 개인 세부 정보 (이름 또는 직업적 역할 등).

## 메모리 관리하기

mAI에 저장된 정보에 대해 완전한 제어권을 가집니다.
1. 애플리케이션 전역 설정으로 이동합니다.
2. **메모리 (Memory)** 탭을 엽니다.
3. 카드 형태로 저장된 사실들의 목록을 볼 수 있습니다.
4. 항목을 수동으로 편집하거나, 메모리 데이터베이스에서 해당 사실을 영구적으로 삭제하여 에이전트가 잊게 만들 수 있습니다.
