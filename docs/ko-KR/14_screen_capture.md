# 화면 캡처 📸

**mAI** 애플리케이션에는 직관적인 화면 캡처 도구가 포함되어 있어 사용자가 화면의 일부를 신속하게 캡처하여 에이전트의 시각적 분석을 위해 대화창에 직접 전송할 수 있습니다.

## 캡처 도구 사용하기

스크린샷을 찍는 방법:
1. 채팅 입력창에 있는 **카메라 / 캡처** 아이콘을 클릭하거나 전용 단축키를 사용합니다.
2. 캡처하려는 영역이나 창을 선택할 수 있도록 사용자 인터페이스(UI)가 일시적으로 숨겨집니다.
3. 선택이 완료되면 이미지가 채팅 입력 상자에 자동으로 첨부됩니다.

## May와 함께하는 시각적 분석

mAI에 구성된 언어 모델의 멀티모달 기능을 활용하여 캡처한 이미지에 대해 다음과 같은 질문을 할 수 있습니다.
- "이 오류 메시지를 설명해 줘."
- "이 이미지에 있는 텍스트를 번역해 줘."
- "이 레이아웃을 검토하고 디자인 개선 사항을 제안해 줘."
- "이 UI 목업에 해당하는 코드를 작성해 줘."
