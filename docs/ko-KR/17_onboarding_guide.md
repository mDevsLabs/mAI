# 온보딩 가이드 🚀

**mAI**에 오신 것을 환영합니다! 이 가이드는 새로운 사용자가 애플리케이션의 첫 단계를 차근차근 밟아 나갈 수 있도록 설계되었습니다.

## 1단계: May의 환영 메시지

애플리케이션을 처음 실행하면 기본 가상 비서인 **May**가 자동으로 대화창을 엽니다. May가 초기 구성 단계를 안내하고 앱에 대해 궁금한 점에 즉시 답변해 드릴 것입니다.

## 2단계: API 키 구성하기

mAI가 작동하려면 하나 이상의 언어 모델 제공업체를 설정해야 합니다.
1. May가 안내하는 구성 버튼을 클릭하거나 설정 창(`Ctrl + ,`)을 엽니다.
2. **언어 모델 (Language Models)** 탭을 선택합니다.
3. 원하는 제공업체(OpenAI, Anthropic Claude, Google Gemini 등)의 API 키를 입력하거나 Ollama 같은 로컬 제공업체를 활성화합니다.
4. 설정을 저장하여 API 키를 기기에 로컬로 안전하게 보관합니다.

## 3단계: 첫 대화 나누기

1. 왼쪽 사이드바에서 **새 채팅 (New Chat)**을 클릭합니다.
2. 화면 하단의 텍스트 영역에 메시지를 입력합니다.
3. **Enter** 키를 눌러 May와의 첫 대화를 시작해 보세요!

## 4단계: 에이전트 둘러보기 및 생성하기

**에이전트 (Agents)** 탭으로 이동하여 사전 구성된 프로필을 둘러보거나, **에이전트 생성 (Create Agent)**을 클릭하여 특정 요구 사항에 맞는 가상 파트너를 직접 디자인해 보세요.
