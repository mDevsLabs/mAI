# 키보드 단축키 ⌨️

생산성을 극대화하고 키보드에서 손을 떼지 않고도 신속하게 탐색할 수 있도록, **mAI**는 다양한 구성 가능한 단축키를 제공합니다.

## 전역 탐색 단축키

- `Ctrl + P` (Mac의 경우 `Cmd + P`): 빠른 명령 메뉴를 엽니다.
- `Ctrl + ,` (Mac의 경우 `Cmd + ,`): 애플리케이션 설정을 엽니다.
- `Ctrl + N` (Mac의 경우 `Cmd + N`): 새 대화를 시작합니다.
- `Ctrl + \` (Mac의 경우 `Cmd + \`): 왼쪽 사이드바를 토글(표시/숨기기)합니다.

## 채팅 단축키

- `Enter`: 메시지를 보냅니다.
- `Shift + Enter`: 입력 필드에 줄 바꿈을 삽입합니다.
- `Ctrl + E`: 마지막으로 보낸 메시지를 편집합니다.
- `Ctrl + R`: 에이전트의 마지막 응답을 다시 생성합니다.
- `Ctrl + Delete`: 현재 대화를 삭제합니다.

## 화면 관리 단축키

- `Alt + 1`: 채팅(Chats) 탭으로 전환합니다.
- `Alt + 2`: 에이전트(Agents) 탭으로 전환합니다.
- `Alt + 3`: 둘러보기(Discover / 마켓플레이스) 탭으로 전환합니다.
