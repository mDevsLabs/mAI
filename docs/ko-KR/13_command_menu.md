# 명령 메뉴 🔍

**mAI**의 빠른 명령 메뉴는 마우스를 전혀 대지 않고도 애플리케이션을 탐색하고 제어할 수 있는 중앙 제어 허브입니다. 단축키 `Ctrl + P`(macOS의 경우 `Cmd + P`)를 사용하여 즉시 활성화할 수 있습니다.

## 핵심 기능

메뉴가 열리면 다음과 같은 명령어를 입력할 수 있습니다.
- **에이전트 전환**: 즉시 전환하려면 에이전트 이름(예: "May" 또는 "Developer")을 입력하세요.
- **채팅 검색 또는 생성**: 키워드로 기존 대화를 검색하거나, `/new`를 입력하여 새 채팅 세션을 시작하세요.
- **설정 액세스**: 시스템 설정 페이지로 바로 이동하려면 `settings`를 입력하세요.
- **모델 변경**: 현재 대화에 대해 다른 LLM을 선택하세요 (예를 들어, "gpt" 또는 "claude"를 입력하면 일치하는 모델이 필터링됩니다).

## 키보드 제어 및 팁

- 키보드의 `Up`(위) 및 `Down`(아래) 방향키를 사용하여 목록을 탐색합니다.
- 선택 항목을 확인하려면 `Enter`를 누릅니다.
- 명령 메뉴를 닫으려면 언제든지 `Esc`를 누릅니다.
