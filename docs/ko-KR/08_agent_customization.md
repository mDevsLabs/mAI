# 에이전트 사용자 정의 ⚙️

**mAI**에서 에이전트를 미세 조정하면 에이전트의 동작, 응답 스타일 및 액세스할 수 있는 도구를 깊이 있게 수정할 수 있습니다.

## 기본 설정

각 에이전트에 대해 다음을 구성할 수 있습니다.
- **시스템 프롬프트**: 에이전트의 성격, 규칙 및 역할을 정의하는 기초 지침입니다. 에이전트의 응답에 가장 큰 영향을 미치는 설정입니다.
- **언어 모델**: 사용할 기본 LLM을 선택합니다(예: GPT-4o, Claude 3.5 Sonnet, Llama 3).

## 고급 설정

생성 하이퍼파라미터를 수정할 수도 있습니다.
- **온도 (Temperature)**: 응답의 창의성을 제어합니다. 낮은 온도(예: 0.2)는 보다 사실적이고 결정론적인 답변을 생성합니다. 높은 온도(예: 0.9)는 창의적인 답변을 유도합니다.
- **Top P**: 응답 다양성을 제어하는 또 다른 방법입니다.
- **존재 / 빈도 페널티 (Presence / Frequency Penalty)**: 에이전트가 동일한 단어를 반복하거나 논의 중인 주제에서 벗어나는 것을 조장하거나 억제합니다.
- **최대 토큰 (Max Tokens)**: 생성되는 응답의 최대 길이를 제한합니다.
