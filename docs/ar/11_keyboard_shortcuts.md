# اختصارات لوحة المفاتيح ⌨️

لزيادة إنتاجيتك إلى الحد الأقصى والتنقل بسرعة دون مغادرة لوحة المفاتيح، يتضمن **mAI** مجموعة من الاختصارات القابلة للتهيئة.

## اختصارات التنقل العامة

- `Ctrl + P` (أو `Cmd + P` على نظام Mac): يفتح قائمة الأوامر السريعة.
- `Ctrl + ,` (أو `Cmd + ,` على نظام Mac): يفتح إعدادات التطبيق.
- `Ctrl + N` (أو `Cmd + N` على نظام Mac): يبدأ محادثة جديدة.
- `Ctrl + \` (أو `Cmd + \` على نظام Mac): يبدل إظهار/إخفاء الشريط الجانبي الأيسر.

## اختصارات المحادثة

- `Enter`: إرسال الرسالة.
- `Shift + Enter`: إدراج سطر جديد في حقل الإدخال.
- `Ctrl + E`: تعديل آخر رسالة تم إرسالها.
- `Ctrl + R`: إعادة توليد آخر استجابة للعميل.
- `Ctrl + Delete`: حذف المناقشة الحالية.

## اختصارات إدارة العرض

- `Alt + 1`: الانتقال إلى علامة تبويب المحادثات (Chats).
- `Alt + 2`: الانتقال إلى علامة تبويب العملاء (Agents).
- `Alt + 3`: الانتقال إلى علامة تبويب الاكتشاف (Discover/Marketplace).
