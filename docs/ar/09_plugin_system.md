# نظام المكونات الإضافية (Plugin System) 🔌

يتميز **mAI** بنظام بيئي قوي للمكونات الإضافية يتيح للعملاء الحواريين التغلب على قيود بيانات تدريبهم من خلال التفاعل مع الأدوات والخدمات الخارجية.

## المكونات الإضافية المدمجة

فيما يلي المكونات الإضافية الأساسية المتاحة افتراضيًا:

1. **تصفح الويب (Web Browsing)**: يسمح للعميل بالبحث في الإنترنت في الوقت الفعلي لجلب معلومات محدثة.
2. **العناصر المنتجة (Artifacts)**: يتيح توليد وتعديل ومعاينة التعليمات البرمجية أو المستندات أو تصميمات واجهة المستخدم بشكل تفاعلي في شريط جانبي مخصص.
3. **الطقس (Weather)**: يستعلم من واجهات برمجة تطبيقات الطقس لاسترداد التوقعات المحلية.
4. **حاسبة الأكواد (Code Calculator)**: ينفذ البرامج النصية المحلية لحل الحسابات المعقدة بدقة.

## التنشيط والتهيئة

يمكن لكل عميل أن يكون لديه مجموعته الخاصة من المكونات الإضافية الممكنة. يمكنك تمكين أو تعطيل المكونات الإضافية لعميل معين عبر علامة التبويب **المكونات الإضافية (Plugins)** في إعدادات العميل. قد تتطلب بعض المكونات الإضافية (مثل البحث التجاري على الويب) إعدادات إضافية، مثل مفاتيح واجهة برمجة التطبيقات (API keys)، في التكوين العام.
