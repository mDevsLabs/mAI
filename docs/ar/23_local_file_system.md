# نظام الملفات المحلي 📂

يتميز تطبيق **mAI** المخصص لسطح المكتب بتكامل عميق مع نظام ملفات نظام التشغيل الخاص بك، مما يتيح لك الوصول إلى ملفاتك المحلية وتحليلها بأمان.

## قراءة وتحليل الملفات

يمكنك إرسال الملفات مباشرة إلى عملائك (مثل **May**) للتحليل أو المعالجة:
- **السحب والإفلات**: اسحب مستندًا نصيًا، أو ملف PDF، أو ورقة Excel، أو صورة مباشرة إلى منطقة محادثة mAI.
- **المرفقات**: انقر فوق أيقونة مشبك الورق في المحادثة لتصفح مستكشف الملفات الخاص بك.
- يتم تحليل الملفات ومعالجتها محليًا، ويتم إرسال النص المستخرج فقط (أو الصورة نفسها للنماذج متعددة الوسائط) إلى نموذج اللغة الكبير (LLM).

## الحدود الأمنية

للحفاظ على أمان نظامك:
- لا يقرأ mAI الملفات أبدًا دون إجراء صريح منك (السحب والإفلات أو اختيار الملف يدويًا).
- يمكنك تحديد دليل مخصص في الإعدادات العامة لتقييد وصول mAI إلى هذا المجلد فقط.
