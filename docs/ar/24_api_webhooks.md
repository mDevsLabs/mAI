# واجهة برمجة التطبيقات وخطافات الويب (API & Webhooks) 🔗

يوفر **mAI** واجهات برمجية للسماح بالتكامل الخارجي مع تطبيقاتك الخاصة، أو خدمات الويب، أو نصوص الأتمتة البرمجية.

## واجهة برمجة التطبيقات المحلية (HTTP)

يمكن لتطبيق سطح المكتب لـ mAI تشغيل خادم ويب محلي آمن على منفذ مخصص. يعرض this الخادم عدة نقاط نهاية (endpoints):
- `POST /api/chat`: يرسل رسالة إلى عميل معين ويعيد استجابة مهيكلة (يدعم التنسيق القياسي أو التدفق).
- `GET /api/agents`: يسرد جميع العملاء المتاحين وتكويناتهم.
- `POST /api/agents`: يسمح بإنشاء عميل جديد برمجياً.

## تكامل خطافات الويب (Webhooks)

يمكنك تهيئة خطافات الويب الصادرة في إعدادات mAI المتقدمة. يقوم خطاف الويب بنشر حمولة JSON تلقائيًا إلى عنوان URL خارجي عند حدوث أحداث معينة:
- **اكتملت رسالة المحادثة**: يتم تشغيلها بمجرد أن ينتهي العميل من توليد الاستجابة.
- **تم إنشاء عميل**: يتم تشغيلها عند إضافة ملف تعريف عميل جديد.
- **خطأ في النظام**: يتم تشغيلها أثناء مشكلات التحقق من صحة مفتاح واجهة برمجة التطبيقات أو انقطاع الخدمة من مزود LLM.
