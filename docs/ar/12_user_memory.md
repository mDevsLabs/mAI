# ذاكرة المستخدم 🧠

تتيح ذاكرة المستخدم في **mAI** لعملائك، وخاصة **May**، تذكر الحقائق الرئيسية عنك عبر المحادثات المختلفة. هذا يجعل تجربة المستخدم أكثر تخصيصًا وسلاسة.

## كيف تعمل الذاكرة

أثناء تفاعلك مع mAI، يقوم التطبيق تلقائيًا باستخراج المعلومات السياقية ذات الصلة وتخزينها في قاعدة بيانات الذاكرة المحلية. يمكن أن يشمل ذلك:
- تفضيلات اللغة الخاصة بك أو أسلوب الكتابة.
- المشاريع التي تعمل عليها حاليًا.
- تفضيلات التطوير الخاصة بك (أطر العمل المفضلة، وأنماط الكود).
- التفاصيل الشخصية التي تختار مشاركتها (مثل اسمك أو دورك المهني).

## إدارة ذاكرتك

لديك تحكم كامل في المعلومات المخزنة بواسطة mAI:
1. اذهب إلى إعدادات التطبيق العامة.
2. افتح علامة التبويب **الذاكرة (Memory)**.
3. سترى قائمة بالحقائق المخزنة على شكل بطاقات.
4. يمكنك تعديل أي إدخال يدويًا، أو حذف حقيقة بشكل نهائي من قاعدة بيانات الذاكرة حتى ينساها العميل.
