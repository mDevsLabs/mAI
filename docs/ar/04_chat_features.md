# ميزات المحادثات والعرض 💬

تم تصميم واجهة المحادثة لـ **mAI** لتقديم تجربة سلسة وتفاعلية، مليئة بالميزات المرئية وخيارات التحكم الدقيقة.

## ميزات المحادثة الأساسية

- **التعديل المضمن (Inline Editing)**: يمكنك تعديل أي من رسائلك السابقة في أي وقت. سيقوم mAI تلقائيًا بإعادة تشغيل المحادثة من ذلك الدور فصاعدًا.
- **عرض غني للتنسيق (Markdown)**: يتعامل mAI مع تنسيق Markdown الكامل: الجداول، ومعادلات LaTeX الرياضية، ومجموعات الكود المميزة بالبنية، ولوحات المعاينة التفاعلية.
- **الإجراءات السريعة**: تتميز كل فقاعة رسالة بقائمة سياق تتيح لك نسخ المحتوى الخام، أو إعادة توليد استجابة الذكاء الاصطناعي، أو تعديل الرسالة أو حذفها من سجلك.
- **تتبع السجل**: يتم أرشفة جلسات المحادثة محليًا في قاعدة بيانات SQLite، مما يتيح لك البحث الفوري واستعادة محادثاتك السابقة.
