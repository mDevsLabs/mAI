# الإعدادات العامة ⚙️

تتيح لك الإعدادات العامة في **mAI** ضبط التطبيق وفقًا لتفضيلاتك الجمالية واللغوية والوظيفية.

## واجهة المستخدم والسمات

- **السمة (Theme)**: اختر بين الوضع الفاتح أو الوضع الداكن، أو دع التطبيق يتبع تفضيلات نظام الكمبيوتر الخاص بك.
- **لون التركيز (Accent Color)**: قم بتخصيص لون الأزرار النشطة والعناصر التفاعلية الرئيسية لواجهة المستخدم الرسومية.
- **لغة الواجهة**: حدد لغة التطبيق (الفرنسية، الإنجليزية، إلخ).
- **حجم الخط**: اضبط حجم النص لتوفير راحة قراءة مثالية.

## إدارة البيانات

- **تصدير البيانات**: قم بتصدير جميع محادثاتك وعملائك المخصصين وتكويناتك في ملف أرشيف JSON واحد.
- **استيراد البيانات**: قم باستعادة نسخة احتياطية سابقة أو استيراد تكوينات عملاء خارجية.
- **إعادة تعيين التطبيق**: امسح نهائيًا جميع البيانات المحلية (السجل، مفاتيح واجهة برمجة التطبيقات، الإعدادات) لاستعادة التطبيق إلى حالته الأصلية.
