# دليل التطوير 🛠️

يقدم هذا الدليل إرشادات أساسية للمطورين الذين يرغبون في المساهمة في مشروع **mAI** أو تشغيل الكود المصدري في وضع التطوير.

## هيكل المشروع

تم تنظيم mAI كمستودع أحادي حديث (باستخدام pnpm و Turborepo):
- `apps/web`: تطبيق Next.js الرئيسي.
- `packages/`: الوحدات المشتركة والتكوينات القابلة لإعادة الاستخدام (مكونات واجهة المستخدم، الأنواع، المساعدين).

## التشغيل في وضع التطوير

1. **المتطلبات الأساسية**: تأكد من تثبيت Node.js (يوصى بإصدار LTS) و pnpm أو bun.
2. **تثبيت التبعيات**:
   ```bash
   pnpm install
   ```
3. **بدء تشغيل خادم التطوير**:
   ```bash
   pnpm run dev
   ```
   سيكون التطبيق متاحًا محليًا على `http://localhost:3000`.

## الاختبارات

نحن نستخدم **Vitest** لمجموعات الاختبار الخاصة بنا:
- تشغيل الاختبارات مرة واحدة: `pnpm run test`
- تشغيل الاختبارات في وضع المراقبة (watch mode): `pnpm run test:watch`

يرجى التأكد من اجتياز جميع الاختبارات بنجاح قبل تقديم أي تعديلات.
