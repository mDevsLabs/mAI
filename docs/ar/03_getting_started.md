# البداية ⚡

يساعدك هذا الدليل في إعداد وتشغيل **mAI** محليًا على جهاز التطوير الخاص بك.

## المتطلبات الأساسية

تأكد من تثبيت ما يلي:
- **Node.js** (يوصى بالإصدار 18 أو أحدث)
- **pnpm** أو **bun** لإدارة الحزم ومساحات العمل

## التثبيت

1. **استنساخ المستودع**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **تثبيت التبعيات**:
   باستخدام pnpm:
   ```bash
   pnpm install
   ```
   باستخدام bun:
   ```bash
   bun install
   ```

3. **تشغيل خادم التطوير**:
   ```bash
   pnpm dev
   # أو
   bun run dev
   ```

سيكون التطبيق متاحًا في متصفحك على `http://localhost:3010`. في أول تشغيل لك، ستشهد ترحيبًا من **May**، مساعدتك الأساسية، لمساعدتك في تهيئة تفضيلاتك الأولية!
