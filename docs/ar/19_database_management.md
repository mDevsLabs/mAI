# إدارة قاعدة البيانات 🗄️

يدير **mAI** تخزين بياناته بطريقة منظمة وعالية الأداء لضمان وصول سريع وغير متصل بالإنترنت لمحادثاتك وتكويناتك.

## بنية التخزين

اعتمادًا على بيئة التشغيل (سطح المكتب أو الويب)، يستخدم mAI محركات تخزين مختلفة:
- **تطبيق سطح المكتب**: يستخدم قاعدة بيانات محلية خفيفة الوزن (SQLite) مخزنة في دليل مستخدم التطبيق. ويضمن أداءً ممتازًا في القراءة/الكتابة لأحجام السجلات الكبيرة.
- **تطبيق الويب**: يعتمد على قاعدة البيانات الداخلية للمتصفح (IndexedDB) أو خدمات واجهة برمجة التطبيقات الخارجية المتزامنة مع قاعدة بيانات PostgreSQL إذا تم تمكين الوضع السحابي.

## ترحيل المخطط (Schema Migrations)

يتضمن mAI نظام ترحيل تلقائي لمخطط قاعدة البيانات. مع كل تحديث للتطبيق، إذا تغير هيكل قاعدة البيانات (على سبيل المثال، إضافة حقل جديد للعملاء أو الرسائل)، فسيتم تشغيل عمليات الترحيل بسلاسة عند بدء التشغيل دون تغيير بياناتك الحالية.

## تحسين الأداء

لإبقاء التطبيق سريعًا:
- **التنظيف التلقائي**: يمكن تقليم الرسائل المؤقتة أو الجلسات منتهية الصلاحية بشكل دوري.
- **الفهرسة**: يتم فهرسة رسائل المحادثة للسماح ببحث نصي شبه فوري.
