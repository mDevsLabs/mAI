# 🌈 Variétés de Compagnons

mAI propose différents styles de compagnons pour s'adapter aux goûts de chaque utilisateur. Chaque variété est conçue avec une thématique propre tout en respectant la même grille d'animations.

## 🎨 Styles disponibles

On retrouve plusieurs catégories de compagnons :

- **Style Minimaliste** : Des formes simples et épurées pour une interface discrète.
- **Style Inspiré de l'IA** : Des designs évoquant la technologie, les circuits et les flux de données.
- **Style Anthropomorphe** : Des personnages miniatures avec des expressions marquées.
- **Style Abstrait** : Des formes géométriques et colorées qui jouent sur la dynamique.

L'utilisateur peut basculer entre ces compagnons pour personnaliser son espace de travail.
