# FAQ ❓

Voici les réponses aux questions les plus fréquemment posées sur l'utilisation et la configuration de **mAI**.

## Questions d'utilisation générale

### Puis-je utiliser mAI sans connexion internet ?
Oui, si vous utilisez des modèles locaux via des outils comme **Ollama** ou LocalAI. Si vous utilisez des modèles hébergés sur le cloud (comme OpenAI ou Anthropic), une connexion internet active est requise pour envoyer les messages aux serveurs des fournisseurs.

### Comment réinitialiser les clés d'API ?
Accédez aux Paramètres (`Ctrl + ,`), puis allez dans l'onglet **Modèles de langage**. Vous pouvez y supprimer, modifier ou ajouter de nouvelles clés d'API pour chaque fournisseur.

## Personnalisation et Agents

### Qui est May ?
**May** est l'assistant de base intelligent par défaut de mAI. Elle est conçue pour être extrêmement polyvalente et vous accompagner dès l'installation.

### Mes agents sont-ils partagés avec d'autres utilisateurs ?
Non, par défaut, tous les agents personnalisés que vous créez sont stockés en local et restent strictement privés sur votre machine. Vous pouvez néanmoins choisir de les exporter manuellement pour les partager avec d'autres personnes.

## Sécurité et facturation

### mAI est-il gratuit ?
L'application mAI elle-même est gratuite et open-source. Cependant, l'utilisation de modèles de langage cloud commerciaux (comme OpenAI ou Anthropic) nécessite que vous utilisiez vos propres clés d'API payantes. Vous ne payez que ce que vous consommez auprès des fournisseurs d'API correspondants.
