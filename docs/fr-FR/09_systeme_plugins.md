# Système de plugins 🔌

**mAI** intègre un écosystème de plugins puissant qui permet aux agents conversationnels de dépasser les limites de leur base de connaissances en interagissant avec des outils et des services externes.

## Plugins intégrés

Voici les principaux plugins disponibles par défaut :

1. **Navigation Web (Web Browsing)** : Permet à l'agent d'effectuer des recherches sur Internet en temps réel pour obtenir des informations à jour.
2. **Artefacts (Artifacts)** : Permet de générer, d'éditer et de prévisualiser du code, des documents, ou des maquettes d'interface utilisateur de manière interactive dans un panneau latéral dédié.
3. **Météo** : Interroge des APIs météo pour obtenir les prévisions locales.
4. **Calculateur de Code** : Exécute des scripts en local pour résoudre des calculs complexes de manière exacte.

## Activation et configuration

Chaque agent peut avoir son propre ensemble de plugins activés. Vous pouvez activer ou désactiver des plugins pour un agent spécifique via l'onglet **Plugins** dans les paramètres de l'agent. Pour certains plugins (comme la recherche web payante), des configurations supplémentaires comme une clé d'API peuvent être nécessaires dans les paramètres généraux.
