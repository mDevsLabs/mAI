# Guide d'intégration 🚀

Bienvenue dans **mAI** ! Ce guide est conçu pour accompagner pas à pas les nouveaux utilisateurs lors de leur première prise en main de l'application.

## Étape 1 : Le message de bienvenue de May

Au premier lancement de l'application, **May**, votre assistante virtuelle de base, s'ouvre automatiquement. Elle vous guidera à travers les premières étapes de configuration et répondra à vos questions immédiates sur le fonctionnement de l'application.

## Étape 2 : Configuration des clés API

Pour que mAI fonctionne, vous devez configurer au moins un fournisseur de modèle de langage :
1. Cliquez sur le bouton de configuration proposé par May ou ouvrez les paramètres (`Ctrl + ,`).
2. Sélectionnez l'onglet **Modèles de langage**.
3. Saisissez votre clé API pour le fournisseur de votre choix (OpenAI, Anthropic Claude, Google Gemini) ou activez un fournisseur local comme Ollama.
4. Validez pour enregistrer la clé de façon sécurisée sur votre appareil.

## Étape 3 : Votre première discussion

1. Cliquez sur **Nouvelle discussion** dans le panneau latéral gauche.
2. Saisissez votre message dans la zone de texte en bas de l'écran.
3. Appuyez sur **Entrée** pour lancer votre première discussion avec May !

## Étape 4 : Découvrir et créer des agents

Rendez-vous dans l'onglet **Agents** pour explorer les profils préconfigurés ou cliquez sur **Créer un agent** pour concevoir un compagnon virtuel adapté à vos besoins spécifiques.
