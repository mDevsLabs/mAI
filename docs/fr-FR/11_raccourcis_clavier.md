# Raccourcis clavier ⌨️

Pour maximiser votre productivité et naviguer rapidement sans quitter votre clavier, **mAI** intègre une série de raccourcis configurables.

## Raccourcis de navigation globale

- `Ctrl + P` (ou `Cmd + P` sur Mac) : Ouvre le menu de commandes rapides.
- `Ctrl + ,` (ou `Cmd + ,` sur Mac) : Ouvre les paramètres de l'application.
- `Ctrl + N` (ou `Cmd + N` sur Mac) : Démarre une nouvelle discussion.
- `Ctrl + \` (ou `Cmd + \` sur Mac) : Masque ou affiche la barre latérale gauche.

## Raccourcis dans la discussion

- `Entrée` : Envoyer le message.
- `Maj + Entrée` : Insérer un saut de ligne dans le champ de saisie.
- `Ctrl + E` : Modifier le dernier message envoyé.
- `Ctrl + R` : Régénérer la dernière réponse de l'agent.
- `Ctrl + Suppr` : Supprimer la discussion en cours.

## Raccourcis de gestion d'affichage

- `Alt + 1` : Basculer vers l'onglet Discussions.
- `Alt + 2` : Basculer vers l'onglet Agents.
- `Alt + 3` : Basculer vers l'onglet Découverte (Marketplace).
