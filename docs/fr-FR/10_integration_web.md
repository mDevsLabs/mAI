# Intégration Web 🌐

**mAI** est conçu comme une application multiplateforme. En plus de l'application de bureau (Desktop), mAI est entièrement accessible via une version Web optimisée, permettant un accès fluide depuis n'importe quel navigateur internet.

## Fonctionnalités de la version Web

La version Web de mAI offre une parité de fonctionnalités presque totale avec l'application de bureau :
- Interface utilisateur fluide, responsive et réactive.
- Accès à tous les modèles de langage via vos propres clés d'API (sauvegardées localement dans votre navigateur ou synchronisées).
- Utilisation de la majorité des plugins (les plugins nécessitant des privilèges système locaux, comme l'accès direct aux fichiers locaux, peuvent être restreints).

## Synchronisation Cloud

Si vous utilisez mAI sur plusieurs appareils, vous pouvez activer la synchronisation cloud sécurisée. Vos données de discussion, vos agents personnalisés et vos préférences peuvent être sauvegardés de manière chiffrée de bout en bout sur votre serveur d'hébergement ou via des solutions de stockage cloud compatibles (telles que WebDAV ou des services de base de données synchronisés).
