# Capture d'écran 📸

L'application **mAI** intègre un outil de capture d'écran intuitif qui permet aux utilisateurs de capturer rapidement des zones de leur écran et de les envoyer directement dans la discussion à des fins d'analyse visuelle par l'agent.

## Utilisation de l'outil de capture

Pour effectuer une capture d'écran :
1. Cliquez sur l'icône **Appareil photo / Capture** située dans la zone de saisie du chat ou utilisez le raccourci dédié.
2. L'interface se masque temporairement pour vous laisser sélectionner la zone ou la fenêtre que vous souhaitez capturer.
3. Une fois sélectionnée, l'image est automatiquement importée en tant que pièce jointe dans votre boîte de saisie.

## Analyse visuelle avec May

Grâce aux capacités multimodales des modèles de langage configurés dans mAI, vous pouvez poser des questions sur l'image capturée :
- "Explique-moi ce message d'erreur."
- "Traduis ce texte présent sur l'image."
- "Commente le design ou trouve des idées d'amélioration pour cette mise en page."
- "Écris le code correspondant à cette maquette d'interface."
