# Gestion des agents 👥

Dans **mAI**, vous pouvez créer et configurer des agents spécialisés pour répondre à des besoins et des cas d'utilisation spécifiques. La gestion des agents s'effectue directement depuis l'interface utilisateur graphique ou via des fichiers de configuration.

## Créer un nouvel agent

Pour créer un agent dans mAI :
1. Accédez au panneau latéral gauche et cliquez sur le bouton **Agents**.
2. Cliquez sur **Créer un agent** en haut de la liste.
3. Remplissez les informations de base :
   - **Nom** : Identifiant de votre agent.
   - **Avatar** : Choisissez une icône, un emoji ou téléchargez une image.
   - **Description** : Un résumé rapide du rôle de l'agent.

## Modifier un agent existant

Vous pouvez éditer n'importe quel agent personnalisé en cliquant sur l'icône de paramétrage associée dans la liste des agents. Vous aurez accès à des configurations avancées telles que la définition du prompt système, le choix du modèle de langage par défaut et la gestion de ses plugins.

## Supprimer un agent

Si un agent personnalisé n'est plus nécessaire, vous pouvez le supprimer définitivement via les options de configuration de l'agent. Les discussions existantes liées à cet agent ne seront pas supprimées, mais l'agent lui-même ne sera plus disponible pour de nouvelles conversations.
