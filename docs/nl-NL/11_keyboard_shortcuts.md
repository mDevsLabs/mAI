# Sneltoetsen ⌨️

Om uw productiviteit te maximaliseren en snel te navigeren zonder uw toetsenbord te verlaten, bevat **mAI** een reeks configureerbare sneltoetsen.

## Globale navigatiesneltoetsen

- `Ctrl + P` (of `Cmd + P` op Mac): Opent het snelle commandomenu.
- `Ctrl + ,` (of `Cmd + ,` op Mac): Opent de applicatie-instellingen.
- `Ctrl + N` (of `Cmd + N` op Mac): Start een nieuw gesprek.
- `Ctrl + \` (of `Cmd + \` op Mac): Schakelt de linkerzijbalk in/uit.

## Sneltoetsen voor de chat

- `Enter`: Verstuur het bericht.
- `Shift + Enter`: Voeg een regelafstand in in het invoerveld.
- `Ctrl + E`: Bewerk uw laatst verzonden bericht.
- `Ctrl + R`: Genereer het laatste antwoord van de agent opnieuw.
- `Ctrl + Delete`: Verwijder de huidige discussie.

## Sneltoetsen voor schermweergave

- `Alt + 1`: Schakel over naar het tabblad Chats.
- `Alt + 2`: Schakel over naar het tabblad Agenten.
- `Alt + 3`: Schakel over naar het tabblad Ontdekken (Marktplaats).
