# Plug-insysteem 🔌

**mAI** beschikt over een krachtig plug-in-ecosysteem waarmee conversationele agenten de beperkingen van hun trainingsgegevens kunnen overwinnen door interactie met externe tools en services.

## Ingebouwde plug-ins

Dit zijn de belangrijkste plug-ins die standaard beschikbaar zijn:

1. **Web Browsing**: Stelt de agent in staat om in realtime op internet te zoeken om up-to-date informatie op te halen.
2. **Artifacts**: Maakt het interactief genereren, bewerken en bekijken van code, documenten of UI-ontwerpen in een speciale zijbalk mogelijk.
3. **Weer**: Vraagt weer-API's af om lokale weersvoorspellingen op te halen.
4. **Codecalculator**: Voert lokale scripts uit om complexe berekeningen nauwkeurig op te lossen.

## Activering en configuratie

Elke agent kan zijn eigen set geactiveerde plug-ins hebben. U kunt plug-ins voor een specifieke agent in- of uitschakelen via het tabblad **Plug-ins** in de agentinstellingen. Sommige plug-ins (zoals commerciële webzoekopdrachten) vereisen mogelijk aanvullende instellingen, zoals API-sleutels, in de algemene configuratie.
