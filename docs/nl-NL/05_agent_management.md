# Agentenbeheer 👥

In **mAI** kunt u gespecialiseerde agenten maken en configureren om aan specifieke behoeften en use cases te voldoen. Het agentenbeheer wordt rechtstreeks via de grafische gebruikersinterface of via configuratiebestanden uitgevoerd.

## Een nieuwe agent maken

Om een agent te maken in mAI:
1. Ga naar de linkerzijbalk en klik op de knop **Agenten**.
2. Klik op **Agent maken** bovenaan de lijst.
3. Vul de basisinformatie in:
   - **Naam**: De identificatie van uw agent.
   - **Avatar**: Kies een pictogram, emoji of upload een afbeelding.
   - **Beschrijving**: Een korte samenvatting van de rol van de agent.

## Een bestaande agent bewerken

U kunt elke aangepaste agent bewerken door op het bijbehorende instellingenpictogram in de agentenlijst te klikken. U krijgt toegang tot geavanceerde configuraties zoals het definiëren van de systeemprompt, het kiezen van het standaardtaalmodel en het beheren van de bijbehorende plug-ins.

## Een agent verwijderen

Als een aangepaste agent niet langer nodig is, kunt u deze permanent verwijderen via de instellingsopties van de agent. Bestaande discussies die aan deze agent zijn gekoppeld, worden niet verwijderd, maar de agent zelf is niet langer beschikbaar voor nieuwe gesprekken.
