# Menu de commande 🔍

Le menu de commande rapide de **mAI** est la commande centrale pour naviguer et contrôler l'application sans toucher à votre souris. Il s'active instantanément en utilisant le raccourci `Ctrl + P` (ou `Cmd + P` sur macOS).

## Fonctionnalités principales

Une fois le menu ouvert, vous pouvez saisir des commandes pour :
- **Changer d'agent** : Tapez le nom d'un agent (comme "May" ou "Développeur") pour basculer instantanément.
- **Créer ou changer de discussion** : Recherchez dans vos discussions existantes par mots-clés ou tapez `/new` pour démarrer une nouvelle conversation.
- **Accéder aux paramètres** : Tapez `paramètres` pour vous rendre directement dans les réglages système.
- **Changer de modèle** : Sélectionnez un autre LLM pour la discussion active (par exemple, taper "gpt" ou "claude" affichera les modèles correspondants).

## Raccourcis et astuces

- Utilisez les flèches `Haut` et `Bas` de votre clavier pour naviguer dans la liste.
- Appuyez sur `Entrée` pour valider une sélection.
- Appuyez sur `Échap` pour fermer le menu de commande à tout moment.
