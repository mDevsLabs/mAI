# Agents intégrés 📦

**mAI** est livré par défaut avec plusieurs agents système préconfigurés pour vous aider immédiatement dans des tâches courantes sans nécessiter de configuration manuelle.

## Liste des agents système

Voici les principaux agents intégrés à l'application :

1. **May** : Votre assistante de base polyvalente, prête pour toutes les tâches d'écriture, de brainstorming et d'aide générale.
2. **Développeur Logiciel** : Un agent configuré avec un prompt système axé sur la programmation, l'écriture de code propre et le débogage.
3. **Traducteur Universel** : Spécialisé dans la traduction fluide entre plusieurs dizaines de langues, respectant les expressions idiomatiques.
4. **Rédacteur de Contenu** : Optimisé pour le marketing, la rédaction d'articles de blog et la formulation de messages clairs pour les réseaux sociaux.

## Utilisation des agents intégrés

Ces agents sont immédiatement visibles dans votre liste d'agents après l'installation. Vous pouvez démarrer une nouvelle conversation avec l'un d'eux d'un simple clic. Bien qu'ils soient verrouillés pour certaines modifications système, vous pouvez personnaliser leurs paramètres locaux (comme le modèle de langage associé) pour chaque discussion.
