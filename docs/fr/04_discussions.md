# Discussions et Rendu Chat 💬

L'interface de chat de **mAI** a été conçue pour offrir l'expérience la plus fluide possible, riche en fonctionnalités visuelles et en contrôles interactifs.

## Fonctionnalités Clés du Chat

- **Édition à la volée** : Vous pouvez éditer n'importe lequel de vos messages précédents à tout moment. mAI relancera automatiquement la conversation à partir de ce point.
- **Rendu Markdown enrichi** : mAI gère le formatage en Markdown complet : tableaux, équations mathématiques LaTeX, rendu de code syntaxique et blocs interactifs.
- **Actions rapides** : Chaque bulle de message dispose d'un menu contextuel permettant de copier le contenu brut, de régénérer la réponse, d'éditer le message ou de le supprimer définitivement de l'historique local.
- **Suivi de l'historique** : Les sessions de discussion sont archivées localement dans la base de données SQLite, vous permettant de retrouver vos échanges passés instantanément grâce à la recherche.
