# Keyboard Shortcuts

mAI provides a comprehensive set of keyboard shortcuts so you can work without constantly reaching for the mouse. Master a few key ones and your workflow noticeably speeds up ‚Äî especially when switching Agents, managing conversations, or navigating the interface.

## How to Access Shortcut Settings

Click your user avatar in the top-right corner ‚Üí **App Settings** ‚Üí **Shortcuts**. You can view all shortcuts and customize most of them.

## Essential Shortcuts

| Action                             | Shortcut                               |
| ---------------------------------- | -------------------------------------- |
| **Open Command Menu**              | `‚åò + K` (Mac) / `Ctrl + K` (Win/Linux) |
| **Focus search**                   | `‚åò + J` / `Ctrl + J`                   |
| **Switch to pinned Agent**         | `Ctrl + 1‚Äì9`                           |
| **Switch to default conversation** | `Ctrl + 0`                             |
| **Toggle left panel**              | `‚åò + [` / `Ctrl + [`                   |
| **Toggle right panel**             | `‚åò + ]` / `Ctrl + ]`                   |
| **Focus mode**                     | `‚åò + \` / `Ctrl + \`                   |
| **Shortcut help**                  | `Ctrl + Shift + ?`                     |

**Focus mode** hides all panels except the current conversation ‚Äî ideal for distraction-free writing or long reads.

## Conversation Shortcuts

| Action                          | Shortcut                             |
| ------------------------------- | ------------------------------------ |
| **Open conversation settings**  | `‚åò + ,` / `Ctrl + ,`                 |
| **Regenerate message**          | `‚åò + R` / `Ctrl + R`                 |
| **Delete last message**         | `‚åò + D` / `Ctrl + D`                 |
| **Delete and regenerate**       | `‚åò + Shift + R` / `Ctrl + Shift + R` |
| **New Topic**                   | `‚åò + N` / `Ctrl + N`                 |
| **Add message without sending** | `‚åò + Enter` / `Ctrl + Enter`         |
| **Edit message**                | `Ctrl + Alt` + double-click message  |

**Add message without sending** ‚Äî useful when you want to add context to the conversation without triggering an immediate response. The Agent will see the new message when it next replies.

## Custom Shortcuts

Most shortcuts can be customized. In the Shortcuts settings panel, click the shortcut you want to change, then press your desired key combination. Useful when:

- Your muscle memory conflicts with another app
- You want to prioritize actions you use most
- You're on a different keyboard layout

**Tip:** Start with the Command Menu (`‚åò + K` / `Ctrl + K`) ‚Äî it's the fastest way to jump to any Agent, Topic, or setting.

## Use Cases

**Power-user workflow** ‚Äî `‚åò + K` to open Command Menu ‚Üí type Agent name ‚Üí Enter. Switch Agents in under a second.

**Quick iteration** ‚Äî `‚åò + R` to regenerate. `‚åò + Shift + R` if you want to delete the last message first, then regenerate.

**Distraction-free writing** ‚Äî `‚åò + \` for Focus mode. No sidebar, no panels ‚Äî just the conversation.

**Multi-Agent work** ‚Äî Pin your top Agents to the sidebar, then use `Ctrl + 1‚Äì9` to switch between them without touching the mouse.
