# Cattura dello schermo 📸

L'applicazione **mAI** include uno strumento intuitivo di cattura dello schermo che consente agli utenti di catturare rapidamente aree dello schermo e inviarle direttamente nella chat per l'analisi visiva da parte dell'agente.

## Utilizzo dello strumento di cattura

Per effettuare uno screenshot:
1. Fai clic sull'icona **Fotocamera / Cattura** nella barra di inserimento della chat o usa la scorciatoia dedicata.
2. L'interfaccia si nasconderà temporaneamente per consentirti di selezionare l'area o la finestra che desideri catturare.
3. Una volta selezionata, l'immagine viene automaticamente allegata al campo di input della chat.

## Analisi visiva con May

Sfruttando le funzionalità multimodali dei modelli di linguaggio configurati in mAI, puoi porre domande sull'immagine catturata:
- "Spiegami questo messaggio di errore."
- "Traduci il testo presente in questa immagine."
- "Analizza il layout e suggerisci miglioramenti del design."
- "Scrivi il codice corrispondente a questo mockup UI."
