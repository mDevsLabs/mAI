# Agenti integrati 📦

**mAI** viene fornito per impostazione predefinita con diversi agenti di sistema preconfigurati per assisterti immediatamente nelle attività quotidiane senza richiedere alcuna configurazione manuale.

## Elenco degli agenti di sistema

Ecco i principali agenti integrati nell'applicazione:

1. **May**: La tua assistente principale versatile, pronta per tutte le attività di scrittura, brainstorming e aiuto generale.
2. **Sviluppatore software**: Un agente con un prompt di sistema incentrato sulla programmazione, la scrittura di codice pulito e il debug.
3. **Traduttore universale**: Specializzato nella traduzione fluida tra decine di lingue, rispettando le espressioni idiomatiche.
4. **Content Writer**: Ottimizzato per il marketing, la stesura di post sul blog e la scrittura di testi chiari per i social media.

## Utilizzo degli agenti integrati

Questi agenti sono immediatamente visibili nell'elenco degli agenti dopo l'installazione. Puoi avviare una nuova conversazione con uno di essi con un solo clic. Sebbene siano bloccati contro alcune modifiche di sistema, puoi personalizzare i loro parametri locali (come il modello di linguaggio associato) per ogni chat.
