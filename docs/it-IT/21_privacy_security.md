# Privacy e Sicurezza 🔒

La sicurezza dei dati e il rispetto della tua privacy sono principi fondamentali nella progettazione di **mAI**.

## Archiviazione locale crittografata

Tutti i tuoi dati sensibili sono memorizzati localmente sul tuo dispositivo:
- **Chiavi API**: Sono crittografate utilizzando algoritmi standard del settore prima di essere salvate nella memoria locale. Le tue chiavi API non vengono mai inviate ai nostri server.
- **Cronologia chat**: Le tue conversazioni rimangono sul tuo computer (a meno che tu non abiliti esplicitamente la sincronizzazione cloud sicura).

## Connessioni sicure

Tutte le richieste in uscita alle API utilizzano protocolli di comunicazione sicuri (HTTPS / SSL), garantendo che i dati scambiati non possano essere intercettati sulla rete.
