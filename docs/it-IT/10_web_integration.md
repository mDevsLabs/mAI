# Integrazione Web 🌐

**mAI** è progettato come un'applicazione multipiattaforma. Oltre all'app desktop, mAI è completamente accessibile tramite una versione Web ottimizzata, consentendo un accesso fluido da qualsiasi browser web.

## Funzionalità della versione Web

La versione Web di mAI offre quasi la totale parità di funzionalità con l'applicazione desktop:
- Interfaccia utente fluida, reattiva e veloce.
- Accesso a tutti i modelli di linguaggio utilizzando le proprie chiavi API (salvate localmente nel browser o sincronizzate).
- Utilizzo della maggior parte dei plugin (i plugin che richiedono privilegi di sistema locali, come l'accesso diretto al file system locale, potrebbero essere limitati).

## Sincronizzazione Cloud

Se utilizzi mAI su più dispositivi, puoi abilitare la sincronizzazione cloud sicura. I dati delle chat, gli agenti personalizzati e le preferenze possono essere salvati con crittografia end-to-end sul tuo server di hosting o tramite soluzioni di archiviazione cloud compatibili (come WebDAV).
