# FAQ ❓

Qui troverai le risposte alle domande più frequenti sulla configurazione e sull'utilizzo di **mAI**.

## Utilizzo generale

### Posso usare mAI senza una connessione Internet?
Sì, a condizione di utilizzare modelli locali tramite servizi come **Ollama** o LocalAI. Se si utilizzano modelli cloud (come OpenAI o Anthropic), è richiesta una connessione Internet attiva per comunicare con i loro server.

### Come aggiorno le mie chiavi API?
Vai su Impostazioni (`Ctrl + ,`) e apri la scheda **Modelli di linguaggio**. Da lì puoi aggiungere, modificare o rimuovere le chiavi API per ciascun fornitore supportato.

## Personalizzazione degli agenti

### Chi è May?
**May** è l'assistente virtuale predefinita intelligente in mAI. È altamente versatile e configurata per assisterti sin dal primo avvio.

### I miei agenti sono condivisi con altri utenti?
No. Per impostazione predefinita, tutti gli agenti personalizzati che crei sono memorizzati localmente e rimangono privati sul tuo dispositivo. Tuttavia, puoi esportarli manualmente per condividerli.
