# Guida rapida 🚀

Segui questi passaggi per configurare ed eseguire **mAI** sul tuo computer locale in pochi minuti.

## Requisiti di sistema

- **Node.js**: Versione 18.x o successiva (LTS consigliata).
- **Gestore pacchetti**: Si consiglia **pnpm** per l'installazione delle dipendenze.
- **Git**: Per clonare il repository.

## Installazione

1. **Clona il repository**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```
2. **Installa le dipendenze**:
   ```bash
   pnpm install
   ```
3. **Avvia il server di sviluppo**:
   ```bash
   pnpm run dev
   ```
4. **Apri il browser** all'indirizzo `http://localhost:3000` per visualizzare l'interfaccia utente di mAI e iniziare a chattare con la tua assistente **May**.
