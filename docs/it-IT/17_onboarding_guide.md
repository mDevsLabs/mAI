# Guida di Onboarding 🚀

Benvenuto in **mAI**! Questa guida è progettata per accompagnare i nuovi utenti passo dopo passo durante il primo avvio dell'applicazione.

## Passaggio 1: Il messaggio di benvenuto di May

Al primo avvio dell'applicazione, si aprirà automaticamente **May**, la tua assistente virtuale predefinita. Ti guiderà attraverso le prime fasi di configurazione e risponderà alle tue domande immediate sul funzionamento dell'app.

## Passaggio 2: Configurazione delle chiavi API

Affinché mAI funzioni, devi configurare almeno un fornitore di modelli di linguaggio:
1. Fai clic sul pulsante di configurazione suggerito da May o apri le impostazioni (`Ctrl + ,`).
2. Seleziona la scheda **Modelli di linguaggio**.
3. Inserisci la tua chiave API per il fornitore desiderato (OpenAI, Anthropic Claude, Google Gemini) o abilita un fornitore locale come Ollama.
4. Salva per archiviare la chiave in modo sicuro sul tuo dispositivo.

## Passaggio 3: La tua prima chat

1. Fai clic su **Nuova chat** nella barra laterale sinistra.
2. Scrivi il tuo messaggio nel campo di testo in basso.
3. Premi **Invio** per avviare la conversazione con May!
