# Sistema dei plugin 🔌

**mAI** dispone di un potente ecosistema di plugin che consente agli agenti conversazionali di superare i limiti delle loro conoscenze di base interagendo con strumenti e servizi esterni.

## Plugin integrati

Ecco i principali plugin disponibili per impostazione predefinita:

1. **Ricerca Web**: Consente all'agente di effettuare ricerche su Internet in tempo reale per ottenere informazioni aggiornate.
2. **Artefatti**: Consente di generare, modificare e visualizzare in anteprima codice, documenti o progetti UI in modo interattivo in una barra laterale dedicata.
3. **Meteo**: Interroga le API meteo per ottenere previsioni locali.
4. **Calcolatore di codice**: Esegue script locali per risolvere calcoli complessi in modo accurato.

## Attivazione e configurazione

Ogni agente può avere il proprio set di plugin abilitati. Puoi abilitare o disabilitare i plugin per un agente specifico tramite la scheda **Plugin** nelle impostazioni dell'agente. Alcuni plugin (come la ricerca web a pagamento) potrebbero richiedere chiavi API aggiuntive nelle impostazioni generali.
