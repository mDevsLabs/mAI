# File system locale 📂

L'applicazione desktop **mAI** offre una profonda integrazione con il file system del tuo sistema operativo, consentendoti di accedere e analizzare i tuoi file locali in tutta sicurezza.

## Lettura e analisi dei file

Puoi inviare file direttamente ai tuoi agenti (come **May**) per l'analisi o l'elaborazione:
- **Trascina e rilascia (Drag and Drop)**: Trascina un documento di testo, un file PDF, un foglio Excel o un'immagine direttamente nell'area della chat di mAI.
- **Allegato**: Fai clic sull'icona della graffetta nella chat per sfogliare il file explorer.
- I file vengono analizzati ed elaborati localmente e solo il testo estratto (o l'immagine stessa per i modelli multimodali) viene inviato al LLM.

## Limiti di sicurezza

Per mantenere sicuro il tuo sistema:
- mAI non legge mai i file senza la tua azione esplicita.
- Puoi specificare una directory dedicata nelle impostazioni generali per limitare l'accesso di mAI solo a quella cartella.
