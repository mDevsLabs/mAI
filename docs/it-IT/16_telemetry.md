# Telemetria 📊

In **mAI**, la trasparenza e il rispetto della tua privacy sono le nostre massime priorità. Questo documento descrive in dettaglio il funzionamento della telemetria all'interno dell'applicazione e quali dati possono essere raccolti.

## Raccolta dati anonimi

Al fine di aiutarci a migliorare la stabilità dell'applicazione e a identificare bug dell'interfaccia, mAI include una telemetria anonima e opzionale. Se abiliti questa opzione, le seguenti informazioni verranno inviate in formato completamente anonimo:
- Rapporti sui crash ed eccezioni JavaScript non gestite.
- Statistiche generali sull'uso (es. il numero di agenti creati, i modelli di linguaggio più selezionati).
- Informazioni di base sul sistema (versione dell'applicazione, sistema operativo, risoluzione dello schermo).

## Cosa non raccogliamo MAI

La telemetria di mAI rispetta rigorosamente la tua privacy. Non raccogliamo, leggiamo o memorizziamo **mai**:
- Il contenuto dei tuoi messaggi o conversazioni.
- I tuoi prompt di sistema o configurazioni personalizzate degli agenti.
- Le tue chiavi API o credenziali d'accesso.
- Informazioni personali identificabili (nomi, indirizzi e-mail, ecc.).
