# Menu dei comandi 🔍

Il menu dei comandi rapidi di **mAI** è il centro di controllo principale per navigare e gestire l'applicazione senza mai toccare il mouse. Si attiva all'istante con la scorciatoia `Ctrl + P` (o `Cmd + P` su macOS).

## Funzionalità principali

Una volta aperto il menu, puoi digitare comandi per:
- **Cambiare agente**: Digita il nome di un agente (come "May" o "Sviluppatore") per passare all'istante a quel profilo.
- **Cercare o creare chat**: Cerca tra le conversazioni esistenti per parole chiave o digita `/new` per avviare una nuova chat.
- **Accedere alle impostazioni**: Digita `impostazioni` per andare direttamente alla pagina delle opzioni.
- **Cambiare modello**: Seleziona un LLM diverso per la chat attiva (ad esempio, digitando "gpt" o "claude" filtrerai i modelli disponibili).
