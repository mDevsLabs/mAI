# Impostazioni generali ⚙️

Le impostazioni generali di **mAI** ti consentono di adattare l'applicazione alle tue preferenze estetiche, linguistiche e funzionali.

## Interfaccia utente e temi

- **Tema**: Scegli tra tema chiaro, tema scuro o consenti all'applicazione di seguire le preferenze di sistema del tuo computer.
- **Colore principale**: Personalizza la tonalità dei pulsanti attivi e degli elementi grafici interattivi primari.
- **Lingua dell'interfaccia**: Seleziona la lingua dell'applicazione.
- **Dimensione caratteri**: Regola la dimensione del testo per un comfort di lettura ottimale.

## Gestione dei dati

- **Esporta dati**: Esporta tutte le tue conversazioni, gli agenti personalizzati e le impostazioni in un unico file di archivio JSON.
- **Importa dati**: Ripristina un backup precedente o importa configurazioni di agenti esterni.
- **Ripristina applicazione**: Cancella definitivamente tutti i dati locali (cronologia, chiavi API, impostazioni) per ripristinare l'applicazione allo stato iniziale di installazione.
