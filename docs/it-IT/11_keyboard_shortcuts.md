# Scorciatoie da tastiera ⌨️

Per massimizzare la produttività e navigare rapidamente senza usare il mouse, **mAI** include una serie di scorciatoie da tastiera configurabili.

## Scorciatoie di navigazione globale

- `Ctrl + P` (o `Cmd + P` su Mac): Apre il menu dei comandi rapidi.
- `Ctrl + ,` (o `Cmd + ,` su Mac): Apre le impostazioni dell'applicazione.
- `Ctrl + N` (o `Cmd + N` su Mac): Avvia una nuova conversazione.
- `Ctrl + \` (o `Cmd + \` su Mac): Mostra o nasconde la barra laterale sinistra.

## Scorciatoie nella chat

- `Invio`: Invia il messaggio.
- `Maiusc + Invio`: Inserisce un'interruzione di riga nel campo di inserimento.
- `Ctrl + E`: Modifica l'ultimo messaggio inviato.
- `Ctrl + R`: Rigenera l'ultima risposta dell'agente.
- `Ctrl + Canc`: Elimina la conversazione corrente.
