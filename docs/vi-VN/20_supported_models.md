# Mô hình Ngôn ngữ Được hỗ trợ 🧠

**mAI** là một giao diện độc lập với mô hình. Nó hỗ trợ nhiều nhà cung cấp trí tuệ nhân tạo dựa trên đám mây thương mại và mã nguồn mở cục bộ.

## Các Nhà cung cấp Đám mây Lớn

Bạn có thể cấu hình các nhà cung cấp đám mây sau bằng cách nhập khóa API cá nhân của mình:
- **OpenAI**: GPT-4o, GPT-4 Turbo, GPT-3.5, v.v.
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku.
- **Google Gemini**: Gemini 1.5 Pro, Gemini 1.5 Flash.
- **Mistral AI**: Mistral Large, Mistral Medium, Codestral.
- **Groq**: Truy cập cực nhanh vào các mô hình mã nguồn mở như Llama 3 và Mixtral.

## Tích hợp Mô hình Cục bộ

Đối với những người dùng quan tâm đến sự riêng tư hoàn toàn của dữ liệu của họ, mAI hỗ trợ đầy đủ việc thực thi cục bộ:
- **Ollama**: Tích hợp gốc được đơn giản hóa. Hãy chạy Ollama ở chế độ nền, và mAI sẽ tự động phát hiện các mô hình được cài đặt cục bộ của bạn (chẳng hạn như Llama 3, Mistral hoặc Phi-3).
- **Các API tương thích với OpenAI**: Kết nối bất kỳ máy chủ suy luận cục bộ hoặc từ xa nào (như LocalAI hoặc LM Studio) bằng cách chỉ định URL gốc API tùy chỉnh.
