# Tích hợp Web 🌐

**mAI** được thiết kế như một ứng dụng đa nền tảng. Ngoài ứng dụng dành cho máy tính để bàn, mAI có thể truy cập đầy đủ thông qua phiên bản web được tối ưu hóa, cho phép truy cập mượt mà từ bất kỳ trình duyệt web nào.

## Các tính năng của Phiên bản Web

Phiên bản web của mAI cung cấp tính năng gần như tương đương hoàn toàn với ứng dụng dành cho máy tính để bàn:
- Giao diện người dùng mượt mà, phản hồi và phản ứng nhanh.
- Truy cập vào tất cả các mô hình ngôn ngữ bằng cách sử dụng các khóa API của riêng bạn (được lưu cục bộ trong trình duyệt của bạn hoặc được đồng bộ hóa).
- Sử dụng hầu hết các plugin (các plugin yêu cầu đặc quyền hệ thống cục bộ, như truy cập hệ thống tệp cục bộ trực tiếp, có thể bị hạn chế).

## Đồng bộ hóa Đám mây

Nếu bạn sử dụng mAI trên nhiều thiết bị, bạn có thể bật tính năng đồng bộ hóa đám mây an toàn. Dữ liệu trò chuyện, tác tử tùy chỉnh và tùy chọn của bạn có thể được lưu bằng mã hóa đầu cuối trên máy chủ lưu trữ của riêng bạn hoặc thông qua các giải pháp lưu trữ đám mây tương thích (như WebDAV hoặc các dịch vụ cơ sở dữ liệu được đồng bộ hóa).
