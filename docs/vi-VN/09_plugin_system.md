# Hệ thống Plugin 🔌

**mAI** có một hệ sinh thái plugin mạnh mẽ cho phép các tác tử hội thoại vượt qua các hạn chế của dữ liệu đào tạo của chúng bằng cách tương tác với các công cụ và dịch vụ bên ngoài.

## Plugin Tích hợp sẵn

Dưới đây là các plugin chính có sẵn theo mặc định:

1. **Duyệt Web (Web Browsing)**: Cho phép tác tử tìm kiếm internet theo thời gian thực để tìm nạp thông tin cập nhật mới nhất.
2. **Hiện vật (Artifacts)**: Cho phép tạo, chỉnh sửa và xem trước mã, tài liệu hoặc thiết kế giao diện người dùng một cách tương tác trong một thanh bên chuyên dụng.
3. **Thời tiết (Weather)**: Truy vấn API thời tiết để lấy dự báo địa phương.
4. **Máy tính Mã (Code Calculator)**: Thực thi các tập lệnh cục bộ để giải quyết chính xác các phép tính phức tạp.

## Kích hoạt và Cấu hình

Mỗi tác tử có thể có bộ plugin được bật của riêng nó. Bạn có thể bật hoặc tắt các plugin cho một tác tử cụ thể thông qua tab **Plugins** trong cài đặt tác tử. Một số plugin (như tìm kiếm web thương mại) có thể yêu cầu cài đặt bổ sung, chẳng hạn như khóa API, trong cấu hình chung.
