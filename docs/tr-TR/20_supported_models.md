# Desteklenen Dil Modelleri 🧠

**mAI**, model bağımsız (model-agnostic) bir arayüzdür. Çok çeşitli ticari bulut tabanlı ve yerel açık kaynaklı yapay zeka sağlayıcılarını destekler.

## Büyük Bulut Sağlayıcıları

Kişisel API anahtarınızı girerek aşağıdaki bulut sağlayıcılarını yapılandırabilirsiniz:
- **OpenAI**: GPT-4o, GPT-4 Turbo, GPT-3.5 vb.
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku.
- **Google Gemini**: Gemini 1.5 Pro, Gemini 1.5 Flash.
- **Mistral AI**: Mistral Large, Mistral Medium, Codestral.
- **Groq**: Llama 3 ve Mixtral gibi açık kaynaklı modellere ultra hızlı erişim.

## Yerel Modeller Entegrasyonu

Verilerinin tam gizliliğine önem veren kullanıcılar için mAI yerel yürütmeyi tamamen destekler:
- **Ollama**: Basitleştirilmiş yerel entegrasyon. Arka planda Ollama'yı çalıştırın; mAI yerel olarak yüklenmiş modellerinizi (Llama 3, Mistral veya Phi-3 gibi) otomatik olarak algılayacaktır.
- **OpenAI Uyumlu API'ler**: Özel bir API taban URL'si belirterek herhangi bir yerel veya uzak çıkarım (inference) sunucusunu (LocalAI veya LM Studio gibi) bağlayın.
