# Klavye Kısayolları ⌨️

Verimliliğinizi en üst düzeye çıkarmak ve klavyenizden ayrılmadan hızlıca gezinmek için **mAI**, bir dizi yapılandırılabilir kısayol içerir.

## Global Gezinme Kısayolları

- `Ctrl + P` (Mac'te `Cmd + P`): Hızlı komut menüsünü açar.
- `Ctrl + ,` (Mac'te `Cmd + ,`): Uygulama ayarlarını açar.
- `Ctrl + N` (Mac'te `Cmd + N`): Yeni bir konuşma başlatır.
- `Ctrl + \` (Mac'te `Cmd + \`): Sol kenar çubuğunu açar/kapatır.

## Sohbet Kısayolları

- `Enter`: Mesajı gönderir.
- `Shift + Enter`: Giriş alanına satır sonu ekler.
- `Ctrl + E`: Son gönderdiğiniz mesajı düzenler.
- `Ctrl + R`: Temsilcinin son yanıtını yeniden oluşturur.
- `Ctrl + Delete`: Mevcut tartışmayı siler.

## Ekran Yönetimi Kısayolları

- `Alt + 1`: Sohbetler (Chats) sekmesine geçer.
- `Alt + 2`: Temsilciler (Agents) sekmesine geçer.
- `Alt + 3`: Keşfet (Discover / Marketplace) sekmesine geçer.
