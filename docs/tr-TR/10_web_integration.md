# Web Entegrasyonu 🌐

**mAI**, platformlar arası bir uygulama olarak tasarlanmıştır. Masaüstü uygulamasına ek olarak mAI, optimize edilmiş bir web sürümü aracılığıyla tamamen erişilebilirdir ve herhangi bir web tarayıcısından akıcı erişim sağlar.

## Web Sürümü Özellikleri

mAI'nin web sürümü, masaüstü uygulamasıyla neredeyse tamamen aynı özellikleri sunar:
- Akıcı, duyarlı (responsive) ve reaktif kullanıcı arayüzü.
- Kendi API anahtarlarınızı kullanarak tüm dil modellerine erişim (tarayıcınızda yerel olarak kaydedilir veya senkronize edilir).
- Çoğu eklentinin kullanımı (doğrudan yerel dosya sistemine erişim gibi yerel sistem ayrıcalıkları gerektiren eklentiler kısıtlanabilir).

## Bulut Senkronizasyonu

mAI'yi birden fazla cihazda kullanıyorsanız, güvenli bulut senkronizasyonunu etkinleştirebilirsiniz. Sohbet verileriniz, özel temsilcileriniz ve tercihleriniz kendi barındırma sunucunuzda veya uyumlu bulut depolama çözümleri (WebDAV veya senkronize veritabanı hizmetleri gibi) aracılığıyla uçtan uca şifreleme ile kaydedilebilir.
