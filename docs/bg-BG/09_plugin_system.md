# Система от плъгини 🔌

**mAI** разполага с мощна екосистема от плъгини, която позволява на разговорните агенти да преодолеят ограниченията на данните си за обучение чрез взаимодействие с външни инструменти и услуги.

## Вградени плъгини

Ето основните плъгини, налични по подразбиране:

1. **Уеб браузване (Web Browsing)**: Позволява на агента да търси в интернет в реално време, за да извлича актуална информация.
2. **Артефакти (Artifacts)**: Активира генерирането, редактирането и прегледа на код, документи или UI дизайни интерактивно в специална странична лента.
3. **Времето (Weather)**: Запитва API за времето, за да извлече локални прогнози.
4. **Калкулатор за код (Code Calculator)**: Изпълнява локални скриптове за точно решаване на сложни изчисления.

## Активиране и конфигуриране

Всеки агент може да има свой собствен набор от активирани плъгини. Можете да активирате или деактивирате плъгини за конкретен агент чрез раздела **Плъгини (Plugins)** in настройките на агента. Някои плъгини (като комерсиално уеб търсене) може да изискват допълнителни настройки, като например API ключове, в общата конфигурация.
