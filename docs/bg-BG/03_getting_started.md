# Първи стъпки ⚡

Това ръководство ви помага да настроите и стартирате **mAI** локално на вашата машина за разработка.

## Предварителни изисквания

Уверете се, че имате инсталирани следните:
- **Node.js** (препоръчва се v18 или по-нова)
- **pnpm** или **bun** за управление на пакети и работни пространства

## Инсталация

1. **Клонирайте хранилището**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **Инсталирайте зависимостите**:
   С pnpm:
   ```bash
   pnpm install
   ```
   С bun:
   ```bash
   bun install
   ```

3. **Стартирайте сървъра за разработка**:
   ```bash
   pnpm dev
   # или
   bun run dev
   ```

Приложението ще бъде достъпно във вашия браузър на адрес `http://localhost:3010`. При първото стартиране ще бъдете посрещнати от **May**, вашия основен асистент, за да конфигурирате първоначалните си предпочитания!
