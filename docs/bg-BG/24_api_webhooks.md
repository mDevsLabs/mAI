# API и уеб куки (Webhooks) 🔗

**mAI** предоставя програмни интерфейси, за да позволи външна интеграция с вашите собствени приложения, уеб услуги или скриптове за автоматизация.

## Локално API (HTTP)

Настолното приложение mAI може да стартира сигурен локален уеб сървър на определен порт. Този сървър излага няколко крайни точки (endpoints):
- `POST /api/chat`: Изпраща съобщение до конкретен агент и връща структуриран отговор (поддържа стандартен или стрийминг формат).
- `GET /api/agents`: Изброява всички налични агенти и техните конфигурации.
- `POST /api/agents`: Позволява програмно създаване на нов агент.

## Интеграция на уеб куки (Webhooks)

Можете да конфигурирате изходящи уеб куки в разширените настройки на mAI. Уеб куката автоматично изпраща JSON данни към външен URL адрес, когато възникнат конкретни събития:
- **Завършено съобщение в чата**: Задейства се веднага след като агентът приключи с генерирането на отговор.
- **Създаден агент**: Задейства се, когато бъде добавен нов профил на агент.
- **Системна грешка**: Задейства се по време на проблеми с валидирането на API ключове или прекъсвания на доставчика на LLM.
