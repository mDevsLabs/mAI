# Ръководство за разработка 🛠️

Това ръководство предоставя основни инструкции за разработчици, които искат да допринесат за проекта **mAI** или да стартират изходния код в режим на разработка.

## Структура на проекта

mAI е структуриран като модерно монорепо (използващо pnpm и Turborepo):
- `apps/web`: Основното Next.js приложение.
- `packages/`: Споделени модули и конфигурации за многократна употреба (UI компоненти, типове, помощни програми).

## Стартиране в режим на разработка

1. **Предварителни изисквания**: Уверете се, че имате инсталирани Node.js (препоръчва се LTS версия) и pnpm или bun.
2. **Инсталиране на зависимости**:
   ```bash
   pnpm install
   ```
3. **Стартиране на сървъра за разработка**:
   ```bash
   pnpm run dev
   ```
   Приложението ще бъде достъпно локално на адрес `http://localhost:3000`.

## Тестване

Ние използваме **Vitest** за нашите тестови пакети:
- Изпълнение на тестове веднъж: `pnpm run test`
- Изпълнение на тестове в режим на наблюдение (watch mode): `pnpm run test:watch`

Моля, уверете се, че всички тестове преминават успешно, преди да изпратите каквито и да било промени.
