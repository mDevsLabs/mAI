# FAQ ❓

Here are the answers to the most frequently asked questions about configuring and using **mAI**.

## General Usage

### Can I use mAI without an internet connection?
Yes, you can, provided you are using local models run through services like **Ollama** or LocalAI. If you are using cloud models (such as OpenAI or Anthropic), an active internet connection is required to communicate with their servers.

### How do I update my API keys?
Go to Settings (`Ctrl + ,`), and open the **Language Models** tab. From there, you can add, change, or remove API keys for each supported provider.

## Agent Customization

### Who is May?
**May** is the default intelligent core assistant in mAI. She is highly versatile and configured to assist you from your very first launch.

### Are my agents shared with other users?
No. By default, all custom agents you create are stored locally and remain private to your device. You can, however, export them manually to share them with others.

## Cost and Billing

### Is mAI free to use?
The mAI application itself is completely free and open-source. However, utilizing commercial cloud models (like OpenAI or Anthropic) requires you to supply your own API keys. You will only pay for your actual API usage directly to the model providers.
