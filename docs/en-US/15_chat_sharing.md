# Chat Sharing 📤

**mAI** makes it easy to share your conversations with colleagues, friends, or on social media using beautiful visual and text-based sharing tools.

## Generating a Share Image

To create an aesthetic and stylized preview of your chat:
1. Click the **Share** button (export icon) at the top right of the discussion screen.
2. Select the **Generate Image** option.
3. Customize the output:
   - Select the specific messages you want to include.
   - Choose a background style (modern gradient, solid color, or transparent).
   - Show or hide the agent avatar (such as **May**) and the application logo.
4. Click **Download** to save the image, or **Copy to Clipboard** to paste it directly into another application.

## Sharing via a Public Link (Share Link)

If you are using the Web version or have configured a shared storage service:
- Select the **Create Share Link** option.
- mAI will generate a unique, anonymized URL containing the transcript of your conversation.
- Anyone with the link will be able to read the conversation in a read-only viewer and import it into their own mAI instance to continue the discussion.
