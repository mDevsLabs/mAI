# Changelog v0.0.6 - 30-05-2026

## Changes Made

### 🧩 New Features
* **ExtensionsSidebarPanel**: Designed and implemented the dynamic lateral navigation panel to access mServices (Quizzly, Vibs, Cooker, Health) with custom cubic Bezier transitions and a dedicated Zustand store (`useExtensionStore`).

### 🏷️ Branding & Display
* Replaced all occurrences of **LobeHub** with **mAI** and **Lobe AI** with **May** in the user-facing display across all JSON locale files in the project.

### 🔌 Integrations & MCP
* Completely hid default MCP server providers like **X (Twitter)** and **YouTube** from the user interface.

### 🧹 Cleaning & Documentation
* Recursively deleted all Chinese-language (`zh-CN`) documentation files under the `docs/` directory.
* Bumped the global project version to `0.0.6` in `package.json` and updated `CHANGELOG.md` at the root.
