# User Memory 🧠

User memory in **mAI** allows your agents, and particularly **May**, to remember key facts about you across different conversations. This makes the user experience much more personalized and seamless.

## How Memory Works

As you interact with mAI, the application automatically extracts relevant contextual information and stores it in a local memory database. This can include:
- Your language preferences or writing style.
- Projects you are currently working on.
- Your development preferences (preferred frameworks, code styles).
- Personal details you choose to share (such as your name or professional role).

## Managing Your Memory

You have total control over the information stored by mAI:
1. Go to the general application settings.
2. Open the **Memory** tab.
3. You will see a list of facts stored as cards.
4. You can manually edit any entry, or permanently delete a fact from the memory database so the agent forgets it.
