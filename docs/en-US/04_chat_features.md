# Chats and Rendering Features 💬

The **mAI** conversational interface has been designed to offer a fluid, interactive experience, packed with visual features and precise control options.

## Core Chat Features

- **Inline Editing**: You can edit any of your previous messages at any point. mAI will automatically re-run the conversation from that turn onwards.
- **Rich Markdown Rendering**: mAI handles full Markdown formatting: tables, LaTeX mathematical equations, syntax-highlighted code blocks, and interactive preview panes.
- **Quick Actions**: Each message bubble features a context menu allowing you to copy the raw content, regenerate the AI response, edit, or delete the message from your history.
- **History Tracking**: Chat sessions are archived locally in the SQLite database, allowing you to instantly search and recover your past conversations.
