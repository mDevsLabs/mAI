# Getting Started ⚡

This guide helps you set up and run **mAI** locally on your development machine.

## Prerequisites

Ensure you have the following installed:
- **Node.js** (v18 or higher recommended)
- **pnpm** or **bun** for package and workspace management

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **Install dependencies**:
   With pnpm:
   ```bash
   pnpm install
   ```
   With bun:
   ```bash
   bun install
   ```

3. **Run the development server**:
   ```bash
   pnpm dev
   # or
   bun run dev
   ```

The application will be accessible in your browser at `http://localhost:3010`. On your first launch, you will be welcomed by **May**, your core assistant, to configure your initial preferences!
