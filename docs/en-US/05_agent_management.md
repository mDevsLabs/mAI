# Agent Management 👥

In **mAI**, you can create and configure specialized agents to address specific needs and use cases. Agent management is done directly through the graphical user interface or via configuration files.

## Creating a New Agent

To create an agent in mAI:
1. Go to the left sidebar and click on the **Agents** button.
2. Click on **Create Agent** at the top of the list.
3. Fill in the basic information:
   - **Name**: The identifier of your agent.
   - **Avatar**: Choose an icon, emoji, or upload an image.
   - **Description**: A quick summary of the agent's role.

## Editing an Existing Agent

You can edit any custom agent by clicking on the associated settings icon in the agents list. You will have access to advanced configurations such as defining the system prompt, choosing the default language model, and managing its plugins.

## Deleting an Agent

If a custom agent is no longer needed, you can delete it permanently via the agent's settings options. Existing discussions linked to this agent will not be deleted, but the agent itself will no longer be available for new conversations.
