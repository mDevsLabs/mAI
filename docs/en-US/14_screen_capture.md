# Screen Capture 📸

The **mAI** application includes an intuitive screen capture tool that allows users to quickly grab portions of their screen and send them directly into the discussion for visual analysis by the agent.

## Using the Capture Tool

To take a screenshot:
1. Click the **Camera / Capture** icon located in the chat input bar or use the dedicated shortcut.
2. The UI temporarily hides to let you select the region or window you want to capture.
3. Once selected, the image is automatically attached to your chat input box.

## Visual Analysis with May

Leveraging the multimodal capabilities of the language models configured in mAI, you can ask questions about the captured image:
- "Explain this error message to me."
- "Translate the text present in this image."
- "Review this layout and suggest design improvements."
- "Write the code corresponding to this UI mockup."
