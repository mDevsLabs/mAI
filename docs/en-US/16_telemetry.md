# Telemetry 📊

In **mAI**, transparency and respect for your privacy are our top priorities. This document details how telemetry works in the application and what data may be collected.

## Anonymous Data Collection

To help us improve the application's stability and identify interface bugs, mAI includes anonymous telemetry. If you opt-in, the following information is transmitted in a fully anonymized format:
- Crash reports and unhandled JavaScript exceptions.
- Global usage statistics (for example, the number of custom agents created, the most frequently selected language models, or plugin activation rates).
- Basic system information (application version, operating system, screen resolution).

## What We NEVER Collect

The telemetry in mAI strictly respects your privacy. We **never** collect, read, or store:
- The content of your messages or conversations.
- Your custom system prompts or agent configurations.
- Your API keys or credential details.
- Any personally identifiable information (names, email addresses, etc.).

## Enabling or Disabling Telemetry

You can choose to participate in improving mAI or opt-out at any time:
1. Go to the application **Settings**.
2. Click on the **About / Telemetry** tab.
3. Toggle the **Allow anonymous crash reports and usage statistics** option.
