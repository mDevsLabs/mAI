# General Settings ⚙️

The general settings in **mAI** allow you to adjust the application to your aesthetic, linguistic, and functional preferences.

## User Interface and Themes

- **Theme**: Choose between light mode, dark mode, or let the application follow your computer's system preferences.
- **Accent Color**: Customize the color of active buttons and key interactive elements of the graphical UI.
- **Interface Language**: Select the application language (French, English, etc.).
- **Font Size**: Adjust the text size for optimal reading comfort.

## Data Management

- **Export Data**: Export all your conversations, custom agents, and configurations in a single JSON archive file.
- **Import Data**: Restore a previous backup or import external agent configurations.
- **Reset Application**: Permanently erase all local data (history, API keys, settings) to restore the application to its original state.
