# Atajos de Teclado ⌨️

Para maximizar su productividad y navegar rápidamente sin soltar el teclado, **mAI** incluye una serie de atajos configurables.

## Atajos de Navegación Global

- `Ctrl + P` (o `Cmd + P` en Mac): Abre el menú de comandos rápidos.
- `Ctrl + ,` (o `Cmd + ,` en Mac): Abre la configuración de la aplicación.
- `Ctrl + N` (o `Cmd + N` en Mac): Inicia una nueva conversación.
- `Ctrl + \` (o `Cmd + \` en Mac): Muestra/oculta la barra lateral izquierda.

## Atajos de Chat

- `Enter`: Enviar el mensaje.
- `Shift + Enter`: Insertar un salto de línea en el campo de entrada.
- `Ctrl + E`: Editar su último mensaje enviado.
- `Ctrl + R`: Regenerar la última respuesta del agente.
- `Ctrl + Delete`: Eliminar la conversación actual.

## Atajos de Gestión de Pantalla

- `Alt + 1`: Cambiar a la pestaña de Chats.
- `Alt + 2`: Cambiar a la pestaña de Agentes.
- `Alt + 3`: Cambiar a la pestaña de Descubrir (Marketplace).
