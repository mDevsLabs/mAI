# Captura de Pantalla 📸

La aplicación **mAI** incluye una herramienta de captura de pantalla intuitiva que permite a los usuarios tomar fragmentos de su pantalla rápidamente y enviarlos directamente al chat para un análisis visual por parte del agente.

## Uso de la Herramienta de Captura

Para realizar una captura de pantalla:
1. Haga clic en el icono de **Cámara / Captura** situado en la barra de entrada del chat o use el atajo dedicado.
2. La interfaz de usuario se oculta temporalmente para permitirle seleccionar la región o ventana que desea capturar.
3. Una vez seleccionada, la imagen se adjunta automáticamente a su cuadro de entrada de chat.

## Análisis Visual con May

Aprovechando las capacidades multimodales de los modelos de lenguaje configurados en mAI, puede hacer preguntas sobre la imagen capturada:
- "Explícame este mensaje de error."
- "Traduce el texto que aparece en esta imagen."
- "Revisa este diseño y sugiere mejoras de interfaz."
- "Escribe el código correspondiente a este boceto de interfaz de usuario."
