# Guía de inicio rápido 🚀

Siga estos pasos para configurar y ejecutar **mAI** en su computadora local en unos pocos minutos.

## Requisitos del sistema

- **Node.js**: Versión 18.x o superior (se recomienda LTS).
- **Gestor de paquetes**: Se recomienda **pnpm** para la instalación de dependencias.
- **Git**: Para clonar el repositorio.

## Instalación

1. **Clone el repositorio**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```
2. **Instale las dependencias**:
   ```bash
   pnpm install
   ```
3. **Inicie el servidor de desarrollo**:
   ```bash
   pnpm run dev
   ```
4. **Abra su navegador** en `http://localhost:3000` para ver la interfaz de usuario de mAI y comenzar a chatear con su asistente **May**.
