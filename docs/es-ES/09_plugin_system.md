# Sistema de Plugins 🔌

**mAI** cuenta con un potente ecosistema de plugins que permite a los agentes conversacionales superar las limitaciones de sus datos de entrenamiento interactuando con herramientas y servicios externos.

## Plugins Integrados

Estos son los principales plugins disponibles por defecto:

1. **Navegación Web**: Permite al agente buscar en internet en tiempo real para obtener información actualizada.
2. **Artefactos**: Permite generar, editar y previsualizar código, documentos o diseños de interfaz de usuario de forma interactiva en una barra lateral dedicada.
3. **Clima**: Consulta las API del clima para obtener pronósticos locales.
4. **Calculadora de Código**: Ejecuta scripts locales para resolver cálculos complejos con precisión.

## Activación y Configuración

Cada agente puede tener su propio conjunto de plugins habilitados. Puede activar o desactivar plugins para un agente específico a través de la pestaña **Plugins** en la configuración del agente. Algunos plugins (como la búsqueda web comercial) pueden requerir configuraciones adicionales, como claves API, en la configuración general.
