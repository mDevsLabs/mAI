# Memoria de Usuario 🧠

La memoria de usuario en **mAI** permite a sus agentes, y en particular a **May**, recordar datos clave sobre usted a lo largo de diferentes conversaciones. Esto hace que la experiencia del usuario sea mucho más personalizada y fluida.

## Cómo Funciona la Memoria

A medida que interactúa con mAI, la aplicación extrae automáticamente información contextual relevante y la almacena en una base de datos de memoria local. Esto puede incluir:
- Sus preferencias de idioma o estilo de escritura.
- Proyectos en los que está trabajando actualmente.
- Sus preferencias de desarrollo (frameworks preferidos, estilos de código).
- Detalles personales que decida compartir (como su nombre o rol profesional).

## Gestión de su Memoria

Tiene un control total sobre la información almacenada por mAI:
1. Vaya a la configuración general de la aplicación.
2. Abra la pestaña **Memoria**.
3. Verá una lista de hechos almacenados en forma de tarjetas.
4. Puede editar manualmente cualquier entrada, o eliminar permanentemente un hecho de la base de datos de memoria para que el agente lo olvide.
