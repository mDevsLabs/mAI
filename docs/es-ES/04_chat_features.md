# Funciones de chat 💬

**mAI** ofrece una interfaz de chat intuitiva y rica en funciones para aumentar su productividad al trabajar con modelos de inteligencia artificial.

## Funciones principales

- **Renderizado de Markdown**: Soporte completo para mostrar texto formateado, tablas, fórmulas matemáticas (LaTeX) y resaltado de sintaxis para bloques de código.
- **Edición de mensajes**: Puede editar cualquier mensaje enviado para redirigir la conversación.
- **Regeneración de respuestas**: Si no está satisfecho con la respuesta de un agente, haga clic en "Regenerar" para obtener una nueva.
- **Bifurcación (Branching)**: Cree diferentes ramas en una conversación para probar soluciones alternativas.
- **Transmisión en tiempo real**: Las respuestas se transmiten palabra por palabra en tiempo real.
