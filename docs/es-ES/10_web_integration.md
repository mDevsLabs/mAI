# Integración Web 🌐

**mAI** está diseñado como una aplicación multiplataforma. Además de la aplicación de escritorio, mAI es totalmente accesible a través de una versión web optimizada, lo que permite un acceso fluido desde cualquier navegador web.

## Características de la Versión Web

La versión web de mAI ofrece una paridad de características casi completa con la aplicación de escritorio:
- Interfaz de usuario fluida, adaptativa y reactiva.
- Acceso a todos los modelos de lenguaje utilizando sus propias claves API (guardadas localmente en su navegador o sincronizadas).
- Uso de la mayoría de los plugins (los plugins que requieren privilegios del sistema local, como el acceso directo al sistema de archivos local, pueden estar restringidos).

## Sincronización en la Nube

Si utiliza mAI en varios dispositivos, puede habilitar la sincronización segura en la nube. Sus datos de chat, agentes personalizados y preferencias se pueden guardar con cifrado de extremo a extremo en su propio servidor de alojamiento o a través de soluciones de almacenamiento en la nube compatibles (como WebDAV o servicios de base de datos sincronizados).
