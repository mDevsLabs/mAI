# P+F (Preguntas Frecuentes) ❓

Aquí encontrará las respuestas a las preguntas más frecuentes sobre la configuración y el uso de **mAI**.

## Uso General

### ¿Puedo usar mAI sin conexión a Internet?
Sí, puede hacerlo siempre que utilice modelos locales ejecutados a través de servicios como **Ollama** o LocalAI. Si utiliza modelos en la nube (como OpenAI o Anthropic), se requiere una conexión a Internet activa para comunicarse con sus servidores.

### ¿Cómo actualizo mis claves API?
Vaya a Configuración (`Ctrl + ,`) y abra la pestaña **Modelos de Lenguaje**. Desde allí, puede añadir, cambiar o eliminar claves API para cada proveedor compatible.

## Personalización de Agentes

### ¿Quién es May?
**May** es la asistente principal inteligente por defecto en mAI. Es muy versátil y está configurada para ayudarle desde el primer inicio de la aplicación.

### ¿Se comparten mis agentes con otros usuarios?
No. Por defecto, todos los agentes personalizados que cree se almacenan localmente y siguen siendo privados en su dispositivo. Sin embargo, puede exportarlos manualmente para compartirlos con otros.

## Costos y Facturación

### ¿Es mAI de uso gratuito?
La aplicación mAI en sí es completamente gratuita y de código abierto. Sin embargo, para utilizar modelos comerciales en la nube (como OpenAI o Anthropic), debe proporcionar sus propias claves API. Solo pagará por su uso real de la API directamente a los proveedores de modelos.
