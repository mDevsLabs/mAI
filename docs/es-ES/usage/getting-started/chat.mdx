# Interfaz de Chat y Ramificación

mAI ofrece una interfaz de chat sofisticada que va más allá de las conversaciones lineales. Explora múltiples caminos y direcciones en tus discusiones con agentes de IA utilizando la ramificación de conversaciones, la edición de mensajes y el soporte de contenido enriquecido.

## Funciones Principales de Chat

La interfaz de chat proporciona una experiencia moderna y receptiva optimizada tanto para escritorio como para móviles:

- **Respuestas fluidas en tiempo real**: Mira cómo se generan las respuestas de la IA en tiempo real con animaciones suaves.
- **Soporte de contenido enriquecido**: Muestra texto formateado, bloques de código, imágenes y elementos multimedia.
- **Edición de mensajes**: Edita mensajes anteriores para refinar tus preguntas.
- **Reacciones a mensajes**: Reacciona a los mensajes para dar una respuesta rápida.
- **Preservación del contexto**: Las conversaciones mantienen todo el contexto para discusiones coherentes.

## Ramificación de Conversaciones

Una de las características más potentes de mAI es la ramificación de conversaciones, que transforma los chats lineales en estructuras dinámicas en forma de árbol.

### Cómo funciona la ramificación

Desde cualquier mensaje de tu conversación, puedes crear una nueva rama para explorar diferentes direcciones mientras conservas el contexto original.

**Para crear una rama:**

1. Pasa el cursor sobre cualquier mensaje de la conversación.
2. Haz clic en el icono de ramificación en las acciones del mensaje.
3. Elige tu modo de ramificación.
4. Continúa la conversación en una nueva dirección.

### Modos de Ramificación

<Tabs>
  <Tab title="Modo de Continuación">
    El **Modo de Continuación** amplía tu discusión actual manteniendo el valioso contexto de los mensajes anteriores.

    **Cuándo usarlo:**

    - Explorar soluciones alternativas al mismo problema.
    - Probar diferentes enfoques manteniendo la información de fondo.
    - Comparar múltiples respuestas a la misma pregunta.

    **Ejemplo de flujo de trabajo:**

    ```
    Original: "¿Cómo puedo optimizar mi aplicación React?"
    ├─ Rama 1: "Centrarse en estrategias de división de código (code splitting)"
    ├─ Rama 2: "Centrarse en la optimización de imágenes"
    └─ Rama 3: "Centrarse en la reducción del tamaño del bundle"
    ```
  </Tab>

  <Tab title="Modo Independiente">
    El **Modo Independiente** inicia un nuevo tema desde cero basado en cualquier mensaje anterior.

    **Cuándo usarlo:**

    - Profundizar en un subtema específico.
    - Iniciar una discusión relacionada pero independiente.
    - Aislar un aspecto particular para una exploración enfocada.

    **Ejemplo de flujo de trabajo:**

    ```
    Conversación principal: "Diseño de bases de datos para comercio electrónico"
    └─ Rama independiente: "Háblame específicamente sobre la seguridad en el procesamiento de pagos"
    ```
  </Tab>
</Tabs>

### Navegando por las Ramas

Cuando un mensaje tiene múltiples ramas, verás los controles de navegación de ramas:

- **Botones de flecha**: Navega entre diferentes ramas.
- **Contador de ramas**: Muestra la rama actual (por ejemplo, "2/5").
- **Indicador de rama**: Señal visual que muestra los puntos de ramificación.

Haz clic en las flechas izquierda/derecha para cambiar entre diferentes rutas de conversación y comparar respuestas.

## Casos de Uso

<Tabs>
  <Tab title="Desarrollo de Software">

    - Crea ramas para diferentes enfoques de implementación.

    - Prueba casos extremos sin perder el flujo principal de tu conversación.

    ```
    "¿Cómo debo estructurar mi API?"
    ```

## Consejos para una Ramificación Efectiva

<AccordionGroup>
  <Accordion title="Mantén las ramas organizadas">
    Usa mensajes iniciales descriptivos en cada rama para recordar qué explora cada ruta. Esto facilita volver a navegar por ellas más tarde.
  </Accordion>

  <Accordion title="Compara resultados lado a lado">
    Usa la navegación de ramas para cambiar rápidamente entre respuestas e identificar el mejor enfoque para tus necesidades.
  </Accordion>

  <Accordion title="No ramifiques en exceso">
    Aunque la ramificación es potente, crear demasiadas ramas puede resultar abrumador. Concéntrate en 2 a 4 alternativas clave.
  </Accordion>

  <Accordion title="Usa la continuación para iteraciones">
    Al refinar respuestas, el modo de continuación ayuda a mantener el contexto mientras exploras variaciones.
  </Accordion>

  <Accordion title="Usa el modo independiente para profundizar">
    Cuando quieras explorar un tema secundario sin desordenar tu conversación principal, el modo independiente mantiene todo limpio.
  </Accordion>
</AccordionGroup>

## Gestión de Chats

### Organizar conversaciones

- **Buscar**: Encuentra mensajes o temas específicos en todas las conversaciones.
- **Anclar chats importantes**: Mantén las conversaciones de acceso frecuente en la parte superior.
- **Archivar**: Limpia tu espacio de trabajo sin eliminar conversaciones valiosas.
- **Exportar**: Guarda conversaciones en múltiples formatos para compartirlas o hacer copias de seguridad.

### Acciones del mensaje

Cada mensaje incluye acciones rápidas:

- **Copiar**: Copia el contenido del mensaje al portapapeles.
- **Editar**: Modifica tus mensajes enviados.
- **Eliminar**: Elimina mensajes de la conversación.
- **Regenerar**: Obtén una nueva respuesta de la IA.
- **Ramificar**: Crea una nueva ruta de conversación.
- **Compartir**: Exporta mensajes individuales o hilos.

Las conversaciones ramificadas se conservan en tu historial de conversaciones, por lo que siempre puedes volver a explorar diferentes rutas más tarde.
