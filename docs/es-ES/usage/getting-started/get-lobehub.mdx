# Obtener mAI

mAI funciona dondequiera que trabajes: navegador, escritorio o teléfono. Inicia sesión una vez y tus Agentes, Páginas y Memoria se mantendrán sincronizados en todos los dispositivos.

## Resumen de la plataforma

| Plataforma | Disponibilidad | Cómo obtenerlo |
| :--- | :--- | :--- |
| **Web** | Cualquier navegador moderno | [m-ai-officiel.vercel.app](https://m-ai-officiel.vercel.app) |
| **macOS** | Aplicación nativa | [Descargar](https://mprojects.odoo.com/download) |
| **Windows** | Aplicación nativa | [Descargar](https://mprojects.odoo.com/download) |
| **iOS** | App Store | [App Store](https://apps.apple.com/app/id6471212236) |
| **Android** | Google Play / APK | [Google Play](https://play.google.com/store/apps/details?id=com.lobehub.app) · [APK](https://mprojects.odoo.com/download) |

## Web

No requiere instalación. Abre [m-ai-officiel.vercel.app](https://m-ai-officiel.vercel.app) en cualquier navegador moderno y comienza de inmediato. Los datos se sincronizan con la nube: cambia de dispositivo en cualquier momento.

**Ideal para:** Acceso rápido, probar mAI antes de instalarlo, o usarlo en computadoras compartidas o restringidas.

## Escritorio

Descarga la aplicación nativa desde la [Página de Descargas](https://mprojects.odoo.com/download). Disponible para **macOS** y **Windows** (aún no disponible en la Mac App Store o Microsoft Store).

**Ideal para:** Acceso sin conexión, integración a nivel de sistema y una ventana dedicada para un trabajo concentrado.
