# Generación de Imágenes

Describe lo que quieres: mAI convierte el texto en imágenes. Prototipos de productos, inspiración de diseño, ilustraciones o exploración creativa: elige un modelo, define tus parámetros y obtén imágenes de alta calidad en segundos. Todas las imágenes generadas se guardan automáticamente en tu Biblioteca de Recursos.

## Primeros Pasos

Haz clic en **Dibujo** en la interfaz principal de mAI para abrir la página de generación de imágenes.

## Introducir un Prompt

Describe la imagen que deseas en la caja de entrada. Cuanto más específica sea tu descripción, más preciso será el resultado.

**Estructura de un prompt efectivo:**

```
[Sujeto] [Estilo/Medio] [Entorno/Fondo] [Iluminación] [Estado de ánimo] [Detalles técnicos]
```

Ejemplos:

```
"El horizonte de una ciudad futurista al atardecer, arte digital, estilo cyberpunk, luces de neón reflejándose en calles mojadas, iluminación cinematográfica, detalles en 4K"

"El interior de una acogedora cafetería, ilustración en acuarela, luz dorada y cálida entrando por las ventanas, plantas en macetas en los alféizares, atmósfera suave y acogedora"

"Una foto de producto de una cartera de cuero minimalista sobre un fondo blanco limpio, iluminación de estudio, enfoque nítido, estilo de fotografía comercial"
```

**Consejos para los prompts:**

- **Sé específico sobre el estilo** — "óleo", "acuarela", "arte digital", "fotorrealista", "anime", "ilustración vectorial".
- **Describe la iluminación** — "sombras dramáticas", "luz suave difusa", "hora dorada", "iluminación de estudio".
- **Especifica la composición** — "vista de retrato", "gran angular", "primer plano", "vista de pájaro".
- **Añade modificadores de calidad** — "alto detalle", "4K", "enfoque nítido", "calidad profesional".
- **Evita la vaguedad** — Palabras como "hermoso", "bonito", "bueno" aportan poco; describe lo que realmente deseas ver.

## Elegir un Modelo de IA

mAI ofrece múltiples modelos de generación de imágenes por IA. Diferentes modelos tienen diferentes fortalezas:

![Elegir un Modelo](/blog/assetsdd913561927c64d32bd390cee6846f9a.webp)

| Modelo | Ideal para |
| :--- | :--- |
| **DALL-E 3** | Fotos realistas, ilustraciones, seguimiento preciso del prompt |
| **Flux** | Estilos artísticos, imágenes creativas, generación rápida |
| **Stable Diffusion** | Altamente personalizable, estilos comunitarios y modelos afinados (fine-tuned) |
| **Modelos fal.ai** | Varios estilos especializados y generación rápida |

Prueba diferentes modelos con el mismo prompt para ver cuál ofrece los mejores resultados para tu caso de uso.

## Seleccionar Imágenes de Referencia (Opcional)

Si tienes imágenes de referencia, súbelas para guiar el proceso de generación. Haz clic en el botón de subida o arrastra y suelta tus imágenes de referencia directamente. Puedes subir múltiples imágenes de referencia.

![Subir Imágenes de Referencia para Generación de Imágenes](/blog/assets3c160860feef0bd7c653eeb46f683445.webp)

Las imágenes de referencia ayudan al modelo a comprender el estilo, la composición o la paleta de colores que deseas.

## Elegir la Relación de Aspecto de la Imagen

Selecciona una relación de aspecto según el uso que le vayas a dar:

- **1:1** — Publicaciones en redes sociales, fotos de perfil.
- **16:9** — Pantalla panorámica, presentaciones, banners.
- **9:16** — Pantallas móviles, stories, reels.
- **4:3** — Uso general, formatos de pantalla antiguos.
- **3:2** — Estándar de fotografía, impresiones.

## Definir el Número de Imágenes

Elige cuántas imágenes deseas generar de una sola vez. Generar múltiples imágenes a la vez te ofrece variaciones para elegir. Comienza con 2–4 para encontrar el mejor resultado.

## Ver y Descargar Imágenes

Una vez generadas, las imágenes aparecen en la página de dibujo. Puedes:

- Previsualizar cualquier imagen a tamaño completo haciendo clic en ella.
- Seleccionar tus favoritas y descargarlas.
- Compartir directamente desde el visor de imágenes.

Todas las imágenes generadas se guardan automáticamente en tu Biblioteca de Recursos.

![Imágenes Generadas en la Biblioteca de Activos](/blog/assets974acc551878f2f395518a3fbb9bd924.webp)

## Consejos para Obtener Mejores Resultados

**Itera en tus prompts** — Si el primer resultado no es el correcto, ajusta un elemento a la vez en lugar de volver a escribir todo el prompt. Añade más detalles, cambia el descriptor de estilo o especifica lo que no quieres.

**Usa una imagen de referencia** — Subir una imagen de referencia junto con tu prompt ayuda al modelo a igualar el estilo, la paleta de colores o la composición deseada.

**Prueba múltiples variaciones** — Genera más de 4 imágenes a la vez y elige la mejor. La generación de imágenes por IA tiene una aleatoriedad inherente: algunas variaciones serán significativamente mejores que otras.

**Adapta el modelo a la tarea** — Usa modelos fotorrealistas (DALL-E 3, Flux) para fotos de productos y escenas realistas; utiliza modelos enfocados en estilo para ilustraciones artísticas.
