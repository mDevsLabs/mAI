# Agentes integrados 📦

**mAI** viene preconfigurado con varios agentes del sistema de forma predeterminada para ayudarle de inmediato en las tareas diarias sin necesidad de configuración manual.

## Lista de agentes del sistema

Aquí están los principales agentes integrados en la aplicación:

1. **May**: Su asistente principal versátil, lista para todas las tareas de escritura, lluvia de ideas y ayuda general.
2. **Desarrollador de software**: Un agente con un prompt del sistema centrado en la programación, la escritura de código limpio y la depuración.
3. **Traductor universal**: Especializado en la traducción fluida entre docenas de idiomas, respetando las expresiones idiomáticas.
4. **Content Writer**: Optimizado para marketing, redacción de publicaciones en blogs y redacción de textos claros para redes sociales.

## Uso de agentes integrados

Estos agentes son visibles inmediatamente en su lista de agentes después de la instalación. Puede iniciar una nueva conversación con cualquiera de ellos con un solo clic. Aunque están bloqueados para ciertas modificaciones del sistema, puede personalizar sus parámetros locales (como el modelo de lenguaje asociado) para cada chat.
