# Menú de Comandos 🔍

El menú de comandos rápidos en **mAI** es el centro de control principal para navegar y controlar la aplicación sin necesidad de tocar el ratón. Se activa instantáneamente usando el atajo `Ctrl + P` (o `Cmd + P` en macOS).

## Características Principales

Una vez abierto el menú, puede escribir comandos para:
- **Cambiar de Agente**: Escriba el nombre de un agente (como "May" o "Developer") para cambiar al instante.
- **Buscar o Crear Chats**: Busque en sus conversaciones existentes mediante palabras clave o escriba `/new` para iniciar una nueva sesión de chat.
- **Acceder a la Configuración**: Escriba `configuración` para ir directamente a las páginas de configuración del sistema.
- **Cambiar de Modelo**: Seleccione un LLM diferente para la conversación activa (por ejemplo, al escribir "gpt" o "claude" se filtrarán los modelos coincidentes).

## Controles del Teclado y Consejos

- Utilice las teclas de flecha `Arriba` y `Abajo` en su teclado para navegar por la lista.
- Presione `Enter` para confirmar una selección.
- Presione `Esc` para cerrar el menú de comandos en cualquier momento.
