# 屏幕截图 📸

**mAI** 应用包含一个直观的屏幕截图工具，允许用户快速抓取屏幕的特定部分，并将其直接发送到聊天中，供智能体进行视觉分析。

## 使用截图工具

进行屏幕截图：
1. 点击位于聊天输入栏中的 **相机 / 截图** 图标，或使用专用的快捷键。
2. 界面会暂时隐藏，以便您选择要捕获的区域或窗口。
3. 选择完成后，图像会自动附加到您的聊天输入框中。

## 配合 May 进行视觉分析

利用在 mAI 中配置的语言模型的多模态能力，您可以针对捕获的图像提出问题：
- "帮我解释一下这个错误信息。"
- "翻译这张图片中的文本。"
- "审查这个布局并建议设计改进。"
- "编写与此 UI 原型相对应的代码。"
