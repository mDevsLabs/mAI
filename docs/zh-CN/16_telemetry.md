# 遥测 📊

在 **mAI** 中，透明度和尊重您的隐私是我们的首要任务。本文档详细说明了应用中遥测的工作原理以及可能收集的数据。

## 匿名数据收集

为了帮助我们提高应用的稳定性并识别界面错误，mAI 包含匿名遥测功能。如果您选择加入，以下信息将以完全匿名化的格式进行传输：
- 崩溃报告和未处理的 JavaScript 异常。
- 全局使用统计数据（例如，创建的自定义智能体数量、最常选择的语言模型或插件启用率）。
- 基本系统信息（应用版本、操作系统、屏幕分辨率）。

## 我们绝不收集的信息

mAI 的遥测严格尊重您的隐私。我们 **绝不** 收集、读取或存储：
- 您的消息或对话内容。
- 您的自定义系统提示词或智能体配置。
- 您的 API 密钥或凭据详细信息。
- 任何个人身份信息（姓名、电子邮件地址等）。

## 启用或禁用遥测

您可以随时选择参与改进 mAI 或退出：
1. 前往应用 **设置**。
2. 点击 **关于 / 遥测 (About / Telemetry)** 选项卡。
3. 切换 **允许匿名崩溃报告和使用统计** 选项。
