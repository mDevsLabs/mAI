# 内置智能体 📦

**mAI** 预装了几个预配置的系统智能体，使您无需进行任何手动设置即可立即在常见任务中获得协助。

## 系统智能体列表

以下是应用中主要的内置智能体：

1. **May**：您的多功能核心助手，随时准备协助完成所有写作、头脑风暴和日常求助任务。
2. **软件开发人员 (Software Developer)**：配置有专门针对编程、编写干净代码和调试的系统提示词的智能体。
3. **通用翻译器 (Universal Translator)**：专注于在数十种语言之间进行流畅翻译，并尊重习惯用法。
4. **内容撰写者 (Content Writer)**：针对营销、博客文章草拟和撰写清晰的社交媒体文案进行了优化。

## 使用内置智能体

这些智能体在安装后会立即显示在您的智能体列表中。只需单击一下，即可与其中任何一个开始新对话。虽然它们的部分系统设置是锁定的，但您可以为每次对话自定义它们的本地参数（例如关联的语言模型）。
