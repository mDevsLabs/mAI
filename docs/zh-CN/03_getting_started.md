# 快速开始 ⚡

本指南将帮助您在开发机器上本地安装并运行 **mAI**。

## 前提条件

请确保您已安装以下软件：
- **Node.js**（推荐 v18 或更高版本）
- **pnpm** 或 **bun** 用于包和工作区管理

## 安装步骤

1. **克隆仓库**：
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **安装依赖**：
   使用 pnpm：
   ```bash
   pnpm install
   ```
   使用 bun：
   ```bash
   bun install
   ```

3. **运行开发服务器**：
   ```bash
   pnpm dev
   # 或
   bun run dev
   ```

应用将在您的浏览器中通过 `http://localhost:3010` 进行访问。在您第一次启动时，您的核心助手 **May** 将欢迎您并协助您配置初始偏好！
