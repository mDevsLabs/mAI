# 隐私与安全 🔒

数据安全和尊重您的隐私是 **mAI** 设计中的根本原则。

## 加密本地存储

您的所有敏感 data 都存储在您的设备本地：
- **API 密钥**：在保存到您的本地存储之前，它们会使用行业标准算法进行加密。您的 API 密钥绝不会发送 to 我们的服务器；它们会直接从您的设备分发到相应的 LLM 提供商服务器。
- **聊天历史**：您的对话将保留在您的机器上（除非您显式启用了安全的云端同步）。

## 安全连接

发往语言模型 API 的所有传出请求都使用安全通信协议 (HTTPS / SSL)，以确保交换的数据不会在网络上被拦截。

## 推荐的企业实践

为了在企业环境中获得最高级别的安全：
- 使用通过 **Ollama** 或私有推理服务器本地托管的模型（这意味着没有任何数据离开公司网络）。
- 在通用设置选项卡中选择退出匿名遥测。
