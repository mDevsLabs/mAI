# 智能体管理 👥

在 **mAI** 中，您可以创建并配置专门的智能体 (Agents) 来解决特定需求和使用场景。智能体管理可以直接通过图形用户界面或配置文件来完成。

## 创建新智能体

要在 mAI 中创建一个智能体：
1. 前往左侧边栏，点击 **智能体 (Agents)** 按钮。
2. 点击列表顶部的 **创建智能体**。
3. 填写基本信息：
   - **名称**：您智能体的标识符。
   - **头像**：选择一个图标、表情符号或上传一张图片。
   - **描述**：对该智能体角色的快速概述。

## 编辑现有智能体

您可以通过点击智能体列表中关联的设置图标来编辑任何自定义智能体。您将能够访问高级配置，例如定义系统提示词 (System Prompt)、选择默认语言模型以及管理其插件。

## 删除智能体

如果不再需要某个自定义智能体，您可以通过智能体设置选项将其永久删除。与该智能体关联的现有对话不会被删除，但该智能体本身将无法再用于开启新对话。
