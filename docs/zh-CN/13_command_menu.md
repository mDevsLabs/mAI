# 命令菜单 🔍

**mAI** 中的快速命令菜单是您在不接触鼠标的情况下导航和控制应用的中央控制枢纽。它可以使用快捷键 `Ctrl + P`（在 macOS 上为 `Cmd + P`）瞬间激活。

## 核心功能

菜单打开后，您可以输入命令以执行以下操作：
- **切换智能体**：输入智能体的名称（如 "May" 或 "Developer"）以实现瞬间切换。
- **搜索或创建聊天**：通过关键词搜索您现有的对话，或输入 `/new` 开启一个新聊天会话。
- **访问设置**：输入 `settings` 直接进入系统设置页面。
- **更改模型**：为当前对话选择不同的 LLM（例如，输入 "gpt" 或 "claude" 将筛选匹配的模型）。

## 键盘控制与建议

- 使用键盘上的 `上` 和 `下` 方向键在列表中导航。
- 按 `Enter` 键确认选择。
- 随时按 `Esc` 键关闭命令菜单。
