# 本地文件系统 📂

**mAI** 桌面端应用与您操作系统的文件系统进行了深度集成，允许您安全地访问和分析您的本地文件。

## 读取与分析文件

您可以直接将文件发送给您的智能体（如 **May**）进行分析或处理：
- **拖放**：将文本文件、PDF 文件、Excel 表格或图像直接拖入 mAI 聊天区域。
- **附件**：点击聊天框中的回形针图标来浏览您的文件资源管理器。
- 文件会在本地进行解析和处理，只有提取出的文本（或多模态模型的图像本身）会发送给 LLM。

## 安全边界

为了确保您的系统安全：
- mAI 绝不会在没有您显式操作（拖放或手动选择文件）的情况下读取文件。
- 您可以在通用设置中指定一个专用目录，以限制 mAI 仅能访问该文件夹。
