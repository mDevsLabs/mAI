# 聊天与渲染功能 💬

**mAI** 对话界面旨在提供流畅、互动的体验，并配有丰富的视觉功能 and 精准的控制选项。

## 核心聊天功能

- **行内编辑**：您可以在任何时候编辑之前发送的任何消息。mAI 会自动从该轮对话重新运行。
- **富 Markdown 渲染**：mAI 支持完整的 Markdown 格式：表格、LaTeX 数学公式、带语法高亮的代码块以及交互式预览窗格。
- **快捷操作**：每个消息气泡都配有一个上下文菜单，允许您复制原始文本、重新生成 AI 响应、编辑或从历史记录中删除该消息。
- **历史记录追踪**：聊天会话会本地归档在 SQLite 数据库中，让您可以瞬间搜索并找回过去的对话。
