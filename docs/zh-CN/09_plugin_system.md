# 插件系统 🔌

**mAI** 拥有强大的插件生态系统，允许对话智能体通过与外部工具和服务交互，来克服其训练数据的局限性。

## 内置插件

以下是默认可用的主要插件：

1. **网页浏览 (Web Browsing)**：允许智能体实时搜索互联网以获取最新信息。
2. **沙盒组件 (Artifacts)**：允许在专用的侧边栏中以交互方式生成、编辑和预览代码、文档或用户界面设计。
3. **天气 (Weather)**：查询天气 API 以获取本地天气预报。
4. **代码计算器 (Code Calculator)**：运行本地脚本以精确解决复杂的计算。

## 启用与配置

每个智能体都可以拥有自己启用的插件集。您可以通过智能体设置中的 **插件** 选项卡为特定智能体启用或禁用插件。某些插件（如商业网页搜索）可能需要在全局配置中进行额外设置（例如 API 密钥）。
