# 常见问题 (FAQ) ❓

以下是关于配置和使用 **mAI** 最常见问题的解答。

## 通用使用

### 我可以在没有互联网连接的情况下使用 mAI 吗？
是的，可以，前提是您使用通过 **Ollama** 或 LocalAI 等服务运行的本地模型。如果您使用云端模型（如 OpenAI 或 Anthropic），则需要处于活动状态的互联网连接以与其服务器通信。

### 如何更新我的 API 密钥？
前往设置 (`Ctrl + ,`)，然后打开 **语言模型 (Language Models)** 选项卡。在此处，您可以为每个支持的提供商添加、更改或删除 API 密钥。

## 智能体自定义

### May 是谁？
**May** 是 mAI 中的默认智能核心助手。她非常多才多艺，配置为从您第一次启动时就为您提供协助。

### 我的智能体会与其他用户共享吗？
不会。默认情况下，您创建的所有自定义智能体都会存储在本地，并且仅在您的设备上保持私密。但是，您可以手动导出它们以便与他人分享。

## 费用与计费

### 使用 mAI 是免费的吗？
mAI 应用本身完全免费且开源。然而，使用商业云端模型（如 OpenAI 或 Anthropic）需要您提供自己的 API 密钥。您只需直接向模型提供商支付您的实际 API 使用费。
