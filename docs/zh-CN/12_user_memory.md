# 用户记忆 🧠

**mAI** 中的用户记忆功能允许您的智能体，特别是 **May**，跨越不同的对话记住关于您的关键事实。这使得用户体验更加个性化且无缝衔接。

## 记忆如何工作

当您与 mAI 交互时，应用会自动提取相关的上下文信息并将其存储在本地记忆数据库中。这可以包括：
- 您的语言偏好或写作风格。
- 您当前正在开展的项目。
- 您的开发偏好（偏好的框架、代码风格）。
- 您选择分享的个人细节（如您的名字或专业角色）。

## 管理您的记忆

您对 mAI 存储的信息拥有完全的控制权：
1. 前往应用全局设置。
2. 打开 **记忆 (Memory)** 选项卡。
3. 您将看到以卡片形式存储的事实列表。
4. 您可以手动编辑任何条目，或者从记忆数据库中永久删除某个事实，以便智能体遗忘它。
