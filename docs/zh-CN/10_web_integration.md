# 网页集成 🌐

**mAI** 被设计为一个跨平台的应用。除了桌面端应用外，mAI 还可以通过优化的网页版完全访问，从而允许从 any 网页浏览器进行流畅访问。

## 网页版功能

mAI 网页版提供与桌面端应用几乎完全相同的功能对等：
- 流畅、自适应且响应迅速的用户界面。
- 使用您自己的 API 密钥访问所有语言模型（保存在浏览器本地或进行同步）。
- 使用大多数插件（需要本地系统特权的插件，如直接访问本地文件系统，可能会受到限制）。

## 云端同步

如果您在多个设备上使用 mAI，可以启用安全的云端同步。您的聊天数据、自定义智能体和偏好设置可以使用端到端加密保存到您自己的托管服务器或通过兼容的云存储解决方案（如 WebDAV 或同步数据库服务）中。
