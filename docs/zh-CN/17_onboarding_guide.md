# 新手引导指南 🚀

欢迎使用 **mAI**！本指南旨在带新手用户逐步完成他们在应用中的首要步骤。

## 步骤 1：May 的欢迎消息

首次启动应用时，**May**（您的默认虚拟助手）将自动打开。她将引导您完成第一步的配置，并解答您对应用的任何即时疑问。

## 步骤 2：配置 API 密钥

为了使 mAI 正常运行，您需要配置至少一个语言模型提供商：
1. 点击 May 提示的配置按钮或打开设置 (`Ctrl + ,`)。
2. 选择 **语言模型 (Language Models)** 选项卡。
3. 输入您选择的提供商（OpenAI、Anthropic Claude、Google Gemini）的 API 密钥，或启用如 Ollama 的本地提供商。
4. 保存以在您的设备上本地安全存储您的密钥。

## 步骤 3：您的第一次对话

1. 点击左侧边栏中的 **新聊天 (New Chat)**。
2. 在屏幕底部的文本区域中输入您的消息。
3. 按 **Enter** 键开启您与 May 的第一次对话！

## 步骤 4：探索与创建智能体

前往 **智能体 (Agents)** 选项卡探索预配置的智能体档案，或点击 **创建智能体 (Create Agent)** 来设计契合您特定需求的虚拟伴侣。
