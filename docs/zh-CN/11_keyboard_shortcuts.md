# 键盘快捷键 ⌨️

为了最大化您的工作效率并无需离开键盘即可快速导航，**mAI** 包含了一系列可配置的快捷键。

## 全局导航快捷键

- `Ctrl + P`（在 Mac 上为 `Cmd + P`）：打开快速命令菜单。
- `Ctrl + ,`（在 Mac 上为 `Cmd + ,`）：打开应用设置。
- `Ctrl + N`（在 Mac 上为 `Cmd + N`）：开始新对话。
- `Ctrl + \`（在 Mac 上为 `Cmd + \`）：显示/隐藏左侧边栏。

## 聊天快捷键

- `Enter`：发送消息。
- `Shift + Enter`：在输入框中插入换行符。
- `Ctrl + E`：编辑您最后发送的消息。
- `Ctrl + R`：重新生成智能体最后一次回复。
- `Ctrl + Delete`：删除当前对话。

## 界面管理快捷键

- `Alt + 1`：切换到聊天 (Chats) 选项卡。
- `Alt + 2`：切换到智能体 (Agents) 选项卡。
- `Alt + 3`：切换到发现 (Discover / Marketplace) 选项卡。
