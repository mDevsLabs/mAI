# mAI 介绍 🚀

欢迎阅读 **mAI** 官方文档。mAI 是下一代人工智能应用，旨在最大化您的工作效率并简化您与大语言模型 (LLM) 的交互。

## 什么是 mAI？

**mAI** 是一个现代、快速且完全可自定义的对话框架与智能助手平台。它允许用户在优雅流畅的用户界面中，集中访问多个 AI 提供商（例如 OpenAI GPT、Claude、Gemini 或通过 Ollama 运行的本地模型）。

## May：您的核心助手 🤖

mAI 的核心是 **May**，您的默认智能助手。May 旨在具备高度的多功能性，从您第一次启动应用开始便为您提供引导、解答复杂问题、协助编程或整理您的思路。May 能够通过持久记忆和使用外部工具（插件）的能力来适应您的需求。

## 为什么选择 mAI？

- **多模型支持**：在不同的 AI 提供商之间瞬间切换。
- **隐私第一**：您的 API 密钥和聊天记录都安全地存储在本地或您自己的基础设施中。
- **功能丰富**：集成了屏幕截图、视觉化聊天分享、插件支持等众多功能。
