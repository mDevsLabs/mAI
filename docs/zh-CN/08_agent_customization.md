# 智能体自定义 ⚙️

在 **mAI** 中微调智能体允许您深度修改它们的行为、回复风格以及它们可以访问的工具。

## 基本设置

对于每个智能体，您可以配置：
- **系统提示词 (System Prompt)**：定义智能体个性、规则和角色的基础指令。这是对智能体回复影响最大的设置。
- **语言模型**：选择默认使用的 LLM（例如 GPT-4o、Claude 3.5 Sonnet、Llama 3）。

## 高级设置

您还可以修改生成超参数：
- **温度 (Temperature)**：控制回复的创造力。较低的温度（例如 0.2）产生更具事实性和确定性的回答。较高的温度（例如 0.9）鼓励创造力。
- **Top P**：控制回复多样性的另一种方法。
- **存在 / 频率惩罚 (Presence / Frequency Penalty)**：鼓励或阻止智能体重复相同的词语或偏离讨论的主题。
- **最大 Token 数 (Max Tokens)**：限制生成的回复的最大长度。
