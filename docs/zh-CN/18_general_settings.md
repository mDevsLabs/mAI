# 通用设置 ⚙️

**mAI** 中的通用设置允许您根据自身的美学、语言和功能偏好来调整应用。

## 用户界面与主题

- **主题**：在浅色模式、深色模式之间选择，或让应用遵循您电脑的系统偏好设置。
- **主题色 (Accent Color)**：自定义图形用户界面中激活按钮和关键交互元素的颜色。
- **界面语言**：选择应用的语言（法语、英语等）。
- **字体大小**：调整文本大小以获得最佳的阅读舒适度。

## 数据管理

- **导出数据**：将您的所有对话、自定义智能体和配置导出到单个 JSON 归档文件中。
- **导入数据**：恢复以前的备份或导入外部智能体配置。
- **重置应用**：永久擦除所有本地数据（历史记录、API 密钥、设置）以将应用恢复到初始状态。
