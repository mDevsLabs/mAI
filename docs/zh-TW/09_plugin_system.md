# 外掛系統 🔌

**mAI** 擁有強大的外掛程式生態系統，允許對話智慧體（Agent）透過與外部工具和服務進行互動，突破其訓練資料的限制。

## 內置外掛程式

以下是預設可用的主要外掛程式：

1. **網頁瀏覽 (Web Browsing)**：允許智慧體即時搜尋網路以獲取最新資訊。
2. **產出物 (Artifacts)**：允許在專用側邊欄中以互動方式生成、編輯和預覽程式碼、文件或 UI 設計。
3. **天氣 (Weather)**：查詢天氣 API 以獲取本地天氣預報。
4. **程式碼計算器 (Code Calculator)**：執行本地腳本以精確解決複雜的計算。

## 啟用與配置

每個智慧體都可以擁有自己啟用的外掛程式集。您可以透過智慧體設定中的 **外掛 (Plugins)** 索引標籤，為特定智慧體啟用或停用外掛程式。某些外掛程式（例如商業網頁搜尋）可能需要在全域配置中進行額外設定，例如 API 金鑰。
