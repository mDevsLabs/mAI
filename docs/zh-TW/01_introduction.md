# mAI 簡介 🚀

歡迎閱讀 **mAI** 的官方文件。mAI 是一款旨在最大化您的生產力並簡化您與大型語言模型（LLM）互動的次世代人工智慧應用程式。

## 什麼是 mAI？

**mAI** 是一個現代、快速且完全可客製化的對話框架與對話智慧體平台。它允許使用者在時尚且流暢的使用者介面中，集中存取多個 AI 提供商（例如 OpenAI GPT、Claude、Gemini 或透過 Ollama 的本地模型）。

## May：您的核心助理 🤖

mAI 的核心是 **May**（前身為 Lobe AI），您的預設智慧助理。May 設計得非常多功能，從首次啟動開始便為您提供協助，引導您、回答複雜問題、協助編碼或整理您的思維。May 透過持久性記憶以及使用外部工具（外掛程式）的能力，來適應您的需求。

## 為什麼選擇 mAI？

- **多模型支援**：在不同的 AI 提供商之間即時切換。
- **隱私優先**：您的 API 金鑰和聊天記錄安全地儲存在本地或您自己的基礎設施中。
- **功能豐富**：整合螢幕截圖、視覺化聊天分享、外掛程式支援等等。
