# 新手引導指南 🚀

歡迎使用 **mAI**！本指南旨在引導新使用者逐步完成在應用程式中的首次使用步驟。

## 步驟 1：May 的歡迎訊息

首次啟動應用程式時，您的預設虛擬助理 **May** 將自動開啟。她將引導您完成第一步的配置，並回答您對應用程式的任何即時問題。

## 步驟 2：配置 API 金鑰

為了讓 mAI 正常運作，您需要配置至少一個語言模型提供商：
1. 按一下 May 提示的配置按鈕，或開啟設定 (`Ctrl + ,`)。
2. 選擇 **語言模型 (Language Models)** 索引標籤。
3. 輸入您選擇的提供商（OpenAI、Anthropic Claude、Google Gemini）的 API 金鑰，或啟用如 Ollama 等本地提供商。
4. 儲存以安全地將金鑰儲存在您的裝置本地。

## 步驟 3：您的首次對話

1. 按一下左側側邊欄中的 **新對話 (New Chat)**。
2. 在畫面底部的文字區域中輸入您的訊息。
3. 按下 **Enter** 鍵，開始您與 May 的第一次對話！

## 步驟 4：探索與建立智慧體

前往 **智慧體 (Agents)** 索引標籤以探索預先配置的智慧體設定檔，或按一下 **建立智慧體 (Create Agent)** 來設計一個專為您的特定需求而定制的虛擬夥伴。
