# 智慧體管理 👥

在 **mAI** 中，您可以創建與配置專用的智慧體（Agent），以滿足特定需求與使用場景。智慧體管理可直接透過圖形使用者介面或透過配置檔案進行。

## 創建新智慧體

若要在 mAI 中創建智慧體：
1. 前往左側側邊欄，然後按一下 **智慧體 (Agents)** 按鈕。
2. 按一下列表頂部的 **創建智慧體 (Create Agent)**。
3. 填寫基本資訊：
   - **名稱**：您智慧體的識別名稱。
   - **頭像**：選擇一個圖示、表情符號，或上傳圖片。
   - **描述**：簡要總結該智慧體的職責。

## 編輯現有智慧體

您可以透過按一下智慧體列表中相關聯的設定圖示，來編輯任何自訂智慧體。您將能夠存取進階配置，例如定義系統提示詞（system prompt）、選擇預設語言模型以及管理其外掛程式。

## 刪除智慧體

如果不再需要某個自訂智慧體，您可以透過該智慧體的設定選項將其永久刪除。與該智慧體相關聯的現有對話不會被刪除，但該智慧體本身將不再可用於新的對話。
