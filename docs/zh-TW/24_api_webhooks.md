# API 與 Webhooks 🔗

**mAI** 提供程式化介面，允許與您自己的應用程式、網頁服務或自動化腳本進行外部整合。

## 本地 API (HTTP)

mAI 桌面端應用程式可以在指定連接埠上運行安全的本地網頁伺服器。此伺服器公開了多個端點（endpoints）：
- `POST /api/chat`：向特定智慧體傳送訊息並返回結構化回應（支援標準或串流/streaming 格式）。
- `GET /api/agents`：列出所有可用的智慧體及其配置。
- `POST /api/agents`：允許以程式化方式建立新智慧體。

## Webhooks 整合

您可以在 mAI 的進階設定中配置傳出 Webhooks。每當特定事件發生時，Webhook 就會自動向外部 URL 傳送 JSON 承載資料（JSON payload）：
- **聊天訊息已完成**：一旦智慧體完成回應生成即可觸發。
- **智慧體已建立**：新增智慧體設定檔時觸發。
- **系統錯誤**：在 API 金鑰驗證問題或大語言模型提供商停機期間觸發。
