# 常見問題 ❓

以下是關於配置與使用 **mAI** 最常見問題的解答。

## 一般使用

### 我可以在沒有網際網路連線的情況下使用 mAI 嗎？
可以，前提是您使用的是透過 **Ollama** 或 LocalAI 等服務運行的本地模型。如果您使用的是雲端模型（例如 OpenAI 或 Anthropic），則需要有作用中的網際網路連線才能與其伺服器進行通訊。

### 如何更新我的 API 金鑰？
前往設定 (`Ctrl + ,`)，然後開啟 **語言模型 (Language Models)** 索引標籤。在此處，您可以為每個支援的提供商新增、變更或移除 API 金鑰。

## 智慧體客製化

### May 是誰？
**May** 是 mAI 中的預設智慧核心助理。她非常多功能，並且配置為從您的首次啟動開始便為您提供協助。

### 我的智慧體會與其他使用者分享嗎？
不會。預設情況下，您建立的所有自訂智慧體都儲存在本地，並僅保留在您的裝置上。不過，您可以手動匯出它們以與他人分享。

## 費用與帳單

### mAI 是免費使用的嗎？
mAI 應用程式本身完全免費且開源。然而，使用商業雲端模型（如 OpenAI 或 Anthropic）需要您提供自己的 API 金鑰。您只需直接向模型提供商支付您實際使用的 API 費用。
