# 隱私與安全 🔒

資料安全與尊重您的隱私是 **mAI** 設計時的根本原則。

## 加密本地儲存

您所有敏感的資料都儲存在您的裝置本地：
- **API 金鑰**：在儲存到您的本地儲存空間之前，它們會使用業界標準演算法進行加密。您的 API 金鑰絕不會傳送到我們的伺服器；它們會直接從您的裝置發送到各自的大語言模型（LLM）提供商伺服器。
- **對話歷史記錄**：您的對話會保留在您的本機上（除非您明確啟用安全的雲端同步）。

## 安全連線

所有發送到語言模型 API 的連線請求均使用安全傳輸協定（HTTPS / SSL），確保交換的資料不會在網路上被攔截。

## 企業推薦實作方式

為了在企業環境中實現最高層級的安全：
- 使用透過 **Ollama** 或私人推論伺服器在本地託管的模型（這意味著沒有任何資料會離開公司網路）。
- 在一般設定索引標籤中選擇退出匿名遙測。
