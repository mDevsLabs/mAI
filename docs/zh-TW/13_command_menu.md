# 指令選單 🔍

**mAI** 中的快速指令選單是無需使用滑鼠即可導覽與控制應用程式的中央控制中心。可透過快捷鍵 `Ctrl + P`（在 macOS 上為 `Cmd + P`）立即啟用。

## 核心功能

開啟選單後，您可以輸入指令以進行以下操作：
- **切換智慧體**：輸入智慧體的名稱（例如「May」或「Developer」）以即時切換。
- **搜尋或創建聊天**：透過關鍵字搜尋您現有的對話，或輸入 `/new` 來開始新的聊天工作階段。
- **存取設定**：輸入 `settings` 以直接前往系統設定頁面。
- **變更模型**：為目前對話選擇不同的大語言模型（例如，輸入「gpt」或「claude」將篩選出相符的模型）。

## 鍵盤控制與提示

- 使用鍵盤上的 `Up` 和 `Down` 箭頭鍵來導覽列表。
- 按 `Enter` 鍵確認選擇。
- 隨時按 `Esc` 鍵關閉指令選單。
