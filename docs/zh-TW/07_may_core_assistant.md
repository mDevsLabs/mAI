# May：核心助理 🤖

**May** 是 **mAI** 體驗核心的預設助理。她不僅僅是一個聊天智慧體，更扮演著引導者以及連結您與系統強大功能之間橋樑的角色。

## May 的角色與能力

May 被設計為您的日常助手。以下是她的主要角色：
- **歡迎與新手引導**：她能協助您配置首批 API 金鑰並探索使用者介面。
- **技術與通用支援**：她可以回答關於應用程式本身的問題，或一般文化、科學、藝術和技術等任何主題。
- **開發協助**：她理解並能使用多種程式語言編寫程式碼。
- **外掛程式整合**：預設情況下，May 可以存取已啟用的外掛程式（例如網頁瀏覽或產出物系統/artifacts）來豐富其回答。

## 客製化 May

雖然 May 是基礎助理，但您可以在其設定中修改其行為：
- 調整其基礎系統提示詞。
- 變更其預設語言模型。
- 為其聊天工作階段啟用或停用特定外掛程式。
