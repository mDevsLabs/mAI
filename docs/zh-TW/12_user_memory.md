# 使用者記憶 🧠

**mAI** 中的使用者記憶功能允許您的智慧體（特別是 **May**）在不同的對話中記住關於您的關鍵事實。這使使用者體驗更加客製化與流暢。

## 記憶如何運作

當您與 mAI 互動時，應用程式會自動提取相關的上下文資訊，並將其儲存在本地記憶資料庫中。這可能包括：
- 您的語言偏好或寫作風格。
- 您目前正在進行的專案。
- 您的開發偏好（偏好的框架、程式碼風格）。
- 您選擇分享的個人詳細資訊（例如您的姓名或專業角色）。

## 管理您的記憶

您對 mAI 儲存的資訊擁有完全的控制權：
1. 前往全域應用程式設定。
2. 開啟 **記憶 (Memory)** 索引標籤。
3. 您將看到以卡片形式儲存的事實清單。
4. 您可以手動編輯任何條目，或從記憶資料庫中永久刪除某個事實，以便智慧體忘記它。
