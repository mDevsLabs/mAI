# 本地檔案系統 📂

**mAI** 桌面端應用程式與您作業系統的檔案系統具有深度整合，允許您安全地存取與分析您的本地檔案。

## 讀取與分析檔案

您可以將檔案直接傳送給您的智慧體（例如 **May**）進行分析或處理：
- **拖放**：將文字文件、PDF 檔案、Excel 試算表或圖片直接拖放到 mAI 聊天區域中。
- **附件**：按一下聊天中的迴紋針圖示以瀏覽您的檔案總管。
- 檔案將在本地進行解析和處理，且僅將提取的文字（或多模態模型所需的圖片本身）傳送給大語言模型（LLM）。

## 安全界限

為了保持您系統的安全：
- 在沒有您的明確操作（拖放或手動選取檔案）的情況下，mAI 絕不會讀取檔案。
- 您可以在一般設定中指定專用目錄，以將 mAI 的存取權限限制在該資料夾內。
