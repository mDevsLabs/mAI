# 開始使用 ⚡

本指南將協助您在開發本機上設定並運行 **mAI**。

## 先決條件

請確保您已安裝以下內容：
- **Node.js**（建議 v18 或更高版本）
- **pnpm** 或 **bun** 用於套件與工作空間管理

## 安裝步驟

1. **複製存放庫**：
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **安裝依賴項**：
   使用 pnpm：
   ```bash
   pnpm install
   ```
   使用 bun：
   ```bash
   bun install
   ```

3. **運行開發伺服器**：
   ```bash
   pnpm dev
   # 或
   bun run dev
   ```

您可以在瀏覽器中透過 `http://localhost:3010` 存取該應用程式。首次啟動時，您的核心助理 **May** 將會迎接您，引導您進行初始偏好設定！
