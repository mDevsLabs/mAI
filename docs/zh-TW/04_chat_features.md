# 聊天與渲染功能 💬

**mAI** 對話介面旨在提供流暢、互動的體驗，並配備了豐富的視覺功能與精確的控制選項。

## 核心聊天功能

- **行內編輯**：您可以在任何時候編輯您之前的任何訊息。mAI 將自動從該輪對話開始重新運行。
- **富 Markdown 渲染**：mAI 支援完整的 Markdown 格式：表格、LaTeX 數學公式、語法高亮的程式碼區塊以及互動式預覽面板。
- **快速操作**：每個訊息泡泡都配有操作選單，允許您複製原始內容、重新生成 AI 回應、編輯或從歷史記錄中刪除該訊息。
- **歷史記錄追蹤**：聊天工作階段會儲存在本地的 SQLite 資料庫中，使您能夠即時搜尋並恢復過去的對話。
