# 內置智慧體 📦

**mAI** 隨附了多個預先配置的系統智慧體（Agent），以便在不需要任何手動設定的情況下，立即協助您處理常見任務。

## 系統智慧體清單

以下是應用程式中的主要內置智慧體：

1. **May**：您的多功能核心助理，隨時準備處理所有寫作、腦力激盪和一般協助任務。
2. **軟體開發人員 (Software Developer)**：配置了專為編程、撰寫乾淨程式碼和偵錯而定制的系統提示詞的智慧體。
3. **通用翻譯官 (Universal Translator)**：專精於數十種語言之間的流暢翻譯，並尊重習慣用語表達。
4. **內容作家 (Content Writer)**：針對行銷、部落格文章起草和撰寫清晰的社群媒體文案進行了優化。

## 使用內置智慧體

安裝後，這些智慧體會立即顯示在您的智慧體列表中。只需按一下即可與其中任何一個開始新對話。雖然它們被鎖定以防止某些系統修改，但您可以為每次對話自訂它們的本地參數（例如關聯的語言模型）。
