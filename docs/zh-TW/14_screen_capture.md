# 螢幕擷取 📸

**mAI** 應用程式包含一個直覺的螢幕擷取工具，允許使用者快速擷取螢幕的一部分，並將其直接傳送到對話中，供智慧體進行視覺分析。

## 使用擷取工具

若要進行螢幕擷取：
1. 按一下位於聊天輸入列中的 **相機 / 擷取 (Camera / Capture)** 圖示，或使用專用快捷鍵。
2. 使用者介面將暫時隱藏，以便您選擇要擷取的區域或視窗。
3. 選擇完成後，圖片將自動附加到您的聊天輸入框中。

## 與 May 進行視覺分析

利用 mAI 中配置的語言模型的多模態能力，您可以針對擷取的圖片提出問題：
- 「向我解釋此錯誤訊息。」
- 「翻譯此圖片中的文字。」
- 「審查此版面配置並提出設計改進建議。」
- 「編寫此 UI 實體模型對應的程式碼。」
