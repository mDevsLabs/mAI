# 一般設定 ⚙️

**mAI** 中的一般設定允許您根據美學、語言與功能偏好來調整應用程式。

## 使用者介面與主題

- **主題**：選擇淺色模式、深色模式，或讓應用程式遵循您電腦的系統偏好設定。
- **主題色 (Accent Color)**：自訂圖形 UI 中作用中按鈕和主要互動元素的顏色。
- **介面語言**：選擇應用程式語言（法文、英文、繁體中文等）。
- **字型大小**：調整文字大小以獲得最佳閱讀舒適度。

## 資料管理

- **匯出資料**：將您所有的對話、自訂智慧體和配置匯出為單一 JSON 封存檔案。
- **匯入資料**：還原先前的備份，或匯入外部智慧體配置。
- **重設應用程式**：永久清除所有本地資料（歷史記錄、API 金鑰、設定），以將應用程式恢復到其原始狀態。
