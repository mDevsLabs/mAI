# 網頁整合 🌐

**mAI** 被設計為一個跨平台的應用程式。除了桌面端應用程式外，mAI 還可以透過優化的網頁版本進行完整存取，從而允許從任何網頁瀏覽器進行流暢存取。

## 網頁版功能

mAI 網頁版提供與桌面應用程式幾乎完全相同的功能：
- 流暢、響應式且即時的使用者介面。
- 使用您自己的 API 金鑰存取所有語言模型（本地儲存在您的瀏覽器中或進行同步）。
- 使用大多數外掛程式（需要本地系統權限的外掛程式，例如直接本地檔案系統存取，可能會受到限制）。

## 雲端同步

如果您在多個裝置上使用 mAI，您可以啟用安全的雲端同步。您的聊天資料、自訂智慧體和偏好設定可以透過端到端加密，儲存在您自己的代管伺服器上，或透過相容的雲端儲存解決方案（例如 WebDAV 或同步資料庫服務）進行儲存。
