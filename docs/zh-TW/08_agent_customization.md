# 智慧體客製化 ⚙️

在 **mAI** 中微調智慧體（Agent）允許您深入修改它們的行為、回應風格以及它們可以存取的工具。

## 基本設定

對於每個智慧體，您可以配置：
- **系統提示詞 (System Prompt)**：定義智慧體個性、規則和角色的基礎指令。這是對智慧體回應影響最大的設定。
- **語言模型 (Language Model)**：選擇要使用的預設大語言模型（例如 GPT-4o、Claude 3.5 Sonnet、Llama 3）。

## 進階設定

您也可以修改生成超參數（hyperparameters）：
- **溫度 (Temperature)**：控制回應的創造力。較低的溫度（例如 0.2）會產生更符合事實且確定性的回答。較高的溫度（例如 0.9）則會鼓勵創意。
- **Top P**：另一種控制回應多樣性的方法。
- **存在 / 頻率懲罰 (Presence / Frequency Penalty)**：鼓勵或阻止智慧體重複相同的詞彙，或偏離討論的主題。
- **最大 Token 數 (Max Tokens)**：限制生成回應的最大長度。
