# 鍵盤快速鍵 ⌨️

為了最大化您的工作效率並在不離開鍵盤的情況下快速導覽，**mAI** 包含了一系列可配置的快速鍵。

## 全域導覽快速鍵

- `Ctrl + P`（在 Mac 上為 `Cmd + P`）：開啟快速指令選單。
- `Ctrl + ,`（在 Mac 上為 `Cmd + ,`）：開啟應用程式設定。
- `Ctrl + N`（在 Mac 上為 `Cmd + N`）：開始新的對話。
- `Ctrl + \`（在 Mac 上為 `Cmd + \`）：切換左側側邊欄的顯示/隱藏。

## 聊天快速鍵

- `Enter`：傳送訊息。
- `Shift + Enter`：在輸入欄位中插入換行符。
- `Ctrl + E`：編輯您最後傳送的訊息。
- `Ctrl + R`：重新生成智慧體最後的回應。
- `Ctrl + Delete`：刪除目前對話。

## 顯示管理快速鍵

- `Alt + 1`：切換至「對話 (Chats)」索引標籤。
- `Alt + 2`：切換至「智慧體 (Agents)」索引標籤。
- `Alt + 3`：切換至「發現 (Discover / 市集)」索引標籤。
