# 遙測功能 (Telemetry) 📊

在 **mAI** 中，透明度與尊重您的隱私是我們的首要任務。本文件詳細介紹了遙測功能在應用程式中的運作方式以及可能收集的資料。

## 匿名資料收集

為了協助我們改善應用程式的穩定性並找出介面錯誤，mAI 包含了匿名遙測功能。如果您選擇加入，以下資訊將以完全匿名的方式傳送：
- 當機報告（crash reports）與未處理的 JavaScript 異常。
- 全域使用統計（例如，創建的自訂智慧體數量、最常選擇的語言模型或外掛程式啟用率）。
- 基本系統資訊（應用程式版本、作業系統、螢幕解析度）。

## 我們絕不收集的內容

mAI 中的遙測嚴格尊重您的隱私。我們 **絕不** 收集、讀取或儲存：
- 您的訊息或對話內容。
- 您的自訂系統提示詞或智慧體配置。
- 您的 API 金鑰或憑證詳細資訊。
- 任何個人識別資訊（姓名、電子郵件地址等）。

## 啟用或停用遙測

您可以在任何時候選擇參與改善 mAI，或選擇退出：
1. 前往應用程式 **設定 (Settings)**。
2. 按一下 **關於 / 遙測 (About / Telemetry)** 索引標籤。
3. 切換 **允許匿名當機報告與使用統計** 選項。
