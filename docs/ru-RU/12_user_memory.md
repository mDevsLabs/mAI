# Пользовательская память 🧠

Пользовательская память в **mAI** позволяет вашим агентам, и в частности **May**, помнить ключевые факты о вас в различных разговорах. Это делает пользовательский опыт гораздо более персонализированным и плавным.

## Как работает память

По мере вашего взаимодействия с mAI приложение автоматически извлекает соответствующую контекстную информацию и сохраняет ее в локальной базе данных памяти. Это может быть:
- Ваши языковые предпочтения или стиль письма.
- Проекты, над которыми вы сейчас работаете.
- Ваши предпочтения в разработке (предпочитаемые фреймворки, стили кода).
- Личные данные, которыми вы решили поделиться (например, ваше имя или профессиональная роль).

## Управление вашей памятью

У вас есть полный контроль над информацией, хранящейся в mAI:
1. Перейдите в общие настройки приложения.
2. Откройте вкладку **Память** (Memory).
3. Вы увидите список сохраненных фактов в виде карточек.
4. Вы можете вручную отредактировать любую запись или навсегда удалить факт из базы данных памяти, чтобы агент забыл его.
