# Начало работы ⚡

Это руководство поможет вам настроить и запустить **mAI** локально на вашей машине разработчика.

## Предварительные требования

Убедитесь, что у вас установлено следующее:
- **Node.js** (рекомендуется версия 18 или выше)
- **pnpm** или **bun** для управления пакетами и рабочим пространством (workspace)

## Установка

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **Установите зависимости**:
   С помощью pnpm:
   ```bash
   pnpm install
   ```
   С помощью bun:
   ```bash
   bun install
   ```

3. **Запустите сервер разработки**:
   ```bash
   pnpm dev
   # или
   bun run dev
   ```

Приложение будет доступно в вашем браузере по адресу `http://localhost:3010`. При первом запуске вас встретит **May**, ваш главный ассистент, чтобы помочь настроить первоначальные параметры!
