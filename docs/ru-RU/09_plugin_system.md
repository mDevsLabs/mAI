# Система плагинов 🔌

**mAI** имеет мощную экосистему плагинов, которая позволяет разговорным агентам преодолевать ограничения обучающих данных, взаимодействуя с внешними инструментами и сервисами.

## Встроенные плагины

Вот основные плагины, доступные по умолчанию:

1. **Просмотр веб-страниц** (Web Browsing): Позволяет агенту осуществлять поиск в Интернете в режиме реального времени для получения актуальной информации.
2. **Артефакты** (Artifacts): Позволяет создавать, редактировать и просматривать код, документы или дизайн пользовательского интерфейса в интерактивном режиме на выделенной боковой панели.
3. **Погода** (Weather): Запрашивает API погоды для получения локальных прогнозов.
4. **Калькулятор кода** (Code Calculator): Выполняет локальные скрипты для точного решения сложных вычислений.

## Активация и настройка

У каждого агента может быть свой набор включенных плагинов. Вы можете включить или отключить плагины для конкретного агента на вкладке **Плагины** (Plugins) в настройках агента. Некоторым плагинам (например, коммерческому веб-поиску) могут потребоваться дополнительные параметры, такие как API-ключи, в общей конфигурации.
