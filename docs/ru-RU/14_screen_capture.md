# Снимок экрана 📸

Приложение **mAI** включает интуитивно понятный инструмент создания снимков экрана, который позволяет пользователям быстро захватывать области своего экрана и отправлять их непосредственно в обсуждение для визуального анализа агентом.

## Использование инструмента захвата

Чтобы сделать снимок экрана:
1. Нажмите на иконку **Камера / Захват** (Camera / Capture), расположенную на панели ввода чата, или используйте соответствующую горячую клавишу.
2. Интерфейс приложения временно скроется, чтобы вы могли выбрать область или окно для захвата.
3. После выбора изображение автоматически прикрепляется к вашему полю ввода чата.

## Визуальный анализ с помощью May

Используя мультимодальные возможности языковых моделей, настроенных в mAI, вы можете задавать вопросы по захваченному изображению:
- "Объясни мне это сообщение об ошибке."
- "Переведи текст, присутствующий на этом изображении."
- "Оцени этот макет и предложи улучшения дизайна."
- "Напиши код, соответствующий этому макету интерфейса."
