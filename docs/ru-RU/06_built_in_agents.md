# Встроенные агенты 📦

**mAI** поставляется с несколькими предварительно настроенными системными агентами, готовыми немедленно помочь вам в выполнении общих задач без необходимости ручной настройки.

## Список системных агентов

Вот основные встроенные агенты в приложении:

1. **May**: Ваш универсальный главный ассистент, готовый к любым задачам по написанию текстов, мозговому штурму и общей помощи.
2. **Разработчик ПО**: Агент с системным промптом, оптимизированным для программирования, написания чистого кода и отладки.
3. **Универсальный переводчик**: Специализируется на беглом переводе между десятками языков с соблюдением идиоматических выражений.
4. **Контент-райтер**: Оптимизирован для маркетинга, написания постов для блогов и создания понятных текстов для социальных сетей.

## Использование встроенных агентов

Эти агенты сразу видны в вашем списке агентов после установки. Вы можете начать новый диалог с любым из них одним кликом. Хотя они защищены от определенных системных изменений, вы можете настраивать их локальные параметры (например, связанную языковую модель) для каждого конкретного обсуждения.
