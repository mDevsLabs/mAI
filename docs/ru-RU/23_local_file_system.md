# Локальная файловая система 📂

Десктопное приложение **mAI** имеет глубокую интеграцию с файловой системой вашей операционной системы, что позволяет вам безопасно получать доступ к локальным файлам и анализировать их.

## Чтение и анализ файлов

Вы можете отправлять файлы напрямую своим агентам (например, **May**) для анализа или обработки:
- **Перетаскивание** (Drag and Drop): Перетащите текстовый документ, файл PDF, таблицу Excel или изображение прямо в область чата mAI.
- **Вложение**: Нажмите на иконку скрепки в чате, чтобы открыть проводник файлов.
- Файлы анализируются и обрабатываются локально, и только извлеченный текст (или само изображение для мультимодальных моделей) отправляется в LLM.

## Границы безопасности

Чтобы обезопасить вашу систему:
- mAI никогда не читает файлы без вашего явного действия (перетаскивание или ручной выбор файла).
- Вы можете указать выделенный каталог в общих настройках, чтобы ограничить доступ mAI только этой папкой.
