# Горячие клавиши ⌨️

Чтобы максимизировать вашу продуктивность и быстро перемещаться по приложению, не отрываясь от клавиатуры, **mAI** поддерживает ряд настраиваемых горячих клавиш.

## Глобальные сочетания клавиш навигации

- `Ctrl + P` (или `Cmd + P` на Mac): Открывает быстрое командное меню.
- `Ctrl + ,` (или `Cmd + ,` на Mac): Открывает настройки приложения.
- `Ctrl + N` (или `Cmd + N` на Mac): Начинает новый диалог.
- `Ctrl + \` (или `Cmd + \` на Mac): Сворачивает/разворачивает левую боковую панель.

## Сочетания клавиш для чата

- `Enter`: Отправить сообщение.
- `Shift + Enter`: Вставить перенос строки в поле ввода.
- `Ctrl + E`: Редактировать ваше последнее отправленное сообщение.
- `Ctrl + R`: Повторно сгенерировать последний ответ агента.
- `Ctrl + Delete`: Удалить текущее обсуждение.

## Сочетания клавиш управления отображением

- `Alt + 1`: Перейти на вкладку "Чаты" (Chats).
- `Alt + 2`: Перейти на вкладку "Агенты" (Agents).
- `Alt + 3`: Перейти на вкладку "Обзор" (Discover / Marketplace).
