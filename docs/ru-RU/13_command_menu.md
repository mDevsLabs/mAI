# Командное меню 🔍

Быстрое командное меню в **mAI** — это центральный узел управления для навигации и управления приложением без использования мыши. Оно активируется мгновенно с помощью сочетания клавиш `Ctrl + P` (или `Cmd + P` в macOS).

## Основные возможности

Как только меню открыто, вы можете вводить команды для:
- **Переключения агентов**: Введите имя агента (например, "May" или "Developer"), чтобы мгновенно переключиться на него.
- **Поиска или создания чатов**: Ищите существующие обсуждения по ключевым словам или введите `/new`, чтобы начать новую сессию чата.
- **Доступа к настройкам**: Введите `settings`, чтобы перейти непосредственно к страницам настроек системы.
- **Смены моделей**: Выберите другую LLM для активного обсуждения (например, ввод "gpt" или "claude" отфильтрует подходящие модели).

## Управление с клавиатуры и советы

- Используйте клавиши со стрелками `Вверх` (Up) и `Вниз` (Down) на клавиатуре для перемещения по списку.
- Нажмите `Enter` для подтверждения выбора.
- Нажмите `Esc` в любой момент, чтобы закрыть командное меню.
