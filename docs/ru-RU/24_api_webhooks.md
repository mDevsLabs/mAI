# API и вебхуки 🔗

**mAI** предоставляет программные интерфейсы для обеспечения интеграции со сторонними приложениями, веб-службами или скриптами автоматизации.

## Локальный API (HTTP)

Десктопное приложение mAI может запускать защищенный локальный веб-сервер на назначенном порту. Этот сервер предоставляет несколько конечных точек (endpoints):
- `POST /api/chat`: Отправляет сообщение конкретному агенту и возвращает структурированный ответ (поддерживает стандартный или потоковый формат).
- `GET /api/agents`: Возвращает список всех доступных агентов и их конфигурации.
- `POST /api/agents`: Позволяет создавать нового агента программным путем.

## Интеграция вебхуков (Webhooks)

Вы можете настроить исходящие вебхуки в расширенных настройках mAI. Вебхук автоматически отправляет JSON-данные на внешний URL-адрес при возникновении определенных событий:
- **Сообщение чата завершено** (Chat message completed): Срабатывает, как только агент заканчивает генерацию ответа.
- **Агент создан** (Agent created): Срабатывает при добавлении нового профиля агента.
- **Системная ошибка** (System error): Срабатывает при проблемах с валидацией API-ключа или сбоях в работе провайдера LLM.
