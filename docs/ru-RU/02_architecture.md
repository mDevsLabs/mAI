# Архитектура mAI 🏗️

Приложение **mAI** построено на современной архитектуре, разработанной для обеспечения производительности, расширяемости и простоты обслуживания. Оно структурировано как монорепозиторий.

## Ключевые технологии

- **Основной фреймворк**: Next.js (с App Router) для рендеринга и маршрутизации.
- **Управление состоянием**: Zustand для легковесного и реактивного управления состоянием на стороне клиента.
- **Дизайн и UI**: Компоненты Ant Design, стилизованные с помощью `antd-style` и `@lobehub/ui`.
- **База данных**: Локальная SQLite (или PostgreSQL в продакшене) под управлением Drizzle ORM.

## Структура монорепозитория

Кодовая база разделена на переиспользуемые пакеты в каталоге `packages/`:
- `packages/const`: Глобальные константы и конфигурации.
- `packages/builtin-agents`: Системные агенты по умолчанию (включая May).
- `packages/database`: Модели данных, схемы Drizzle и миграции.
- `packages/types`: Общие определения типов TypeScript.

Такое разделение изолирует основную бизнес-логику (например, выполнение API-запросов к моделям или управление плагинами) от основного веб-интерфейса пользователя, расположенного в `src/`.
