# Руководство по разработке 🛠️

Это руководство содержит основные инструкции для разработчиков, которые хотят внести свой вклад в проект **mAI** или запустить исходный код в режиме разработки.

## Структура проекта

mAI структурирован как современный монорепозиторий (с использованием pnpm и Turborepo):
- `apps/web`: Основное приложение Next.js.
- `packages/`: Общие модули и переиспользуемые конфигурации (компоненты пользовательского интерфейса, типы, вспомогательные функции).

## Запуск в режиме разработки

1. **Предварительные требования**: Убедитесь, что у вас установлены Node.js (рекомендуется LTS-версия) и pnpm или bun.
2. **Установка зависимостей**:
   ```bash
   pnpm install
   ```
3. **Запуск сервера разработки**:
   ```bash
   pnpm run dev
   ```
   Приложение будет доступно локально по адресу `http://localhost:3000`.

## Тестирование

Мы используем **Vitest** для наших наборов тестов:
- Запустить тесты один раз: `pnpm run test`
- Запустить тесты в режиме отслеживания изменений (watch mode): `pnpm run test:watch`

Пожалуйста, убедитесь, что все тесты успешно проходят перед отправкой любых изменений.
