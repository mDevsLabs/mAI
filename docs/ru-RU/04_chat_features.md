# Чаты и функции рендеринга 💬

Разговорный интерфейс **mAI** разработан для обеспечения плавного интерактивного взаимодействия, наполненного визуальными возможностями и точными параметрами управления.

## Основные возможности чата

- **Встроенное редактирование**: Вы можете отредактировать любое из ваших предыдущих сообщений в любой момент. mAI автоматически перезапустит диалог с этого шага.
- **Многофункциональный рендеринг Markdown**: mAI поддерживает полное форматирование Markdown: таблицы, математические уравнения LaTeX, блоки кода с подсветкой синтаксиса и интерактивные панели предпросмотра.
- **Быстрые действия**: Каждое текстовое облако сообщения содержит контекстное меню, позволяющее скопировать необработанный контент, заново сгенерировать ответ ИИ, отредактировать или удалить сообщение из вашей истории.
- **Отслеживание истории**: Сессии чатов архивируются локально в базе данных SQLite, что позволяет вам мгновенно искать и восстанавливать ваши прошлые разговоры.
