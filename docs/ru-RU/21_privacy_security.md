# Конфиденциальность и безопасность 🔒

Безопасность данных и уважение к вашей конфиденциальности — фундаментальные принципы разработки **mAI**.

## Зашифрованное локальное хранилище

Все ваши конфиденциальные данные хранятся локально на вашем устройстве:
- **API-ключи**: Они шифруются с использованием отраслевых стандартных алгоритмов перед сохранением в локальном хранилище. Ваши API-ключи никогда не отправляются на наши серверы; они отправляются напрямую с вашего устройства на серверы соответствующих провайдеров LLM.
- **История чатов**: Ваши разговоры остаются на вашей машине (если вы явно не включите безопасную облачную синхронизацию).

## Безопасные соединения

Все исходящие запросы к API языковых моделей используют безопасные протоколы связи (HTTPS / SSL), что гарантирует невозможность перехвата обмениваемых данных в сети.

## Рекомендуемые корпоративные практики

Для максимального уровня безопасности в корпоративных средах:
- Используйте модели, размещенные локально через **Ollama** или частный сервер вывода (это означает, что данные не покидают корпоративную сеть).
- Отключите анонимную телеметрию на вкладке общих настроек.
