# Configurações Gerais ⚙️

As configurações gerais no **mAI** permitem que você ajuste o aplicativo às suas preferências estéticas, linguísticas e funcionais.

## Interface do Usuário e Temas

- **Tema**: Escolha entre o modo claro, modo escuro ou deixe o aplicativo seguir as preferências do sistema do seu computador.
- **Cor de Destaque (Accent Color)**: Personalize a cor dos botões ativos e dos principais elementos interativos da interface gráfica.
- **Idioma da Interface**: Selecione o idioma do aplicativo (Francês, Inglês, etc.).
- **Tamanho da Fonte**: Ajuste o tamanho do texto para um conforto de leitura ideal.

## Gerenciamento de Dados

- **Exportar Dados**: Exporte todas as suas conversas, agentes personalizados e configurações em um único arquivo de arquivo JSON.
- **Importar Dados**: Restaure um backup anterior ou importe configurações de agentes externos.
- **Redefinir Aplicativo**: Apague permanentemente todos os dados locais (histórico, chaves de API, configurações) para restaurar o aplicativo ao seu estado original.
