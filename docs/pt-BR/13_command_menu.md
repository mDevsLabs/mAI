# Menu de Comandos 🔍

O menu de comandos rápidos no **mAI** é a central de controle para navegar e gerenciar o aplicativo sem precisar tocar no mouse. Ele é ativado instantaneamente usando o atalho `Ctrl + P` (ou `Cmd + P` no macOS).

## Principais Recursos

Assim que o menu estiver aberto, você pode digitar comandos para:
- **Alternar Agentes**: Digite o nome de um agente (como "May" ou "Developer") para alternar instantaneamente.
- **Buscar ou Criar Chats**: Busque suas discussões existentes por palavras-chave ou digite `/new` para iniciar uma nova sessão de chat.
- **Acessar Configurações**: Digite `settings` para ir diretamente para as páginas de configurações do sistema.
- **Alterar Modelos**: Selecione um LLM diferente para a discussão ativa (por exemplo, digitar "gpt" ou "claude" filtrará os modelos correspondentes).

## Controles de Teclado e Dicas

- Use as teclas de seta `Para cima` e `Para baixo` no seu teclado para navegar pela lista.
- Pressione `Enter` para confirmar uma seleção.
- Pressione `Esc` para fechar o menu de comandos a qualquer momento.
