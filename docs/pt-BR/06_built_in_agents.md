# Agentes Integrados 📦

O **mAI** vem pré-configurado com vários agentes de sistema integrados para ajudá-lo imediatamente em tarefas comuns, sem a necessidade de qualquer configuração manual.

## Lista de Agentes do Sistema

Aqui estão os principais agentes integrados no aplicativo:

1. **May**: Sua assistente principal versátil, pronta para todas as tarefas de escrita, brainstorming e ajuda geral.
2. **Software Developer (Desenvolvedor)**: Um agente configurado com uma instrução de sistema adaptada para programação, escrita de código limpo e depuração.
3. **Universal Translator (Tradutor)**: Especializado em tradução fluida entre dezenas de idiomas, respeitando expressões idiomáticas.
4. **Content Writer (Redator)**: Otimizado para marketing, redação de postagens de blog e escrita de textos claros para redes sociais.

## Usando Agentes Integrados

Esses agentes ficam imediatamente visíveis na sua lista de agentes após a instalação. Você pode iniciar uma nova conversa com qualquer um deles com um único clique. Embora eles sejam protegidos contra certas modificações do sistema, você pode personalizar seus parâmetros locais (como o modelo de linguagem associado) para cada discussão.
