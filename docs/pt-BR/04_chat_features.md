# Chats e Recursos de Renderização 💬

A interface de conversação do **mAI** foi projetada para oferecer uma experiência fluida e interativa, repleta de recursos visuais e opções de controle preciso.

## Principais Recursos do Chat

- **Edição Inline**: Você pode editar qualquer uma de suas mensagens anteriores a qualquer momento. O mAI executará novamente a conversa de forma automática a partir daquele ponto.
- **Renderização Rica em Markdown**: O mAI suporta formatação Markdown completa: tabelas, equações matemáticas LaTeX, blocos de código com realce de sintaxe e painéis de visualização interativos.
- **Ações Rápidas**: Cada balão de mensagem possui um menu de contexto que permite copiar o conteúdo bruto, regenerar a resposta da IA, editar ou excluir a mensagem do seu histórico.
- **Histórico de Conversas**: As sessões de chat são arquivadas localmente no banco de dados SQLite, permitindo que você busque e recupere instantaneamente suas conversas passadas.
