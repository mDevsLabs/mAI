# Memória do Usuário 🧠

A memória do usuário no **mAI** permite que seus agentes, e particularmente a **May**, lembrem-se de fatos importantes sobre você em diferentes conversas. Isso torna a experiência do usuário muito mais personalizada e contínua.

## Como a Memória Funciona

Conforme você interage com o mAI, o aplicativo extrai automaticamente informações contextuais relevantes e as armazena em um banco de dados de memória local. Isso pode incluir:
- Suas preferências de idioma ou estilo de escrita.
- Projetos nos quais você está trabalhando atualmente.
- Suas preferências de desenvolvimento (frameworks preferidos, estilos de código).
- Detalhes pessoais que você escolhe compartilhar (como seu nome ou cargo profissional).

## Gerenciando sua Memória

Você tem controle total sobre as informações armazenadas pelo mAI:
1. Vá para as configurações gerais do aplicativo.
2. Abra a guia **Memória**.
3. Você verá uma lista de fatos armazenados como cartões.
4. Você pode editar manualmente qualquer entrada ou excluir permanentemente um fato do banco de dados de memória para que o agente o esqueça.
