# Sistema de Plugins 🔌

O **mAI** apresenta um poderoso ecossistema de plugins que permite aos agentes conversacionais superar as limitações de seus dados de treinamento, interagindo com ferramentas e serviços externos.

## Plugins Integrados

Aqui estão os principais plugins disponíveis por padrão:

1. **Navegação Web (Web Browsing)**: Permite que o agente pesquise na internet em tempo real para obter informações atualizadas.
2. **Artefatos (Artifacts)**: Permite gerar, editar e visualizar códigos, documentos ou designs de interface do usuário de forma interativa em uma barra lateral dedicada.
3. **Clima (Weather)**: Consulta APIs de clima para obter previsões locais.
4. **Calculadora de Código (Code Calculator)**: Executa scripts locais para resolver cálculos complexos com precisão.

## Ativação e Configuração

Cada agente pode ter seu próprio conjunto de plugins ativados. Você pode ativar ou desativar plugins para um agente específico através da guia **Plugins** nas configurações do agente. Alguns plugins (como a pesquisa web comercial) podem exigir configurações adicionais, como chaves de API, na configuração geral.
