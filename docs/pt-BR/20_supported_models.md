# Modelos de Linguagem Suportados 🧠

O **mAI** é uma interface agnóstica de modelo. Ele suporta uma ampla variedade de provedores de inteligência artificial baseados em nuvem comercial e de código aberto locais.

## Principais Provedores de Nuvem

Você pode configurar os seguintes provedores de nuvem inserindo sua chave de API pessoal:
- **OpenAI**: GPT-4o, GPT-4 Turbo, GPT-3.5, etc.
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku.
- **Google Gemini**: Gemini 1.5 Pro, Gemini 1.5 Flash.
- **Mistral AI**: Mistral Large, Mistral Medium, Codestral.
- **Groq**: Acesso ultra-rápido a modelos de código aberto como Llama 3 e Mixtral.

## Integração de Modelos Locais

Para usuários preocupados com a privacidade total de seus dados, o mAI suporta totalmente a execução local:
- **Ollama**: Integração nativa simplificada. Execute o Ollama em segundo plano e o mAI detectará automaticamente os modelos instalados localmente (como Llama 3, Mistral ou Phi-3).
- **APIs compatíveis com OpenAI**: Conecte qualquer servidor de inferência local ou remoto (como LocalAI ou LM Studio) especificando uma URL base de API personalizada.
