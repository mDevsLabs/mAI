# Integração Web 🌐

O **mAI** foi projetado como um aplicativo multiplataforma. Além do aplicativo para desktop, o mAI é totalmente acessível através de uma versão web otimizada, permitindo acesso fluido a partir de qualquer navegador web.

## Recursos da Versão Web

A versão web do mAI oferece paridade de recursos quase completa com o aplicativo desktop:
- Interface de usuário fluida, responsiva e reativa.
- Acesso a todos os modelos de linguagem usando suas próprias chaves de API (salvas localmente no seu navegador ou sincronizadas).
- Uso da maioria dos plugins (plugins que exigem privilégios do sistema local, como acesso direto ao sistema de arquivos local, podem ter restrições).

## Sincronização em Nuvem

Se você usa o mAI em múltiplos dispositivos, pode ativar a sincronização segura em nuvem. Seus dados de chat, agentes personalizados e preferências podem ser salvos com criptografia de ponta a ponta no seu próprio servidor de hospedagem ou por meio de soluções de armazenamento em nuvem compatíveis (como WebDAV ou serviços de banco de dados sincronizados).
