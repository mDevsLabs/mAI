# Captura de Tela 📸

O aplicativo **mAI** inclui uma ferramenta intuitiva de captura de tela que permite aos usuários capturar rapidamente partes de sua tela e enviá-las diretamente para a discussão para análise visual pelo agente.

## Usando a Ferramenta de Captura

Para tirar uma captura de tela:
1. Clique no ícone de **Câmera / Captura** localizado na barra de entrada do chat ou use o atalho dedicado.
2. A interface do usuário se oculta temporariamente para permitir que você selecione a região ou janela que deseja capturar.
3. Uma vez selecionada, a imagem é anexada automaticamente ao seu campo de entrada do chat.

## Análise Visual com a May

Aproveitando os recursos multimodais dos modelos de linguagem configurados no mAI, você pode fazer perguntas sobre a imagem capturada:
- "Explique esta mensagem de erro para mim."
- "Traduza o texto presente nesta imagem."
- "Revise este layout e sugira melhorias de design."
- "Escreva o código correspondente a este protótipo de interface."
