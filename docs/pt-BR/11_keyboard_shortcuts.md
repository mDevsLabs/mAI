# Atalhos de Teclado ⌨️

Para maximizar sua produtividade e navegar rapidamente sem tirar as mãos do teclado, o **mAI** inclui uma série de atalhos configuráveis.

## Atalhos de Navegação Global

- `Ctrl + P` (ou `Cmd + P` no Mac): Abre o menu de comandos rápidos.
- `Ctrl + ,` (ou `Cmd + ,` no Mac): Abre as configurações do aplicativo.
- `Ctrl + N` (or `Cmd + N` no Mac): Inicia uma nova conversa.
- `Ctrl + \` (ou `Cmd + \` no Mac): Alterna a exibição da barra lateral esquerda.

## Atalhos do Chat

- `Enter`: Envia a mensagem.
- `Shift + Enter`: Insere uma quebra de linha no campo de entrada.
- `Ctrl + E`: Edita sua última mensagem enviada.
- `Ctrl + R`: Regenera a última resposta do agente.
- `Ctrl + Delete`: Exclui a discussão atual.

## Atalhos de Gerenciamento de Abas

- `Alt + 1`: Alterna para a aba de Chats.
- `Alt + 2`: Alterna para a aba de Agentes.
- `Alt + 3`: Alterna para a aba de Descoberta (Mercado/Marketplace).
