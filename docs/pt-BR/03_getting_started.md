# Primeiros Passos ⚡

Este guia ajuda você a configurar e executar o **mAI** localmente em sua máquina de desenvolvimento.

## Pré-requisitos

Certifique-se de ter o seguinte instalado:
- **Node.js** (v18 ou superior recomendado)
- **pnpm** ou **bun** para gerenciamento de pacotes e workspaces

## Instalação

1. **Clonar o repositório**:
   ```bash
   git clone https://github.com/mDevsLabs/mAI.git
   cd mAI
   ```

2. **Instalar dependências**:
   Com pnpm:
   ```bash
   pnpm install
   ```
   Com bun:
   ```bash
   bun install
   ```

3. **Executar o servidor de desenvolvimento**:
   ```bash
   pnpm dev
   # ou
   bun run dev
   ```

O aplicativo estará acessível no seu navegador em `http://localhost:3010`. No seu primeiro lançamento, você será recebido pela **May**, sua assistente principal, para configurar suas preferências iniciais!
