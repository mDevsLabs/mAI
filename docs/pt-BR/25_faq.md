# FAQ ❓

Aqui estão as respostas para as perguntas mais frequentes sobre a configuração e o uso do **mAI**.

## Uso Geral

### Posso usar o mAI sem conexão com a internet?
Sim, você pode, desde que esteja usando modelos locais executados por meio de serviços como o **Ollama** ou LocalAI. Se você estiver usando modelos em nuvem (como OpenAI ou Anthropic), será necessária uma conexão ativa com a internet para se comunicar com os servidores deles.

### Como faço para atualizar minhas chaves de API?
Vá em Configurações (`Ctrl + ,`) e abra a guia **Modelos de Linguagem**. A partir daí, você pode adicionar, alterar ou remover chaves de API para cada provedor suportado.

## Personalização de Agente

### Quem é a May?
A **May** é a assistente inteligente padrão no mAI. Ela é altamente versátil e configurada para ajudá-lo desde o seu primeiro lançamento.

### Meus agentes são compartilhados com outros usuários?
Não. Por padrão, todos os agentes personalizados que você cria são armazenados localmente e permanecem privados no seu dispositivo. Você pode, no entanto, exportá-los manualmente para compartilhá-los com outras pessoas.

## Custo e Faturamento

### O uso do mAI é gratuito?
O aplicativo mAI em si é totalmente gratuito e de código aberto. No entanto, a utilização de modelos comerciais em nuvem (como OpenAI ou Anthropic) exige que você forneça suas próprias chaves de API. Você pagará apenas pelo seu uso real da API diretamente aos provedores de modelos.
