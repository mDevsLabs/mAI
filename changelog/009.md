# Version 0.0.9

## Changements apportés :
- **Correction des dépendances** 📦 : Mise à jour de 23 dépendances désalignées via `pnpm install` pour correspondre au manifeste.
- **Mise à jour de la version** 🆙 : Passage du projet à la version `0.0.9` dans `package.json`.
- **Corrections de build** 🛠️ :
  - `sheet-editor.tsx` : Passage à l'import nommé `{ DataGrid }` car l'export par défaut n'existait plus dans `react-data-grid` v7-beta.59. Commentaire de `textEditor` qui n'existait plus non plus.
  - `schema.ts` : Correction de `z.record()` pour passer 2 arguments (`z.string(), z.unknown()`) comme requis par la nouvelle version de Zod.
  - `upload/route.ts` : Remplacement de `.errors` par `.issues` pour l'accès aux erreurs Zod.
  - `coder/page.tsx` : Suppression de l'import de `GithubIcon` qui n'était pas trouvé dans `lucide-react` et semblait inutilisé.

## Recommandations futures :
- **Suivre le build** ⏳ : Le build a pris beaucoup de temps à compiler (plus d'une heure) et tournait encore sur l'étape TypeScript. Il faut surveiller s'il se termine avec succès ou s'il y a d'autres erreurs de type à corriger.
- **Rétablir l'éditeur de tableau** 📊 : Une fois le build stable, réactiver l'éditeur avec la bonne API de `react-data-grid` v7.
