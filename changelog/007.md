# Version 0.0.7

## Changements apportés :
- **Badges & XP fonctionnels** : Ajout des tables `UserXP`, `UserBadges` et `XPHistory` dans le schéma de la base de données. Implémentation d'une fonction `creditXP` pour attribuer de l'XP aux utilisateurs (5 XP par message).
- **Page Statistiques** : Mise à jour de la page pour correspondre au design de l'image fournie, avec l'affichage des tokens, des règles d'XP et une modal d'historique des gains d'XP.
- **Application Traduction** : Création de la page `app/(chat)/applications/translation/page.tsx` permettant de traduire du texte dans plusieurs langues simultanément via l'API LibreTranslate. Ajout de sections pour l'analyse lexicale et les synonymes.
- **Icônes SVG personnalisées** : Remplacement des icônes Lucide par des composants SVG personnalisés dans la page Applications pour correspondre au style demandé.
- **Correction de bug** : Résolution d'une erreur de build sur la page Paramètres due à des imports manquants (`Switch` et `Progress`), remplacés par des solutions natives et CSS.
- **Option Réécrire** : Connexion du bouton "Réécrire" à la fonction `regenerate` du chat pour relancer une génération.

## Recommandations futures :
- **Finalisation de l'intégration BDD** : Lier l'interface des statistiques aux données réelles de la base de données (les tables sont prêtes, il reste à créer les API de lecture).
