# Version 0.0.8

## Changements apportés :
- **Application Coder** : Création d'un environnement de développement complet avec IA, terminal simulé, arborescence de fichiers et plus de 20 fonctionnalités basées sur les captures fournies.
- **Application Speaky** : Création d'une application de synthèse vocale utilisant l'API `Web Speech API` avec de nombreux paramètres de personnalisation (voix, style, vitesse, ton) et plus de 20 fonctionnalités.
- **Déclencheurs d'XP automatiques** : Ajout du gain d'XP pour les votes (+3 XP) et la génération d'images (+10 XP).
- **Logique de déblocage des Badges** : Implémentation de la fonction `checkAndUnlockBadges` qui vérifie les conditions (nombre de messages, images) et débloque les badges en BDD après chaque action.
- **Correction de bug** : Résolution d'une erreur de type dans `message.tsx` concernant la variable `regenerate`.

## Recommandations futures :
- **Amélioration du terminal Coder** : Connecter le terminal à un vrai moteur d'exécution (ex: Docker) si la sécurité le permet.
- **Voix premium pour Speaky** : Intégrer une API externe (ex: ElevenLabs) pour des voix plus réalistes que celles du navigateur.
