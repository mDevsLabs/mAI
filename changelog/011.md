# Version 0.0.11 - Correction des bugs de linting et optimisations

Cette version se concentre sur la résolution des avertissements et erreurs remontés par le linter Biome, ainsi que sur l'amélioration de l'accessibilité et du code.

## Changements apportés

- **Correction des bugs de linting** : De nombreux fichiers ont été nettoyés pour supprimer les variables inutilisées, ajouter des valeurs par défaut dans les structures `switch`, et corriger les types évolutifs.
- **Accessibilité (a11y)** : Modification de certains éléments interactifs (comme des `div` cliquables) en `button` ou ajout d'attributs nécessaires pour respecter les standards d'accessibilité.
- **Optimisation des images** : Ajout de commentaires d'ignorance pour les balises `img` là où l'utilisation de `next/image` n'était pas adaptée (par exemple pour les aperçus d'images générées ou les blobs).
- **Mise à jour des versions** : Tous les fichiers modifiés ont vu leur version incrémentée dans l'en-tête.
- **Build validé** : La commande `pnpm run build` a été exécutée avec succès après les corrections.
