# Version 0.0.5

## Changements apportés :
- **Barre de messages (Composer) — Bouton `+`** : Remplacement du bouton trombone par un bouton `+` ouvrant un menu avec plusieurs options (Recherche Web, Canevas, Météo, Création d'images, Quiz, Apprendre & Étudier).
- **Menu des Modèles** : Amélioration du sélecteur de modèles. Suppression des pointillés sur le modèle sélectionné et amélioration du chargement des logos pour GPT, Claude et DeepSeek en mappant correctement les providers.
- **Boutons de la Barre Supérieure** :
    - **Bouton Fantôme** : Ajout d'un bouton pour activer le mode fantôme. Si activé, la conversation n'est pas enregistrée dans la base de données.
    - **Bouton Notifications** : Ajout d'un système de notifications persistantes dans le `localStorage`. Une notification est envoyée lorsque l'IA répond et lorsque des conversations sont supprimées.
- **Menu Utilisateur & Paramètres** :
    - **Page Paramètres** : Création d'une nouvelle page de paramètres accessible depuis le menu utilisateur.
    - **Exporter les données** : Ajout d'une option pour exporter toutes les données et conversations de l'utilisateur dans un fichier JSON.
- **Humanizy** : Squelette de l'application et route API créés dans les étapes précédentes.

## Recommandations futures :
- **Intégration réelle des outils** : Lier les options du menu `+` (Recherche, Météo, etc.) à de vrais outils ou actions dans le chat.
- **Amélioration de Humanizy** : Connecter le détecteur d'IA à une vraie API de détection ou améliorer l'algorithme de simulation.
- **Finalisation des pages secondaires** : Créer les pages Statistiques, Mettre à niveau et Archives mentionnées par l'utilisateur.
