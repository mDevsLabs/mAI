# mAI 🚀

mAIは、エージェントを24時間365日稼働させるために整理・管理します。AIチーム全体の採用、スケジュール設定、およびレポート生成を行います。あなたはオンラインのままでいる必要はなく、常にコントロールを維持できます。

**May**はコアとなるAIアシスタントであり、すべてのタスクをサポートする準備ができています。

## 主な機能

mAIは、あなたと共に進化するエージェントを見つけ、作成し、コラボレーションするためのワークスペースです。mAIでは、エージェントを主要な作業単位として扱い、人間とエージェントが共進化するインフラを提供します。

### 👥 作業単位としてのエージェント
AIチーム全体の活動をスケジュールし、監視します。
- **生産性の向上**: すべてのエージェントを1つの場所に集約します。
- **メッセージ機能 di 統合**: お気に入りのアプリケーションからエージェントと直接チャットできます。

### 🛠️ エージェントビルダー
カスタマイズされたAIチーム의 構築は、エージェントビルダーから始まります。必要な要件を一度記述するだけで、エージェントは即座に使用できるように自動構成されます。
- **統合されたインテリジェンス**: 管理下にあるすべてのモデルやモダリティにシームレスにアクセスできます。
- **スキルの拡張**: 互換性のあるプラグインやMCPサーバーを使用して、エージェントを日常のツールに接続します。

### 🤝 コラボレーション
mAIは、実際のリモートの同僚であるかのようにエージェントと連携できる「エージェントグループ」を導入します。システムはタスクに適したエージェントを組み立て、並行したコラボレーションと継続的な改善を可能にします。
- **ページ**: 共有コンテキストを使用して、複数のエージェントと同時にドキュメントを下書きし、推敲します。
- **スケジューリング**: 実行スケジュールを設定し、オフラインのときでも適切なタイミングでエージェントに作業させることができます。
- **プロジェクト**: プロジェクトごとに作業を整理し、明確な構造を維持します。

### 🧠 人間とエージェントの共進化
最高のAIとは、あなたを深く理解しているAIです。mAIは、やり取りを通じて自己学習し強化されるパーソナルメモリを備えています。
- **継続的な学習**: エージェントはあなたの働き方から学習します。
- **透明性のあるメモリ**: エージェントは、あなたが完全に制御できる、構造化され編集可能なメモリを使用します。
