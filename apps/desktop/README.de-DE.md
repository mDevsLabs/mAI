# 🤯 mAI Desktop-Anwendung

mAI Desktop ist eine plattformübergreifende Desktop-Anwendung für [mAI](https://github.com/mDevsLabs/mAI), entwickelt mit Electron, die ein nativeres Desktop-Erlebnis und native Funktionen bietet.

## ✨ Funktionen

- **🌍 Plattformübergreifende Unterstützung**: Unterstützt macOS (Intel/Apple Silicon), Windows und Linux
- **🔄 Automatische Updates**: Integrierter Update-Mechanismus stellt sicher, dass Sie immer die neueste Version haben
- **🌐 Mehrsprachige Unterstützung**: Vollständige i18n-Unterstützung für über 18 Sprachen mit Lazy Loading
- **🎨 Native Integration**: Tiefgehende Betriebssystem-Integration mit nativen Menüs, Tastenkombinationen und Benachrichtigungen
- **🔒 Sicher & Zuverlässig**: macOS notariell beglaubigt, verschlüsselter Token-Speicher, sicherer OAuth-Ablauf
- **📦 Mehrere Release-Kanäle**: Stable-, Beta- und Nightly-Versionen
- **⚡ Erweiterte Fensterverwaltung**: Multi-Fenster-Architektur mit Designschnittstellen-Synchronisation
- **🔗 Remote-Server-Synchronisierung**: Sichere Datensynchronisierung mit Remote-mAI-Instanzen
- **🎯 Entwicklerwerkzeuge**: Integriertes Entwicklerpanel und umfassende Debugging-Tools

## 🚀 Entwicklungsumgebung einrichten

### Voraussetzungen

- **Node.js** 22+
- **pnpm** 10+
- **Electron**-kompatible Entwicklungsumgebung

### Schnellstart

```bash
# Abhängigkeiten installieren
pnpm install-isolated

# Entwicklungsserver starten
pnpm dev

# Typprüfung
pnpm type-check

# Tests ausführen
pnpm test
```
