# 🤯 Aplicación de Escritorio mAI

mAI Desktop es una aplicación de escritorio multiplataforma para [mAI](https://github.com/mDevsLabs/mAI), construida con Electron, que proporciona una experiencia de escritorio y funcionalidad más nativas.

## ✨ Características

- **🌍 Soporte Multiplataforma**: Soporta macOS (Intel/Apple Silicon), Windows y Linux
- **🔄 Actualizaciones Automáticas**: El mecanismo de actualización incorporado garantiza que siempre tenga la última versión
- **🌐 Soporte Multilingüe**: Soporte completo de i18n para más de 18 idiomas con carga perezosa
- **🎨 Integración Nativa**: Integración profunda con el sistema operativo con menús, accesos directos y notificaciones nativos
- **🔒 Seguro y Confiable**: Notarizado para macOS, almacenamiento de tokens cifrado, flujo de OAuth seguro
- **📦 Múltiples Canales de Lanzamiento**: Versiones estable, beta y nightly
- **⚡ Gestión Avanzada de Ventanas**: Arquitectura de múltiples ventanas con sincronización de temas
- **🔗 Sincronización del Servidor Remoto**: Sincronización segura de datos con instancias remotas de mAI
- **🎯 Herramientas de Desarrollo**: Panel de desarrollo integrado y herramientas de depuración completas

## 🚀 Configuración de Desarrollo

### Requisitos Previos

- **Node.js** 22+
- **pnpm** 10+
- Entorno de desarrollo compatible con **Electron**

### Inicio Rápido

```bash
# Instalar dependencias
pnpm install-isolated

# Iniciar servidor de desarrollo
pnpm dev

# Comprobación de tipos
pnpm type-check

# Ejecutar pruebas
pnpm test
```
