# 🤯 Applicazione Desktop mAI

mAI Desktop è un'applicazione desktop multipiattaforma per [mAI](https://github.com/mDevsLabs/mAI), costruita con Electron, che offre un'esperienza desktop e funzionalità più native.

## ✨ Caratteristiche

- **🌍 Supporto Multipiattaforma**: Supporta macOS (Intel/Apple Silicon), Windows e Linux
- **🔄 Aggiornamenti Automatici**: Il meccanismo di aggiornamento integrato garantisce di avere sempre l'ultima versione
- **🌐 Supporto Multilingue**: Supporto i18n completo per oltre 18 lingue con caricamento differito
- **🎨 Integrazione Nativa**: Profonda integrazione con il sistema operativo con menu nativi, scorciatoie e notifiche
- **🔒 Sicuro e Affidabile**: Notarizzato per macOS, archiviazione token crittografata, flusso OAuth sicuro
- **📦 Canali di Rilascio Multipli**: Versioni stabile, beta e nightly
- **⚡ Gestione Avanzata delle Finestre**: Architettura multi-finestra con sincronizzazione dei temi
- **🔗 Sincronizzazione Server Remoto**: Sincronizzazione sicura dei dati con istanze mAI remote
- **🎯 Strumenti di Sviluppo**: Pannello di sviluppo integrato e strumenti di debug completi

## 🚀 Configurazione di Sviluppo

### Prerequisiti

- **Node.js** 22+
- **pnpm** 10+
- Ambiente di sviluppo compatibile con **Electron**

### Avvio Rapido

```bash
# Installa le dipendenze
pnpm install-isolated

# Avvia il server di sviluppo
pnpm dev

# Controllo dei tipi
pnpm type-check

# Esegui i test
pnpm test
```
